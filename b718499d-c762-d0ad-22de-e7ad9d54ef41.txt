　　山山禅师反应也是极快，手腕一翻，将一副念珠拿在手中，在电光石火之际，套在了打神鞭上，用手一拨，念珠霎时间明亮起来，放出金光，投射在山山禅师面前，一尊巨大的神像形成，头戴法冠，身穿红蓝相间的长袍，双眼紧闭，眉心中间长着一直朱红色的竖眼，面色祥和，却不怒自威。
<br/>　　这可不是二郎神。而是帝释天！
<br/>　　山山禅师，用诸天观想术观想出来的帝释天的幻象。
<br/>　　诸天观想术能观想出诸天神佛，但这些神佛的实力也是有高有低，像四宝偷艺，所观想出来的韦驮天，顶多算是中等实力。帝释天却不同，实力不知道比韦驮天强了多少倍。
<br/>　　帝释天手里没拿什么兵刃，只是端着一盏莲花灯，面对“上古邪神”，他双手搓灯，有五色神光飞出，化作佛门真言，朝道风降下来。
<br/>　　“上古邪神”手持打神鞭，姿势不变，也没用什么法术，就这么垂直地砸下去。
<br/>　　硬碰了一记。
<br/>　　彼此对穿过去。
<br/>　　几秒钟后，帝释天的影像粉碎，山山禅师后退半步，脸上潮红，显得十分痛苦。
<br/>　　“上古邪神”也不好受，被震飞出去，落在地上，一身衣衫尽碎，却露出了一脸的青色长袍，“上古邪神”站起来，脸上的面具也碎了，露出了他那世所罕见的俊美但一点也不娘的脸。
<br/>　　“道风！”
<br/>　　清长风大叫起来，又他妈的是道风，到底有几个道风！
<br/>　　清长风快要吐血了，在山里围观的那些吃瓜大佬们也要吐血了。
<br/>　　怎么会这样啊？
<br/>　　道风转身朝那个融化得只剩下骨骸的“自己”看过去，神色之间，也罕见地带着几分动容。
<br/>　　“你为什么不走？”道风问道。
<br/>　　那骨骸几乎完全融化，露出了他的元神，起初还是道风的模样，但是被法阵一激，立刻也开始融化了，并且现出了本尊的模样：
<br/>　　身材高大、面容丑陋，一对森森的獠牙露出在嘴唇之外。
<br/>　　邪灵？
<br/>　　清长风一愣，猛然间回过神来，失声叫起来：“你才是上古邪神！”
<br/>　　他才是真正的上古邪神。
<br/>　　清长风，还有山里那些大佬们没有看穿他的相貌，只是因为，他长了一张跟道风一模一样的脸，这不是幻变，而是真的，所以才能瞒天过海，让任何人都没能看出来。
<br/>　　他这张脸，是道风赐予的。
<br/>　　上一次前往零界那一战，出发之前，道风就对上古邪神承诺，若是平安归来，赐给他一张脸。
<br/>　　于是在事情了结，回到风之谷之后，道风就拆了自己身外化身的一缕精气，种在他的元神之中，在上古邪神祭炼之下，养成了自己的模样。
<br/>　　主子赐给自己容貌，这是鬼域从古至今的一种象征荣誉的传统，就像古代皇帝赐姓一样，比这多一层的是，被赐了容貌之后，就成为主人的面首，并且可以通过这一缕精气，修炼主人的功法、使用主人的法器。
<br/>　　当时，道风并没有想过有今天。
<br/>　　这个计划，是在叶少阳找自己之前、道风就制定好了的。
<br/>　　就算没有叶少阳，他也要进攻轩辕山，这是他之前就想好的计划。
<br/>　　在出发之前，他跟上古邪神就互相换了身份，他戴上了上古邪神的面具，而上古邪神则展示出了他那张道风的脸，以他的身份，吸引所有人的注意。
<br/>　　上古邪神的任务，就是拖住清长风。正是因为有他的牵制，成功吸引了所有人的注意力，道风才能在与山山禅师在幻境中斗法的间隙，偷偷释放身外化身，在山洞里杀了假叶少阳，然后一路护送芮冷玉出去。
<br/>　　虽然还是被清长风发现，肉身被斩……对他本尊的影响还是很大的。
<br/>　　他能在幻境中击败山山禅师，靠的不光是实力——而是山山禅师用了大如来神咒，在幻想中带道风去了佛国婆娑净土，听佛祖讲经，道风若是入神，或惶恐，或惊叹，或畏惧，只要流露出一点这样的情绪，都会被心魔利用，一点点侵占他的神识，最后改变他的信仰。
<br/>　　说到底，山山禅师是想渡化他进入佛门，而不是要杀了他，否则他若是放开一战，失去了身外化身的道风，未必是他对手。
<br/>　　可惜，他这对一般人百试百灵的法术，到道风这根本不管用——若是一般生灵，不管你是没有信仰，还是信仰别的什么宗教，只要中了山山禅师这法术，一定会被打上信仰的烙印，改信佛教。
<br/>　　然而，山山禅师也没想到的是，道风的信仰会那么坚定，最后被神识之力反噬，这才受了伤。
<br/>　　上古邪神硬刚清长风，也是为了给道风争取时间，在得到了道风那一缕精气之后，他突破瓶颈，突破邪神境界，晋升初级灵尊，又借助了打神鞭和番天印的神威，这才有了与清长风的一战之力，不过他终究不是清长风的对手。
<br/>　　在他受伤之后，道风在神识中让他赶紧找机会逃走，但上古邪神犹豫了。
<br/>　　当时，道风正在与山山禅师斗法，他现出真身，倒是有机会逃走，但那样一来，道风便会暴露，清长风当然不会放过他。
<br/>　　于是，上古邪神炼化了自己的血肉骨骼，来对抗法阵，为道风争取时间——道风在与山山禅师幻境斗法的途中，元神出窍，去了北斗观……那里，才是他此行真正的目的所在。
<br/>　　“道风！好个漫天过海的妙计啊！”
<br/>　　清长风总算明白过来，恨的咬牙切齿。
<br/>　　之前错把上古邪神当成道风，又是控阵又是调集手下围堵的，想到了所有的可能，确定道风插翅难逃，然而从头到尾，他都被忽悠了，连对手是谁都没搞清楚！
<br/>　　如今看清真相，再结合那个弟子前来汇报的情况，这才醒悟，顿时感到彻骨生寒，转身望着山谷深处喊道：“诸位，请速速赶去我北斗观，保护星月奴肉身，若她肉身受损，法术公会数百年谋划毁于一旦，谁也担不起这个责任！”
<br/>　　他这一声喊，被那几个在附近山上吃瓜围观的大佬们听到，当场都愣住了。
