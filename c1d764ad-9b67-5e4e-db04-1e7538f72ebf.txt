　　第381章
<br/>　　希曼·福德托斯，一名出色的炼金术师。在他的领域，已取得不错的成果。在诺顿王国，他曾经给皇家学院的学生讲过课。
<br/>　　他信奉生命之神，坚定的不婚主义者，愿意将灵魂和肉体都献祭给神明。内心坚定的相信，世间万物都由一个基本物质组成。所以毕生钻研在物质转换上：如何把黄土炼制成黄金，如何把转化成油……
<br/>　　然后，几乎没有什么卵用。
<br/>　　毕生精力都消耗在上面，还没办法证明自己的理论，甚至连半个路标都没摸到。但到现在，他仍然坚信自己的方向是正确的。
<br/>　　当然，也不是半点收获没有。在攀登真理高峰的路上，也研究出来很多小玩意。
<br/>　　托尔·凯德利，一名年轻的药剂师，今年还不满三十岁，信仰死亡之神。
<br/>　　一名治病救人的药剂师，却信仰死神，单单这件事，就能让程大雷琢磨好几天。
<br/>　　他的信念是：万物皆有一死，你把越多的人送给死神，自己死后，就能在地狱收获更多的酬劳。
<br/>　　所以，他这个药剂师研究的药物不是为了治病救人，而是……送人去见死神。在这件事上，他矜矜业业，任劳任怨，简直就是死神在人间最忠诚的雇工。
<br/>　　这二人都把心血沉浸在研究上，所以不懂帝国语，充当翻译的是凯恩·道尔。
<br/>　　二人都有些科学家的傲慢，面对程大雷这个异教徒，态度也不如何好，只是板着一张脸坐在那里。通过凯恩了解到二人情况后，程大雷愣了好大一会儿。
<br/>　　奥黛琳娜麾下，也是人才济济呐。
<br/>　　他打点精神，道：“请二位过来，是想请教一件事，你们可不可以制造出一种防伪的纸，特点是成本低廉，易于分辨，能够长时间保存。”
<br/>　　希曼认真把程大雷的要求记下来，程大雷凑过眼睛看，也看不懂他写的是什么。
<br/>　　“我们需要回去研究一下，提出可行的方案，供阁下选择。”
<br/>　　程大雷赞许的点点头，这才是科学家的素质么，谨慎，认真。不会在没有把握的前提下，随便答应。
<br/>　　“那就拜托二位了，这件事不着急，你们可以慢慢研究。”
<br/>　　送走二人后，程大雷又差人叫来另外一个人。
<br/>　　冒险家西泽，21岁，信仰皎洁的明月女神，信念是用自己的双脚丈量大地，扯开船帆穿过海洋，然后画出这个世界第一份世界地图。
<br/>　　“你会绘图？”程大雷道。
<br/>　　西泽是懂帝国语的，他懂的语言还不止一种，甚至包括一些偏僻之地的方言。
<br/>　　“我懂地图，建筑图，还包括油画，谦虚的说，我是个出色的画家。”
<br/>　　程大雷撇撇嘴，文艺工作者和科技工作者说话风格就是不一样啊。
<br/>　　“那先试试看吧，我有幅建筑图，你帮我画出来。”
<br/>　　西泽将他的颜料铺开，程大雷按照脑海中的建筑图纸指点给他。西泽就像个打印机一般，完整的把图纸拓了出来。完成一座军策府，只用了一个时辰的时间。
<br/>　　程大雷啧啧称奇，这还真是个人才。等完成这件事，可要要他把整个琴川关的地图绘制一份，拥有一份精确的地图，自己打仗的时候，就可以少很多麻烦。
<br/>　　“来，你再帮我画个蛤蟆。”
<br/>　　……
<br/>　　送走了西泽，程大雷把凯恩留下了。这可是拥有隐藏属性的稀有人物，当然，目前为止，程大雷不知道他的隐藏属性有什么用。
<br/>　　凯恩的信念是周游世界，谱写出可以打动所有人的诗篇。
<br/>　　关键是凯恩这个人相比其他人很正常，程大雷和他交流也没什么困难。
<br/>　　“凯恩阁下，最近在这里还住得习惯么，生活上有没有遇到什么问题。”
<br/>　　“我们在这里收到热情的款待，一切都要感谢大当家的照顾，愿诸神永远祝福你。”
<br/>　　相比诺顿王国，帝国还算是繁华的，琴川关这里也可以称得上衣食无忧。如今的凯恩·道尔颇有种乐不思蜀的感觉。
<br/>　　程大雷也不和他客套了，问：“你们的圣女殿下最近在忙什么呢？”
<br/>　　“圣女……在忙着传教。”
<br/>　　……
<br/>　　奥黛琳娜同学的信念很小就确定下来，那就是替诸神传播信仰，让所有无知的恶人沐浴在神辉下。
<br/>　　也就是抱着这个念头，她才来到东方的，琴川关是她的第一站。这里的人以程大雷为首，又分外邪恶了一些。所以她的信念，就是先让诸神的光辉，照耀这片邪恶的土地。
<br/>　　所以在本职工作完成后，她的全部业余时间都用在传教这件事上。因为信奉诸神的缘故，她是一个格外善良的人，甚至善良的有些过分。不管什么身份，她都丝毫没有架子，而对于这个漂亮的小姑娘，大家也都是很乐意与她聊天的。
<br/>　　她第一个找到的人是秦蛮，在一篇长篇大论，从诸神起源扯到万物生长，奥黛琳娜发现秦蛮一直用一双木讷的眼睛看着她，表情没有任何变化。
<br/>　　“要不我就信仰你们了？”秦蛮试探着问，面对这个漂亮的小姑娘，他的心肠也难免软下来。
<br/>　　“……”奥黛琳娜。
<br/>　　她第二个找到的是高飞豹，高飞豹想了很大一会儿：“你们的教徒是不是不能成亲？”
<br/>　　“不，只有圣女是不可以结婚的，其他教徒都是很自由的。”
<br/>　　“那太可惜了，你长得这么漂亮，有没有想过不当圣女？”高飞豹认真问。
<br/>　　“……”奥黛琳娜。
<br/>　　奥黛琳娜同学再一次崩溃，这些人为什么这样，竟然可以无知的没有信仰活着。难道他们就不会恐惧么，难道他们在恐惧迷茫的时候，就不希望有诸神给他们指明道路。
<br/>　　如果换做一般人，此刻怕已经放弃了。经常和愚蠢打交道，自己很可能也会变得愚蠢。但奥黛琳娜是诸神在世间最坚定的信徒，没有什么可以动摇她心中的信念。生命、灵魂、肉体都是可以献祭给诸神的，这小小的打击就能让她退缩么。
<br/>　　不，这是不可能的，想也不要想。
<br/>　　然后她找到了刘悲。
<br/>　　这是她噩梦的开始，平生第一次，她对自己的信仰产生了动摇。
<br/>
