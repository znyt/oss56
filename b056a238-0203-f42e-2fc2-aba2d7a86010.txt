　　一个，两个，仙人球的数量在不断增加，送了四天之后，杨常林彻底瘫了，任凭那掌令如何叫也爬不起来。
<br/>　　这老灯心里憋屈的要命，但却一点办法也没有，想告状却找不到门路，庭掖局里最高长官就是掌令，连掌令都在坑他，他还能如何？
<br/>　　想出去？不得不说，你想多了，大唐皇宫可不是后世的故宫，可以随便走，想去哪里去哪。
<br/>　　大唐皇宫的另一个名字便是禁宫，禁是啥意思？就是没有令牌随便乱走杀无赦的意思！
<br/>　　多简单，多直接！
<br/>　　所以杨常林这老灯算是彻底没招了，趴在床上一个劲的哼哼，一个小时被四个仙人球曰，这特么神仙也受不了啊。
<br/>　　而那庭掖局掌令似乎也看出这老小子是撑不住了，索性便也不在管他，任他自生自灭。
<br/>　　……
<br/>　　幽州通往长安的火车上，东方亮，或者说钦陵惬意的靠在窗边，窗外微寒的风吹在脸上让人的精神不由微微一振。
<br/>　　林邑国主那个不争气的东西，竟然在兵力占优的情况下被老程带着两千人打的落花流水，真不知道是大唐军队的战力太强还是林邑的军队太弱。
<br/>　　多好的机会啊，如果那程妖精能再晚到一个月，崖州便可易手，到时候再努力一些说不定还可以拿下流球，正可以好好给李承乾的脸上来那么一巴掌。
<br/>　　想着一些有的没的，钦陵打开了随身的背包，从里面拿出一只烤鸡。
<br/>　　不得不说，这背包的确是好东西，也不知道李承乾是怎么想到的，钦陵一边想一边打开包着烤鸡的油纸，然后帅气的将鸡头扭了下来，一抬手……。
<br/>　　……
<br/>　　半晌之后，坐在钦陵对面的一个老头子指了指窗外：“小伙子，你刚刚好像是把鸡丢出去了！”
<br/>　　“……”对于多嘴的老头儿，钦陵什么都没有说，只是默默将鸡头放进嘴里仔细的啃着。
<br/>　　天知道刚刚脑子怎么就抽了，没事儿装毛的逼啊，这下好了，丢人丢到姥姥家去了。
<br/>　　但对面的老头儿好像并不像放过他，瞅他可怜兮兮的样子，不由叹了口气，随后说道：“小伙子，你是当兵的吧？”
<br/>　　“嗯！”钦陵没有抬头。
<br/>　　“我看你那包就知道，我儿子也有一个！”老头儿看着钦陵的眼神虽在有些怪，但却带着那么一丝慈爱，就好像在看自己的儿子。
<br/>　　“那您儿子呢？”钦陵被老头儿看了有些不自在，索性鸡头也不啃了，抬手丢到窗外，与老头儿聊了起来。
<br/>　　“去辽东啦，这不是大开发么，他本来已经退役了，不过为了响应号召便去了辽东。”老头看着窗外，就好像外面就是辽东，他这一望就能看到儿子一样。
<br/>　　“您只有一个儿子？”钦陵看老头儿身边并没有什么年轻人，不由问道。
<br/>　　“三个，有两个战死了，现在这个是老三，不过……也是个不让人省心的！”
<br/>　　“为什么要让他离开？难道他还能不听您的？”不知怎么的钦陵突然就来了谈兴，看着老头儿关心的问道。
<br/>　　“年轻人总有自己的事情要做不是，我老汉还能动，还用不着他给我养老，所以就让他去拼前途吧。”老头儿再次叹了口气，显然这话说的有些言不由衷。
<br/>　　钦陵默默的看着眼前的老头儿，突然发现这老头儿很像自己的老爹，离开家有些年头儿了吧？也不知道老头子在家里怎么样了？还在为大唐镇守边境不？又或者已经辞官了？
<br/>　　当年自己要出来，家里的老头儿好像也反对过，不过他还是出来了，为的只是赌一口气，估计当时老头子的想法应该与对面这老头儿是一样的吧？
<br/>　　想到这里钦陵摇头苦笑了一下，自己苦苦挣扎了几年，本想着给长安那个牛、、逼到不行的家伙惹些麻烦，但现在看来麻烦的确是搞了一些，但好像并没有造成什么不可挽回的后果。
<br/>　　也就是说这些年，他完全就是在瞎折腾。
<br/>　　而就在钦陵有些伤感的时候，对面的老头儿已经恢复了过来，笑着问道：“好啦，小伙子，咱不说我家那小子了，说说你吧，你这是要去长安？”
<br/>　　“嗯，去长安看看。”钦陵点头说道。
<br/>　　“哦，年轻人是应该到处走走。”老头儿说话的时候似乎顿了一下，在别人看来似乎并不算什么。
<br/>　　就这样，老头儿与钦陵有一搭无一搭的聊着，时间一点一点过去。
<br/>　　……
<br/>　　三天之后，或者应该说是两天三夜之后，火车终于到达长安站，对于老头儿和钦陵来说也都到了各自的终点。
<br/>　　互相笑着点了点头，在车厢的尽头分道扬镳，各自向自己的目的地而去。
<br/>　　但是，让钦陵意想不到的是，自己刚刚离开车站没多久，一队骑兵便从后面追了上来，将其堵在了路上。
<br/>　　“几位军爷有事？”钦陵眼中闪过一抹厉色，但很快被他掩饰下去。
<br/>　　“你从哪里来的？来长安所为何事？”那骑兵的头领从马上下来，上上下下打量着钦陵，查户口一般的问道。
<br/>　　“幽州，到长安访友，怎么？有问题么？”钦陵紧了紧背包，笑着问道。
<br/>　　“访什么友？住在什么地方？你又叫什么名字？”那骑兵头领并没有因为钦陵的笑而放松，反而更显紧张，握着刀柄的右手几乎都有青筋暴出来。
<br/>　　“我叫葛尔·钦陵，要找的人叫李承乾，住的地方以前在芙蓉园，现在应该在大明宫、东宫！”
<br/>　　卧槽！几个围着钦陵的大头兵全都吓了一跳，虽然他们不知道葛尔·钦陵是谁，但李承乾他们却知道，至于说大明宫，不好意思，他们只是听过并没有去过。
<br/>　　“怎么了？你们不是要抓我的么？是那个老头子对你们说我不是好人吧？”钦陵此时已经大致上猜到发生了什么事情，露出毫不在乎的表情说道。
<br/>　　这下反而让那几个骑兵不敢轻举妄动了，毕竟这家伙太过镇定，而且听名字好像也不是大唐人，这就更加让人更加不确定他人身份。
