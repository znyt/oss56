　　叶少阳看到了那八个坐在莲花上的人，四个僧人，另外四个，其中一个是玉玑子，另一个是个穿白衣的老者，不认识，还一个是个古装的少年，也不认识，最后一个……却是只有一抹黑色的虚影，勉强能看出是个人形。
<br/>　　看到他，叶少阳心头一惊，想到了一个名字：影魅！
<br/>　　虽然没见过真容，除了知道他是个影子，也不知道别的特征，但叶少阳心头有强烈的直觉：他就是影魅！
<br/>　　“叶少阳，三十年河东转河西，你也有今天，而今你深陷无色海中，老夫估摸着，以你这种嗜杀之人，是到不了婆娑净土的，看来你只能下地狱了……”
<br/>　　玉玑子一脸小人得志的表情。
<br/>　　小马听了这半天，怒道：“你个老棺材瓤子，你可是真是够磨叽的！有本事你进来找他干一架啊！你而今给人当狗，还真有脸了？”
<br/>　　一席话说得玉玑子脸上泛红，冷冷道：“你这狗贼也不要得意，你也是要下地狱的！到时候你就算哭着求老夫，老夫也不会多看你一眼！”
<br/>　　“我求你姥姥！你个生儿子没XX生女儿长XX的狗东西，你莫装逼，等老子待会出去，一砖头拍死你个老东西！”
<br/>　　小马骂人起来格外恶毒，玉玑子老脸通红，不过更加得意地笑起来，“骂骂骂，这就是你们捉鬼联盟的出息啊，以后下了地狱，有你们骂人的时候哈哈哈……”
<br/>　　两人这边打着嘴仗，叶少阳收回了那一缕魂魄，继续调息，耳边突然响起一个声音：“叶少阳，你中了我这法令虫，便如附骨之疽，没有些许工夫，又如何能够清除！”
<br/>　　叶少阳心中一惊，睁开眼睛，朝对面坐在莲叶上那个黑色的影子看去。
<br/>　　“不错，是我。叶少阳，我之前跟了你许久，的确是难得之才，但三界之内，难以证道，你若归顺我门下，我这便带你去轩辕山……别忙着拒绝，我只说一遍，你也仔细想清楚。”
<br/>　　“我为轩辕弟子，千年不染尘身，在轩辕山开宗立派，建有洞府，那里没有三界众生之惘，没有轮回颠倒之苦，是真正的极乐净土……叶少阳，你若随我同去，在轩辕山上，是开宗立派还是潜心隐修，都随你。只要你叫我一声师父，其余不管你。”
<br/>　　“你若从我，便跟着念咒，若不从我，便万念俱灭……”影魅的声音逐渐减弱消失。
<br/>　　念咒？
<br/>　　叶少阳不明白这句话的意思。
<br/>　　突然，坐在另一片莲叶上的青年睁开眼睛，朗声说道：“诸位，我乃星月奴门下大弟子鲲鹏祖师，我身边这四位，乃是盘古高僧，今日合力将无色海水引来，超度你等，凡入婆娑净土，必先渡无色海，湮灭欲望，方至彼岸，今日也算你等大有机缘，我轩辕山开方便之门，只需跟我念咒，便可证始终！”
<br/>　　本来哇哇大叫的这些生灵，听了鲲鹏的话，一个个都安静下来，面面相觑，其中一人壮着胆子问道：“我们这些微末之人，怎么也有机会前往婆娑净土，祖师你……你可不要诓我们，如果真是机缘，又为什么要用法阵将我们罩住？”
<br/>　　鲲鹏看也不看那人，不耐烦地说道：“实话说，今日我等是来擒拿叶少阳，要渡化他，你等只是沾了便宜！这法阵也是为了限制你等，若跟我念咒，便可证始终，若不追从，便堕入无色海中，永不超生！”
<br/>　　大伙吓的不敢出声。
<br/>　　又有一人说道：“婆娑净土不是小圆满的佛门仙境么，需要大机缘大觉悟，或可参悟因果，才有机会一朝飞升，如果只是跟着念咒就去了，那……未免也太简单了吧。”
<br/>　　鲲鹏道：“佛门婆娑净土，就在轩辕山！”
<br/>　　众人大骇。
<br/>　　“这……怎么可能？”
<br/>　　“轩辕神山，跳出三界之内，不在五行之中，为轩辕上帝开创，乃是真正的神界，一干修士，无欲无求，不生不灭，盘古神僧，日日讲经，雷池濯身，万劫不灭，岂不远远超过人间洞天福地，不是婆娑净土，又是什么？”
<br/>　　小马听到这里，朝鲲鹏骂道：“什么婆娑净土，我算是听明白了，就是给你们轩辕山当小弟呗，说得那么高大上，你不过也就是星月奴的一条狗！”
<br/>　　鲲鹏望着他冷笑，“轩辕山乃三界主宰，顺天者昌，逆天者亡！”
<br/>　　这句话每个字都像是一次重击，敲打在众人心头。
<br/>　　这就是绝对的实力，轩辕山的超然地位，那些传说中的神祇，也多半都飞升到了轩辕山，这一点跟空界很相似，因此对于人间法术界来说，轩辕山，和青冥界的空界，多是神居住的地方——虽然像叶少阳这样的了解真相的知道他们并不是神，但毕竟绝大多数的生灵来说，永远没有机会接触到这个层面，对他们来说，空界和轩辕山，就是神界一样。
<br/>　　神，其实也是人，只是因为做了人做不到的人，所以被供奉为神。
<br/>　　因此，在法术界有一个说法，把轩辕山和空界并称为神界，但不同的是，在轩辕山有地位能立山头的，都是最早那一批跟着轩辕上帝飞升过去的大咖，或者这些大咖的传承弟子，原则上不会接受新人成神。
<br/>　　而青冥界，却是接受历朝历代的飞升者，因此抛开实力不谈，轩辕山在三界之内的地位，一向比空界要高班头，而且相对空界，人数也不到几十分之一，更没有那些拉帮结派的战斗，因此在三界众生的眼中，以轩辕山为天，以空界为地。
<br/>　　鲲鹏这么说，虽然自大到了极点，但也不是没有根据的。
<br/>　　听到这番话，场面一时间陷入死寂，很长时间没人再开口说话。
<br/>　　“叶少阳！”
<br/>　　鲲鹏高叫一声，“你若交出山海印，我便放过你这几位兄弟，不然，你们就全都堕入海底，万劫不复！”
<br/>　　说完，大喝一声：“起阵！”
<br/>　　四个僧人一起作法，原本平静的水面，突然翻起了惊涛骇浪，从四个方向朝中心卷过来，
<br/>(本章完)
