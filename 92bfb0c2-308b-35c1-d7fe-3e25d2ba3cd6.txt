　　第144章
<br/>　　城主府。
<br/>　　方伯山屏退左右，与齐德强相对而坐。
<br/>　　“齐将想要我放了程大雷？”
<br/>　　齐德强抿口茶：“不是我，是主公，现在程大雷是主公的人，怎么能让你杀了。”
<br/>　　“但抓住这程大雷可是极其的不容易，他为祸青牛山已经很久，如今已经是青牛山头一号山贼，我这次抓住他也是颇费了一般力气。这次要是放了，下次再想抓就不容易了。”
<br/>　　齐德强微微迟疑，攻打黑石城仍是计划中的事情，让程大雷做炮灰的事情属于机密，也不能泄漏给方伯山。如此，自己也只好做场戏了。
<br/>　　“抓，为何要抓！那程大雷早有向我主投诚之意，忠心可昭，我主爱惜他是个人才，不然怎会封他游骑将军，他如今是主公的人，主公的人你也要抓！”
<br/>　　方伯山吓了一跳：“幽州王竟如此看重程大雷这个山贼？”
<br/>　　“这是自然，还有不要叫山贼，要称呼程将军。”齐德强又端起茶杯：“在牢里，你没对他用刑吧？”
<br/>　　方伯山一激灵，冷汗就渗了出来。
<br/>　　他本无意杀程大雷，本来还有意将程大雷上交给幽州王，领一份赏赐。因为李如来过几次，所以方伯山专门嘱咐手下人照顾照顾牢里的程大雷。当然方伯山口中的『照顾』，就不是字面意思上的照顾了。
<br/>　　以自己手下人的脾性，程大雷现在就算不死，怕也得褪层皮，断胳膊断腿都算是自己手下人积德。
<br/>　　一看方伯山脸上的表情，齐德强就明白了，大牢里什么情形，他怎么能不懂。
<br/>　　“还不快带我去！”
<br/>　　程大雷死不死的其实对幽州王并不重要，但程大雷活着对幽州王很重要。攻打黑石城，青牛山是必经之路，真要程大雷出现个一差二错，青牛山的山贼发了疯，能不能解决先不提，毕竟麻烦不是。
<br/>　　程寨主，希望你生命力顽强些，千万要活着，至少……得到能抢救的地步啊。
<br/>　　齐德强脚步如飞，一路上都忧心忡忡，他从未像今日般担心程大雷的身体状况。
<br/>　　走到大牢门前，齐德强深吸一口气，竟有几分不敢踏进大牢。
<br/>　　“方大人，你确定他还活着么？”
<br/>　　“这个……应该会活着吧。”
<br/>　　是福不是祸，是祸躲不过，齐德强稳了稳心神，大步踏进牢饭。
<br/>　　“齐将，你别太担心，我手下的人我是知道的，手上应该……大概……有轻重吧。”方伯山跟在齐德强身后，其实也没有多少底气。
<br/>　　“那杜老大算得什么，他怎么是我一合之将，要真厉害，还是那个戎族人厉害，是能在我马下走三五个回合的。”
<br/>　　刚走进牢房，就听见一个中气十足的声音在牢内回荡。
<br/>　　“程寨主，程寨主，再给我们讲讲五雷法的事情吧，听说您是在梦中被神仙传艺？”
<br/>　　“哈，这个么，神仙的事情跟你们说了你们懂么。那五雷法威力太大，我轻易是不会用的，杀孽太重。对啦，今天吃什么？”
<br/>　　“禀程爷，咱吃烧肘子和红烧鲟鱼……”
<br/>　　“又吃鱼啊。咱就不能吃点清淡的，不过说到鱼呢，我都是知道一种生鱼片的吃法，配上佐料蘸着吃……却也是别有一番风味。”
<br/>　　当方伯山和齐德强走到牢房门口时，就看到这一幕，一时脸上的表情都愣住了。
<br/>　　程大雷看到齐德强时，深深抽了一下鼻子：“老齐，你可来了，这些日子，我可是受苦了！”
<br/>　　“……”齐德强。
<br/>　　你受苦个香蕉芭拉，你每天吃得伙食比我还好。齐德强分明看到，程大雷比几天前见面时都胖了点。瞧瞧这牢房的摆设，打扫得干干净净，靠墙摆着一只木床，别的犯人睡稻草，你床上却有棉被。碗里还有没啃干净的鸡骨头，床头还摆着几本书。
<br/>　　啧，真没想到，响当当的程寨主还是个读书人呐！
<br/>　　齐德强深吸一口气，只希望陪你表演的我你不要视而不见。
<br/>　　“程寨主，你受委屈啦！”
<br/>　　方伯山也被这一幕震住，他悄悄将牢头，低声道：“这是怎么回事？”
<br/>　　牢头也没想到方伯山会突然赶到，他愣了愣：“不是大人，让我们好好照顾他的么……”
<br/>　　『照顾，我说的照顾是这个意思么！你们照顾得……也他妈太好了。』
<br/>　　方伯山转个念头，也就知道手下拿了程大雷的钱。真丢人啊，不过此时也不是说这个的时候。
<br/>　　看程大雷和齐德强现在的情形，已经快到隔着牢门相拥而泣的地步。原来程大雷不仅是幽州王看重的，和齐德强的交情还不浅。
<br/>　　看来，也是自己表演的时候了，拼演技自己也是实力派好不好！
<br/>　　他深吸一口气，看着程大雷，撩起长袍大步走过去。
<br/>　　“程贤弟在哪里，我的程贤弟在哪里，程贤弟，一时疏忽啊，让程贤弟受委屈了！”
<br/>　　程大雷愣了愣：“这位是？”
<br/>　　齐德强道：“这位就是本地城主方伯山。”
<br/>　　“我们以前见过么？”程大雷。
<br/>　　“虽未谋面，慕名良久，我与程贤弟是神交呐。”方伯山深吸一口气，冲牢头吼道：“还愣着干什么，还不快把牢门打开！”
<br/>　　齐德强将程大雷接出牢房：“程寨主，幽州王已经将你封为游骑左将军，官服和军印都在城主府，还要程寨主和我走一趟。况且程寨主从这种地方出来，也该洗一洗身上的晦气。”
<br/>　　果然和程大雷所料的不差，幽州王最终还是放弃了熊老大，选择了自己。得了这『左将军』，自己就可以在幽州一地横着走。当然，等幽州王用不着自己时，这左将军就是个……屁。
<br/>　　当程大雷在城主府梳洗的时候，方伯山将手下人都召集到一起，阴沉着脸看着他们。
<br/>　　“都说说吧，拿了人家多少银子，我难得糊涂，你们也别以为我真糊涂。吃了不吐出来，很多事咱们就好好聊聊了。”
<br/>　　手下面面相觑，大家此刻都想起，自己这位城主平时看上去好像不太聪明，但在银子的事情上：他比任何人都精明。
<br/>　　
<br/>(本章完)
