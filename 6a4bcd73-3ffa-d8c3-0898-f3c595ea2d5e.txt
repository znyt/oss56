
<br/>　　<br/>　　    隔天，王惊蛰被陈臣接到了总队医院，他妈回来两天了人还在特护病房里躺着，生命体表特征都正常，但唯独人就是不醒，而且比回来那天又消瘦了不少，看着就跟人好想要被活生生的给耗死了一样。<br/>　　<br/>　　    似乎是对王惊蛰很托底，陈臣尽管挺担忧的，可表情还算是很放松的，人与人之间的信任和依赖感一旦要诞生起来的话，对方就会把所有的希望全都寄托在你身上，并且还是一百二十个信任和放心，明显陈臣此时就是这种状态了。<br/>　　<br/>　　    王惊蛰到病房的时候，里面只有陈重和护工还有助理在，对方朝着他疲惫的点了下头，说了声来了之后就拍了下他的肩膀：“一切你看着办，麻烦了”<br/>　　<br/>　　    王惊蛰“嗯”了一声，来到病床旁低头打量着陈夫人，人这时候瘦的都有点脱相了，眼眶子深陷进去把颧骨都给凸显了出来，嘴唇青紫没有血色，脸上苍白的很，耷拉在被子外面的手臂上都是清晰的血管，皮肤暗淡一点光泽都没有，别人看不见但王惊蛰能瞧见的是，人的身体外浮着一层淡淡的黑气。<br/>　　<br/>　　    医院方面现在只能够给挂葡萄糖，除此以外什么方案都拿不出来，完全就是干瞪眼的看着，无计可施了。<br/>　　<br/>　　    “先出院吧，回家就行了，在这里不其实不太适合”医院是阴气最重的地方，陈臣她妈本来就是被阴魂缠身，在这种地方不但起不到任何的作用，而且还一点好处都没有。<br/>　　<br/>　　    陈重愣了下，然后说了声好，就马上吩咐开始把他老婆转移出医院，直接叫了一辆救护车往家里送。<br/>　　<br/>　　    医院方面负责陈臣她妈病情的主任医师听说人要出院，就急忙赶了过来，劝慰着说道：“陈先生，医院今天已经准备了个会诊，由内科和精神科等多名专家组成的，我们觉得晚间差不多就能拿出一个方案了，还有我们还打算连线协和医院的专家远程视频旁听，甚至如果旁听过后他们那边有什么方案的话明天就可以赶过来，但是病人现在这个情形出院太危险了”<br/>　　<br/>　　    “我再给你一天或者两天的时间，你能百分之百的让人醒过来，恢复正常么？”<br/>　　<br/>　　    主人直接摇头说道：“不可能，我们怀疑陈夫人是得了一种慢性病，这个诊断和治疗的过程是很漫长的，得需要做出长期医治的准备，还有……医学上的问题没有谁是百分之百可以做出肯定回答的，哪怕就是医生在面对一个普通的发烧感冒状况也是不可能的”<br/>　　<br/>　　    陈重顿时伸手指着主人的胸膛说道：“你不能，那我能，过两天我亲自让我夫人来医院对你们表示感谢，感谢你们这两天给她吊的盐水”<br/>　　<br/>　　    陈重说完，直接吩咐人把陈臣她妈抬到担架上，然后准备送上救护车回去，医院的主任被他一句话就给怼懵了，半天后才连连摇头说道：“这不可能，这是完全不存在的”<br/>　　<br/>　　    王惊蛰在旁边也没吭声，医生的判断和诊断其实没毛病，不过就是不对症而已，有些问题已经明显脱离医疗范畴了。<br/>　　<br/>　　    病人上了救护车后，王惊蛰跟陈重又吩咐了几句，让他准备一些东西送到家里去。<br/>　　<br/>　　    半个多小时后，回到陈家，陈臣她妈被推到了卧室中，王惊蛰跟陈重说让别人都走吧，有些事最好还是别让人知晓的好。<br/>　　<br/>　　    人都走了后，王惊蛰独自一人进了卧室，让陈臣和陈重在外面等着。<br/>　　<br/>　　    “哗啦”王惊蛰拉上窗帘关上房门，屋内顿时漆黑一片，只有点微光透了过来。<br/>　　<br/>　　    人现在还昏迷不醒，日渐消瘦，明显就是被阴魂给缠的，耗掉了她体内的生气，三魂七魄里的胎光和伏矢被惊走了，胎光主神，中医上说如果胎光远走不归的话，那其人就是慢慢的等死了，伏矢主意识，伏矢不在就人仿若行尸走肉一般，全无知觉跟活死人差不了多少了。<br/>　　<br/>　　    所以有些人生病或者有顽疾，西医是完全看不出个一二三来的，那就是因为三魂七魄中不知哪一魂或者哪一魄给丢了，只能给归结为是神经系统或者精神方面的原因，反倒是这时候我们的中医就能闪亮登场了，只是可惜了，到如今这年代有太多人把中医的疗效给淡化了，甚至还抱着完全不信任的态度。<br/>　　<br/>　　    但凡是那些能被称为国手的中医，可能光凭着几根银针或者几副汤药就能解决那些最权威西医都无法解决的病情。<br/>　　<br/>　　    王惊蛰伸手抬着陈臣她妈给放到了地上，然后拿起两根蜡烛放到了她肩头两边。<br/>　　<br/>　　    “噗”<br/>　　<br/>　　    “噗”<br/>　　<br/>　　    两根蜡烛点起来后，本来屋中就安静的很，一点风都没有，但两盏烛火点起来后就冒出了蓝光，然后就无风自动的摇摆起来，仿佛随时都要熄灭一样。<br/>　　<br/>　　    他这么做就跟传说中的鬼吹灯差不多是一个意思，如若屋中有东西的话，这烛火肯定顿时就熄了。<br/>　　<br/>　　    在蜡烛还没有熄灭之前，王惊蛰拿出两张符纸，用笔沾着朱砂迅速在纸上写了两道符箓，卷到了蜡烛上，上面的烛火顿时不在摇曳了，蜡油顺着蜡烛流到了符纸上，给粘到了一起。<br/>　　<br/>　　    这时陈夫人身上那成淡淡的黑气有些翻滚躁动起来，王惊蛰瞥了一眼也没管，接了一碗清水之后，伸手将蜡烛上沾着的两张符纸又给掀了下来，递到烛火上烧成纸灰然后全都放进了碗中，用手指搅合均匀了，就捏着陈臣她妈的嘴，直接把一碗混了纸灰的清水全都给灌了进去。<br/>　　<br/>　　    一碗水全都一滴没落的进了陈夫人的肚子里，她肩膀的两盏蜡烛顿时火苗就蹿了起来，然后朝着她两肩头缓缓的凑了过去。<br/>　　<br/>　　    就那么一瞬间，陈夫人的脸色忽然转红了不少，身上的阴气也渐渐的少了一些，人的肩头都烧着两簇阳火，这阳火完全是被阴魂给压住了，王惊蛰是让陈夫人肩头本来要灭了的阳火又重新被烧得旺盛了起来。<br/>　　<br/>　　    片刻后，那两根蜡烛要烧到底的时候，王惊蛰忽然划破手指将自己的两滴精血滴到了上面。<br/>　　<br/>　　    “轰！”烛火顿时剧烈的燃烧起来，火苗直接蹿到了棚顶上。
