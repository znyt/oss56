　　“刚才，谢谢你能让我们进来，这说明你还是想要保护我们的。”白小飞停顿了一下，用请求的语气继续说道：“既然你有这么厉害的武器，我们现在需要你的帮助。”
<br/>　　侧着头，用眼角余光看着众人，对于白小飞的请求，保安并没有接话，而是继续往下听着。
<br/>　　“这位小朋友的家，就在前面那栋楼里，他的妈妈很有可能是被楼上的尸兄给困在上面了，光凭我们几个人，想冲上去并不容易，所以我希望你，能帮我一起找他的妈妈。”
<br/>　　微微摇了摇头，保安小哥并没有答应，和这些人非亲非故，他不想冒这个险。
<br/>　　能把这些陌生人给放进超市，就已经让他下了很大决心了。
<br/>　　果然不愿意么……
<br/>　　眼神有些无奈，白小飞摊了摊手，也没有强求，毕竟，这是别人自己的选择。
<br/>　　像这种情况，大部分人，肯定也是想要在一个安全的地方呆着吧。
<br/>　　“叔叔……”
<br/>　　清脆的童音响起，小鹿藏在刘哔身后，露出半个脑袋，有些怯怯的开口。
<br/>　　“我们什么时候能回家啊，小鹿到家了，小鹿要回家，小鹿想妈妈。”
<br/>　　听见小鹿像是要哭出来一般的声音，众人都是有些心酸。
<br/>　　“小鹿乖，大哥哥带你回家。”刘哔蹲下身体，替小鹿整了整衣领：“晓佳，小惠，你们留下来，在这里照看小胖，小飞，我们带着小鹿，一起上去看看。”
<br/>　　犹豫了一下，白小飞没有马上答应下来，他怕几个人留在这里，会有什么危险。
<br/>　　毕竟，这边还有一个带枪的陌生人在，虽然之前救了他们，但对于这个保安谁都没有了解。
<br/>　　万一对方要是起了歹心，那小胖他们可就危险了。
<br/>　　“放心去吧，胖爷虽然受伤了，但战斗力还是有的，这边，就放心交给我吧。”小胖脸上露出灿烂的笑容，像是扯到了伤口，呲了下牙：“我一定会保护好她们周全的。”
<br/>　　仔细想了一下，白小飞也同意了下来，虽然在他原本的打算中，是要明天一早去探查大厦的。
<br/>　　但时间拖得越久，情况对他们来说就越不利，谁知道小鹿的妈妈有没有被感染，如果晚去一会，或许就会造成不可逆转的损失。
<br/>　　“我陪你们去。”不知道想到了什么，保安小哥脸上的肌肉微微抽动了几下，突然开口。
<br/>　　这意料之外的事，反倒将白小飞搞得微微一愣，在他原本的认知中，保安小哥是不会和他们一起前去的。
<br/>　　不过这样也好，不光是多了一个人的力量，而超市这边的隐患，也消除了不少。
<br/>　　“好啊哥们，话说，你一直不说话，我还以为你是哑巴呢。”哈哈一笑，白小飞随口调侃了一句，然后抓起在大桥上带过来的那炳长刀。
<br/>　　这是之前那个血莲教头目的武器，在他逃跑的时候，顺手就将其带过来了。
<br/>　　正好缺一把顺手的武器，这个相当合适。
<br/>　　“等我一下。”保安小哥惜字如金，但动作却是麻利，很快找出一套部队标配的行军装备，穿戴到了自己身上。
<br/>　　看着对方娴熟的动作，白小飞嘴巴微微动了动，眼中有些惊讶。
<br/>　　“之前有部队在这里被袭击了，留下了很多武器，我趁着天黑拿进来很多，用来防身的。”
<br/>　　看到众人眼中的不解，保安简单解释了几句，众人方才露出恍然大悟的表情。
<br/>　　“哈，哥们，这也太专业了吧。”白小飞挠了挠头，讪笑一声：“不过你穿戴这些装备的动作，流畅的并不像是一个保安。”
<br/>　　“我以前当过兵。”
<br/>　　只是解释了这么一句，保安小哥便不再言语，而是跟在众人身后，等待着出发。
<br/>　　片刻，三人都检查完身上的武器和物品之后，白小飞挥了挥手，带头走出了便利店。
<br/>　　外面已经接近午夜，天色无比黑暗，隐约间，传来尸兄恐怖的嘶吼声，唯有远处的大楼里，闪烁着一些并不起眼的灯光。
<br/>　　几人小心翼翼的，向远处的大厦靠去。
<br/>　　因为视野问题，路上虽然已经足够小心，但还是不可避免的碰上了尸兄，甚至期间还引来了一小股尸群，不过很快就被众人解决了。
<br/>　　“快走，刚刚的枪声，没准会引来什么东西。”
<br/>　　刘哔一只手牵着小鹿，微微加快了自己的动作，在这样空旷且尸兄密集的地界开枪，并不是一件明智的选择。
<br/>　　其余两人明显也意识到了这一点，根本不用提醒，自顾加快了自己的动作。
<br/>　　全速前进之下，一行人很快就赶到了大厦门前。
<br/>　　所幸的是，大厦内部还没有断电，只是很多地方，电灯收到了破坏，所以整体才会显得昏暗。
<br/>　　但借助这些光亮，基本情况还是能大概看清的。
<br/>　　似乎是被刚才的枪声吸引过去了，傍晚从远处看见的那些尸群，现在已经没有了踪迹，但还是有零散几只行动不便的，在地上疯狂挪动着。
<br/>　　近在咫尺的血肉味道，让它们发狂。
<br/>　　最引人注目的，是一个头发几乎都要掉光了的幼年尸兄，之所以说引人注目，是因为这个尸兄的脑袋，几乎比身子都要大了。
<br/>　　和其余尸兄不同，对于眼前的人类，大头尸兄并没有表现出太大的进攻欲望。
<br/>　　“这脑袋，X鹿喝多了吧。”嘀咕一声，白小飞双手握刀，将大头尸兄斩成了两截。
<br/>　　强化人的体质，再配上这样锋利的武器，这些普通的尸兄，对他根本不会有什么威胁。
<br/>　　唯一吐槽的是，明明是一柄双手长刀，不知道为何却被白小飞用成了斩马刀的样子。
<br/>　　“走吧，我们要赶快……”
<br/>　　脸上带着笑容，白小飞刚想踏进大楼，话语却突然顿住，脸上的笑容，也瞬间凝固。
<br/>　　刘哔心中也升起了一丝戒备，但他和白小飞的关注点，并不不同。
<br/>　　白小飞是看向楼内，那里有一大群变异蚊子，这刘哔心中清楚。
<br/>　　但他在意的，是刚刚看似被斩杀掉的大头娃娃。
<br/>　　至少，在他进入这个世界的这段时间里，第一次感觉如此危险。
