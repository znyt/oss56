　　李承乾也不知道自己到底算不算是乌鸦嘴，正想着别出人命，别出人命，就看到有几个禁军架着两个内侍软塌塌的身体往外拖，看那两个内侍低垂的脑袋，显然是不活了。
<br/>　　什么情况这是？真的搞出人命了？李承乾心中狐疑，抬手将那几个禁军拦了下来：“怎么回事儿？”
<br/>　　“太……，陛下！”几个禁军见到李承乾，习惯性的想叫太子，但很快反应过来，眼前这位经过了登基大典已经是皇帝的身份，连忙改口。
<br/>　　“说，这两个人是怎么回事儿。”李承乾的注意力显然不在称呼上面，看着那两个一直没有反应的内侍，沉声追问。
<br/>　　“这两人，这两人是长公主殿下下令杖毙的，具体原因臣也不知。”见李承乾追问，那禁军头领连忙说道。
<br/>　　宫里当差规矩多，但最基本的一点就是不要乱说话，所以这禁军头领只是说了这两个内侍的死因，其它完全就是一问三不知，甚至就连这两个内侍是谁的人都不清楚。
<br/>　　不过这对李承乾来说已经够了，只见他轻轻点了点头，便摆手让那一队禁军离开，脸上甚至连一丝喜怒都看不出来。
<br/>　　登基大典的时候他的几个大小老婆是有参加的，而且长乐她们也都是在，只不过后来在在大殿上接受群臣朝拜的时候，这些女人全都回宫了而已。
<br/>　　可是，就是这么短短的一个时辰，竟然便有两条人命报销在了长乐的手上，这让李承乾的不得不感慨这丫头果然是皇家贵女，纵然是再心地善良，也改不了暴龙的本质。
<br/>　　……
<br/>　　一路无话，李承乾回到属于自己的东宫内宫，刚一进院子便看到正在训斥众人的长乐。
<br/>　　这见这小丫头在一身华丽宫装的衬托下显的贵气逼人，就连有着东宫之主美誉的苏玫在这丫头面前都要弱上半分。
<br/>　　而且重要的是，这丫头手里竟然还拿着长孙皇后训诫后宫用的训尺，这东西在后宫里面几乎的威摄力几乎可以与李二手中的‘天子剑’相提并论，不管谁拿在手里，都可以代表长孙皇后来训导后宫。
<br/>　　“你看看你们，你们一个个成什么样子！大哥刚刚登基，你们便闹的不可开交，怎么，真当自己是后宫第一人了？以你们现在的样子，成何提统，简直丢尽了我皇家的脸面。”
<br/>　　“武媚儿，刚刚那两个内侍是你的人吧？两个小小的内侍竟然妄议皇后人选，这就是你平日里对他们的教导？还是你别有用心？”
<br/>　　“苏玫，你不要以为自己身为太子妃，就一定会是皇后的人选，本宫可以明确的告诉你，这皇后可不是想当就能当的，没有母仪天下的胸襟，就不要指望这个位置。”
<br/>　　“程琳，你也一样，不要以为有卢国公的势力你就可以在宫里为所欲为，宫里有宫里的规矩，若是犯了规矩，就是大哥，也保不了你。”
<br/>　　“还有你，孔雯，不要以为你的小动作大家都觉查不到，林晓晓前几天为什么会病倒相必你的心中十分清楚，若有下次你自己掂量着办。”
<br/>　　“……”随着长乐一个个的点名，苏玫等几女全都低下了头，她们并不怕长乐，但是却怕长乐手里的东西。
<br/>　　而且长乐说的那些并不是没有根据，只是为了顾全面子，并没有全都说出来而已，所以几女自然是不敢分辨。
<br/>　　李承乾躲在院子门口把所有的话都听了个一清二楚，长长叹了口气靠到院墙之上，苦笑着摇了摇头。
<br/>　　他知道这个时候他不应该进去，如果进去了反而会让事情变的难以处理。
<br/>　　可是，可是他万万没有想到，这才几天而已，自己的后宫竟然闹腾出了这么多的事情，尤其是林晓晓的事，如果今天暗中听到长乐所说，他还不知道里面竟然有孔雯的手笔。
<br/>　　这特么叫什么事儿啊，亏自己以前还觉得这几个女人在一起相处的挺好，毕竟都是小时候一起长大的，现在看来随着年龄的增长，就是再亲密的小伙伴再铁的友谊，在权利面前也特么会翻船。
<br/>　　“陛下，您……还进去么？”夜魅依旧是李承乾的贴身护卫，不过相比于一边听的津津有味的小白，她的心思明显在这位年轻的皇帝身上。
<br/>　　“不进了！我们四处走走吧！”李承乾犹豫了一下，最后还是摇了摇头，站直了身体向另一个方向走去。
<br/>　　夜魅没有说话，只是看着李承乾背影的目光中多了一些怅然。
<br/>　　得到了天下，却失去了原本应该有的东西，不得不说，这与李承乾最开始的想法是相违被的。
<br/>　　在那个不起眼的小凉亭里，李承乾的对老头子说的那些话其实百分之九十九都是真心话，因为他当时并不知道老头子只是吓唬吓唬他。
<br/>　　所以凭心而论，李承乾最在乎的还是亲情，而皇位最多被他看成一种工作，有多少人会为了工作而放弃亲情？应该没有吧？
<br/>　　不管怎么说，李承乾到底还是经历过现代社会，如果说让他为了皇位，作到冷酷无情，对自己的老婆和孩子视为工具的话，他真的做不到。
<br/>　　可是现在做不到又能如何？几个老婆已经开始为了那个皇后的位置争个不休，甚至现在连长乐都掺和了进来，而且看她手中的训尺估计老妈也知道了这件事情。
<br/>　　怎么办？到底要怎么办？难道真的要走上那条老路了么？让小武上位？然后看着大唐在她的手中分崩离析？
<br/>　　不，这绝对不行，大唐绝不能继续走上那条老路，也不应该走上那么老路。
<br/>　　如果自己来了一次大唐，结果大唐还是重新回到历史的轨迹上，那自己如何对得起自己这一生。
<br/>　　必须想办法分散这几个老娘们儿的注意力，让她们把目光从皇后的那位置上移开，最好是让那几个败家老娘们儿都排斥那个位置才好。
<br/>　　可是有什么办法呢？到底有什么办法可以想呢？李承乾只觉得自己的脑子都快要爆开了。
