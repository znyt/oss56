　　飞行类亚圣器，那艘巨船上。
<br/>　　萧逸一边盘膝坐下，替剑姬前辈治疗着手臂上那道破魂针留下的伤口。
<br/>　　一边，扫视着周遭一道道熟悉的身影。
<br/>　　林劲、铁牛、秦飞扬、上官曦月、凌羽、叶铭等等…
<br/>　　这里的所有人，他都认识。
<br/>　　除却剑姬前辈是他的前辈长辈外，其余的，尽皆是与他一同经历过生死的好友。
<br/>　　林劲等人，更是他当年成长路上的挚友。
<br/>　　这些身影，面容都没有太大变化。
<br/>　　仅仅是对比以往时的稚嫩，现今一个个成熟了不少。
<br/>　　众人虽个个身上都带着血迹，但，萧逸感知过他们的气息，已然平稳。
<br/>　　以他的炼药师本事，还有丹药，帮他们疗伤并非难事。
<br/>　　从一开始远远御空飞来，见到剑姬前辈还有一众好友，他先是惊喜，随后是看到众人个个带伤下的震怒，直到现在，则是微微平伏了心情。
<br/>　　众人无碍，他不必担心。
<br/>　　接下来，只是问问为何剑姬前辈还有大伙会来了中域的事了。
<br/>　　他毕竟深知东域那边缺失了一丝天地武道，任何人都无法突破天极境。
<br/>　　而他也毕竟自己亲身穿越过那片空间乱流，知晓里头的危险。
<br/>　　若无天极境实力，根本不可能穿过那片乱流，来到中域。
<br/>　　“剑姬前辈，你们怎么会来这里的？”
<br/>　　萧逸一边替剑姬前辈治疗着伤势，一边传音问道。
<br/>　　东域那边的事，自然不好明说，故只能传音交谈。
<br/>　　萧逸谨慎下，无声无息地布下一个屏障。
<br/>　　屏障，与一旁的紫电神剑气息相连。
<br/>　　萧逸甚至布了一个阵法，以保万全。
<br/>　　这般层层阻隔下，又是传音交谈，即便是承风殿主，也休想窥探。
<br/>　　即便承风殿主打算强行窥探，萧逸也会瞬间发觉。
<br/>　　不是信不过承风殿主，而是，有些事多一事不如少一事。
<br/>　　剑姬前辈，显然也知晓萧逸的担忧，便也传音回答。
<br/>　　“我在东域突破天极境了，便过来了。”
<br/>　　“过空间乱流时，虽未殒命，但也确实是历经波折。”
<br/>　　“可想而知你小子当年孤身一人，且只有区区地极境便出发，何等艰险。”
<br/>　　边说着，剑姬前辈笑笑，面露赞赏之色。
<br/>　　“突破天极了？”萧逸闻言，微微一惊。
<br/>　　剑姬前辈，继续传音回答。
<br/>　　半晌，剑姬前辈大致解释了一遍，萧逸也大概明白了过来。
<br/>　　知晓剑姬前辈在东域那边，天地武道缺失了一丝的情况下，还能一举突破天极，萧逸着实是惊讶。
<br/>　　但同时，也心头暗暗惊叹剑姬前辈的天赋，果然惊人。
<br/>　　想起来，当年在东域时，剑姬前辈便是历史上最快达到极境之人。
<br/>　　达到极境时，年纪23岁。
<br/>　　而后来，即便是自己达到极境时，虽比剑姬前辈更快，但也快不了多少，顶多半年多些，不足一年。
<br/>　　可想而知剑姬前辈天赋之惊人。
<br/>　　而另外，剑姬前辈在炎武主殿的这艘亚圣器帮助下，其实早便带着大伙从东域赶来中域了。
<br/>　　算算时间，大概是他当年刚离开不足两年，剑姬前辈一行人便已出发了。
<br/>　　所以剑姬前辈等人，其实早就来中域数年了。
<br/>　　不过，中域这边，处处危险，亦处处杀戮。
<br/>　　剑姬前辈虽性格有些胡闹，但她这等天骄，自然非泛泛之辈。
<br/>　　所以来了之后，一直都低调行事。
<br/>　　毕竟她除了自己以外，还要照顾一大群东域天骄。
<br/>　　她们甚至没有加入八殿任何一殿，只是独自闯荡、历练，以尽快提升修为。
<br/>　　故她们在中域之上，没有传出什么太大的声名，也没有闹出什么太大的麻烦事。
<br/>　　自然的，偌大个中域，萧逸也不知晓他们来了。
<br/>　　若非此次惹了莫忧，闹出这般大的动静，萧逸也不知多久后才与众人相遇。
<br/>　　不过，萧逸想起来，当年在东域时，剑姬前辈，可是胡闹得很。
<br/>　　而来到中域后，竟也低调了许多。
<br/>　　若是换了以前，遇到莫忧这种人，以剑姬前辈的性格，怕是教训一顿后，还要抢光她的乾坤戒还有身上重宝，才会罢休。
<br/>　　可来到中域后，却仅仅是教训一顿。
<br/>　　萧逸念及此，不自觉一笑。
<br/>　　但同时，他也能知晓，这些年，剑姬前辈还有众人，必定过得不好。
<br/>　　在中域，这等杀戮乃是家常便饭之地，若背后没有大势力，着实寸步难行。
<br/>　　不难想象，这些年，剑姬前辈为了护着众人，必然也不容易。
<br/>　　“你笑什么？”这时，剑姬前辈没有再传音，而是直接出言问道。
<br/>　　“额。”萧逸尴尬地摸了摸鼻子，总不能直接说剑姬前辈你竟然不再胡闹了，着实让我惊讶这样的话。
<br/>　　故萧逸回答道，“没什么，只是觉得剑姬前辈圣皇境巅峰修为下，便有绝世战力，着实厉害罢了。”
<br/>　　不错，此刻的剑姬前辈，修为达到圣皇境巅峰。
<br/>　　她的天赋，本来就极强。
<br/>　　单论资质、领悟力等等，怕是不比萧逸弱多少。
<br/>　　而她其实来到中域，仅比萧逸慢两年罢了。
<br/>　　且她本身的气泉和小世界大小正常，而不是像萧逸那般小世界大得恐怖。
<br/>　　故她的修为，直追萧逸亦是正常。
<br/>　　萧逸现今的修为，也不过是武道大能罢了。
<br/>　　而说到战力，对于裂天剑姬这样的绝世天骄来说，拥有这般惊人的越级战斗能力，同样不足为奇。
<br/>　　虽说不知道剑姬前辈用的是什么手段。
<br/>　　但直觉告诉萧逸，这和剑姬前辈所修武道有关。
<br/>　　因为，他明显能感觉到，剑姬前辈身上的武道气息之强，远超一般的圣皇境巅峰。
<br/>　　萧逸的判断，不会错。
<br/>　　在他眼中，剑姬前辈，毫无疑问拥有着绝世实力。
<br/>　　当然了，剑姬前辈以前修的是寒冰剑道，可之后修的是什么武道，萧逸倒不是很清楚，也没有多问。
<br/>　　不过，剑姬前辈当年在东域时，能强行自创最后一丝武道，踏入天极，其实已可想而知她的厉害之处。
<br/>　　萧逸的目光，转而看向林劲、凌羽等人。
<br/>　　他感知了一下，微微点了点头。
<br/>　　众人的修为，倒不算太高，大多在圣王境巅峰左右。
<br/>　　若放到中域这里，也就一流天骄层次。
<br/>　　和那些真正的顶尖天骄相比，足足弱了一筹。
<br/>　　不过，这并不代表着他们日后的成就也会弱。
<br/>　　他们对比中域的顶尖天骄，起步更晚，且中域这边的天骄能得到的天材地宝也更强。
<br/>　　他们现今已来中域，假以时日，未必就会真的成就不如别人。
<br/>　　他们的武魂虽弱，但，武魂是判断武者天赋的标准，却从来不是唯一的标准。
<br/>　　他们缺的，仅仅是足够的时间，还有足够的磨砺。
<br/>　　现在的修为低，武魂弱，并不能百分百断定他日之事。
<br/>　　萧逸对他们，充满了期待。
