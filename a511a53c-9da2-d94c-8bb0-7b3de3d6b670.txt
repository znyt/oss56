<br/>				<br/>　　    凤九看了郭老太爷一眼，听到他的话后，这才知道，为何这件事这李家老爷子也来了，而且一来到便是道歉，原来是因为他。<br/>　　<br/>　　    见此，她这才点了点头：“好吧！既然郭老太爷都这么说了，这件事也就算了。”<br/>　　<br/>　　    听到这话，李家老太爷松了口气，上前道谢着：“多谢凤公子，也多谢你了，郭老头。”他露出笑容，朝郭老爷子点了点头。<br/>　　<br/>　　    “你这大儿子确实是不像样，你若是再闭关多几年，估计你这李家也会被他给毁了。”郭老爷摇了摇头说着。<br/>　　<br/>　　    “唉，家门不幸。”李老爷子叹息了一声，对那跪在地上的两人道：“还给滚回去收拾东西！”<br/>　　<br/>　　    父子两人这才站了起来，跟着其他人先行离开，只剩下那几位长老和中年男子站在老者身后。<br/>　　<br/>　　    “李老头，我要出趟门，这些日子也不会在家，等我回来了再找你聚聚吧！”郭老爷子说着，对凤九道：“凤公子，我们可以上路了。”<br/>　　<br/>　　    闻言，凤九应了一声，看了谢家兄妹一眼，两人快步来到他的身边，后面的郭信宁也跟了出去，还有几名随行保护的修士。<br/>　　<br/>　　    郭老爷子手往空中一托，一艘飞船出现在半空中，他对凤九做了个请的手势，便见凤九手往一旁伸去，蹲在她脚边的小银狼纵身一跃扑进她怀里，随着她脚尖一点，身影轻盈跃起进入飞船。<br/>　　<br/>　　    谢家兄妹等人也陆续上了飞船，郭老爷子在最后面，他看着李老头笑了笑，说出来的话让人捉摸不透。<br/>　　<br/>　　    “如何？我没骗你吧？”<br/>　　<br/>　　    李老爷子拱了拱手：“这人情我李家记下了，回头我定请你好好喝一杯。”<br/>　　<br/>　　    “哈哈哈哈！”郭老爷子大笑着，拍了拍他的肩膀，这才提气跃上飞船离去。<br/>　　<br/>　　    李家老爷子与郭家主说了几句话后，便也带着身后的几人离开，往家中而去，随着他们的离去，一名站在家主身后中年男子有些疑惑的询问着：“家主，老太爷刚才那话是什么意思？”<br/>　　<br/>　　    郭家主笑了笑，一边往回走去，一边道：“父亲可以说帮李家避过了一个大劫。”<br/>　　<br/>　　    “何来的大劫？莫不是因为那凤公子？”那人有些不明，就算那凤公子再厉害，莫不是堂堂一个百年家族还怕他不成？<br/>　　<br/>　　    “那时，凤九是动了杀意的，如果李家老爷子来慢一步，估计见到的就会是那李家主的尸体了，李家主一死，李家人定不会善罢甘休，可，若是与凤九一战，李家不算是不会被灭族，百年根基也会被动摇。”<br/>　　<br/>　　    郭家主笑了笑，继续道：“所以，父亲此举，可说是帮李家渡过了一个大劫，若不是父亲念着李家老爷子的交情，估计也不会让人去给他透露消息。”<br/>　　<br/>　　    那跟在旁边的男子一怔：“什么消息？”他怎么越听越糊涂？<br/>　　<br/>　　    郭家主脚步一顿，侧头看了他一眼：“看来，你还得历练历练，这样吧！明天你收拾下东西去郭家的其中一份产业那里当个管事。”<br/>　　<br/>　　    声音一落，他迈步往里走去，留下那中年男子呆站在那里……<br/>
