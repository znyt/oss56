<br/>　　他心中刚刚燃烧起的战意，想要在这逆境中突破，打破楚风眠的镇压，斩杀楚风眠。

<br/>　　这一掌中所蕴含的力量，简直是让欧阳烈如何反抗，都将没有任何意义。
<br/>　　就算是他的力量提升五倍，十倍，都无法在抗衡。

<br/>　　这里是天商会，在这里杀人的话，肯定是会跟天商会起冲突。
<br/>　　天商会是混元大陆前十的势力，楚风眠还要跟天商会来做生意，自然不会现在白白得罪天商会。

<br/>　　敢来招惹楚风眠的人，都必须要付出代价才行。
<br/>　　抽了这欧阳烈身上一般的圣者精血，足以是让他的实力，降到一般寻常巅峰圣者的实力去。
<br/>　　直接相当于是废了欧阳烈的资质，让他以后终生再也无法度过生死大劫。
<br/>　　对欧阳烈这样的天才来说，抽取了他的圣者精血，废了他的资质，比起杀了他还要令他痛苦。
<br/>　　“小畜生，你敢！”
<br/>　　欧阳烈听到楚风眠的话，猛然大声吼道。
<br/>　　“你今日敢动我，天盟肯定会追杀与你的，你的实力再强，也无法跟天盟抗衡！”
<br/>　　“我劝你识相一点，乖乖放我了，将你身上的所有财富都拿出来赔偿我，我还可以不跟你计较！”
<br/>　　“天盟？天盟算是什么东西，楚某早就说过，天盟在我看来，不过只是一群丧家之犬罢了。”
<br/>　　楚风眠猛然之间，一爪已经是向着欧阳烈抓了过去。
<br/>　　“本来我想要给你一个机会，抽取你五成精血，既然你不识好歹，还敢来威胁我？那我就彻底的废了你，抽了你九成精血，让你以后再也无法踏足虚空海！”
<br/>　　楚风眠冷哼一声，出手果断之际，一爪猛然抓向了欧阳烈。
<br/>　　“啊！”
<br/>　　欧阳烈疯狂的惨叫着，在他的身上，一股股圣者精血，直接被楚风眠抽取了出来。
<br/>　　这圣者精血，全部落入到了楚风眠的手中，化为了一枚精粹无比的灵石。
<br/>　　欧阳烈的背后，就算是有着天盟，楚风眠也不会在乎什么，对付这欧阳烈，楚风眠的实力甚至都没有动用三成。
<br/>　　就算是天盟的人来报复，楚风眠也不会怕，来一个楚风眠就杀一个，正好这圣者精血，都是大补之物，可以来炼制出灵石来。

<br/>　　欧阳烈的身躯，在被抽取了九成圣者精血后，直接倒在了地上。
<br/>　　他的力量，虚弱无比，九成的圣者精血被抽取，这欧阳烈简直就已经变成了一个废人。

<br/>　　楚风眠看着欧阳烈，冷冷的开口道。

<br/>　　这些天盟武者，已经都被楚风眠的实力吓破了胆，急急忙忙的离去，根本没有敢跟楚风眠任何抗衡。

<br/>　　“欧阳烈一旦的度过生死大劫，可就是生死境高阶的人物了，这样一尊前途无量的天才，居然就这样被硬生生的废掉了。”
<br/>　　“被人抽取了九成的精血，简直就是跟一个废人，没有任何的区别。”
<br/>　　一尊武者看向楚风眠，都有些被吓破胆了一样，楚风眠的手段实在是太狠。
<br/>　　一尊前途无量的天才，就这样被直接废掉，成为废人，若不是就在他们眼前做到的这一切，谁都不敢想这种事。
<br/>　　“还好我们没有招惹此人。”

<br/>　　本来他们许多人，看着楚风眠这样一个陌生的面孔，也想要来敲诈一番楚风眠的。
