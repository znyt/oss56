　　萧逸此番模样，着实吓人，嗜血的目光、遍地的尸骸，让铁刀和慕容千军心生恐惧。
<br/>　　然而，铁刀大半辈子都在刀口上讨生活，见惯了生死，率先反应了过来。
<br/>　　“混蛋，敢杀我铁刀猎妖队的人，小子，拿命来。”铁刀瞬间暴怒，一拔手上的刀刃，朝萧逸攻去。
<br/>　　铁刀猎妖队，在陨星山脉极有名气，以致雇佣他们的人很多。
<br/>　　而他这次带来帮助慕容千军的手下，全都是他的得力帮手。如今全死了，对铁刀猎妖队乃是极大的损失，这让他如何能不怒。
<br/>　　“别冲动，那小子刚才施展的是萧家的玄阶武技，极为强悍。”慕容千军提醒了一声。
<br/>　　但铁刀没有顾虑，想着自己后天六重的修为，难道还会怕一个毛头小子不成？
<br/>　　“来得好。”萧逸冷喝一声，手中一掌轰出。
<br/>　　在升龙的增幅下，他的实力达到后天四重，配合冰火气丹，他打出的攻击并不亚于后天五重。
<br/>　　也就是说，单论实力，他和铁刀实则只差了一重。
<br/>　　铁刀一刀劈下，攻击极为凌厉，战斗经验也很是丰富，不愧是成名的猎妖师。
<br/>　　萧逸怡然不惧，接连使出了豹形和虎形。
<br/>　　凭借形意五绝，那区区一重的差距，完全被抹平。
<br/>　　萧逸双手成爪，一把扣住了刀刃。
<br/>　　铁刀冷笑道，“臭小子，敢用手挡我的刀，哼，待我把你的手废了。”
<br/>　　萧逸没有说话，那双嗜血的目光紧紧地盯着铁刀，双手握得更加用力。
<br/>　　“嗯？怎么回事。”铁刀立刻发现，自己的刀竟被萧逸死死扣住，动弹不了半分。
<br/>　　另一边，慕容千军双目一凛，低语道，“萧逸竟有如此实力，竟和铁刀战了个平手。看来单凭铁刀是无法收拾他的，那便由我来杀了他。”
<br/>　　话语刚落，慕容千军大手一挥，喝道，“碧波水吟箭。”
<br/>　　漫天箭雨凭空而现，化作一道道蓝芒朝萧逸袭去。
<br/>　　他竟趁着萧逸与铁刀交手之机，暗中偷袭。
<br/>　　萧逸目光一冷，身上腾地冒出一股火焰，手上则继续对付着铁刀。
<br/>　　铁刀双手握刀，限制住萧逸，心头却是一喜，“我与千军公子联手，小子你必死无疑。若你敢分神抵挡那些箭雨，我的铁刀便废了你的双手；若你继续挡我的铁刀，那些箭雨便会将你刺成筛子，哈哈哈哈。”
<br/>　　“哼。”萧逸冷哼一声，脸上只有镇静与冰冷，快速地喝了一声，“噬火百刃。”
<br/>　　几乎是在他声音落下的瞬间，原本扣住大刀的双手上猛地迸发出两把散发着骇人温度的火刃。
<br/>　　铁刀顿时脸色一变，似是想到了萧逸要做什么，“小...小子，你想做什么？”
<br/>　　萧逸脸色不变，冷冷地盯着铁刀，道，“不想做什么，只想要你的命。”
<br/>　　铁刀瞳孔一缩，当即欲要收刀并逃脱。
<br/>　　可是，萧逸不会给他逃脱的机会。
<br/>　　“死吧。”萧逸冷喝一声，两道火刃脱手而出。
<br/>　　“疯子，你这个疯子，这么近的距离，你也会重伤的。”铁刀面露惊恐，他万万没想到，一个少年竟然疯狂到这般地步。
<br/>　　嗤的一声，两道火刃狠狠地打中了铁刀的胸膛，而后轰然爆炸。
<br/>　　嘭的一声，铁刀身上一阵爆炸，而后气孔流血地被击飞。
<br/>　　同一时间，如此近距离的爆炸，也将萧逸的双手炸得血肉模糊，连自身的肉体也受了不轻的伤势。
<br/>　　萧逸的一系列动作很快，几乎是在一瞬间便完成。
<br/>　　此时，漫天箭雨来到了他背后。一部分被他身上的火焰抵消，剩下的全部从他后背穿刺而过。
<br/>　　“嘭、嘭。”萧逸吐出大口鲜血。
<br/>　　当箭雨全部打完，他已经遍体凌伤，身上出现了一个个血洞。
<br/>　　但他看着前方已经死去的铁刀，却满意地笑了起来。
<br/>　　铁刀看起来没有受致命的伤势，但其实那两道火刃，在打入他的胸膛后爆炸，已经将他的内脏和器官炸得粉碎，他早已死得不能再死。
<br/>　　萧逸笑着，即便他现在也身受重伤，可换来铁刀的死亡，他认为是非常值得的。
<br/>　　另一边，慕容千军看着萧逸那笑容，顿觉毛骨悚然。
<br/>　　“这个疯子，这样还能笑得出来？”慕容千军发觉自己竟对萧逸产生了一丝恐惧。
<br/>　　萧逸挺着重伤，转过了身，看向慕容千军，冷冷道，“现在，轮到你了。”
<br/>　　“你...”慕容千军不自觉地退了一步，但很快反应了过来，“哼，他不过是个废物罢了，加上如今已然重伤，我何须惧他。”
<br/>　　想罢，慕容千军又恢复了往日的狂傲，“萧逸，你能杀了铁刀，算你有本事。但，你觉得你今天还能从我手上逃脱？”
<br/>　　“逃脱？哈哈哈哈。”萧逸大笑一声，随后脸色冰冷，“慕容千军，你们慕容家散播谣言，毁我声誉，我本并不在意。但你竟还敢四处诋毁我萧家长辈，多番中伤，我饶不了你。”
<br/>　　当初慕容家散播谣言后，竟还多番中伤萧家三长老萧重不分青红皂白包庇萧逸，两人沆瀣一气、狼狈为奸。
<br/>　　这才是萧逸愿意答应大长老去找慕容千军麻烦的真正原因。
<br/>　　加上如今在密林中受重伤的萧家子弟，新仇旧恨，萧逸直接起了杀意。
<br/>　　“萧壮和萧子木的丹田被毁，日后再也当不了武者，终其一生都只能是废人，你也休想好过。你的命，我代他们两人收下了。”萧逸冷声说罢，瞬间朝慕容千军攻去。
<br/>　　萧逸的重伤只是表面上的，并没有想象中那么严重。之前他早就做好了打算，一边拼着重伤杀掉铁刀，一边则将火焰覆盖到自己身上，特别是要害部位。
<br/>　　以此来抵消掉漫天箭雨。
<br/>　　虽然，虽然他身上有许多血洞，但都不会致命。
<br/>　　“哼，想杀我？大言不惭。”慕容千军冷哼一声，低喝道，“就让你看看你我的差距，碧波叠浪掌。”
<br/>　　话语刚落，慕容千军手上猛地凝聚出一层庞大的水浪，水浪翻滚不停，威势骇人。
<br/>　　碧波叠浪掌，慕容家玄阶低级武技，威力极强。
<br/>　　萧逸脚步不停，手中也依靠控火能力凝聚出一把‘火剑’。
<br/>　　这并不是什么厉害的招数，只是单纯用火焰凝聚成的剑而已。
<br/>　　“崩山斩。”萧逸大喝一声，一剑朝慕容千军劈去。
<br/>　　这时，慕容千军也打出了那道庞大水浪。
<br/>　　火剑瞬间与水浪对轰，炽热火焰与冰冷波浪接触，立刻激起无数水蒸气。
<br/>　　但下一秒，火剑上似乎蕴含一股毁天灭地之力，竟重重地压垮了滔天波浪。
<br/>　　火剑将波浪从中间劈开，而后劈向了慕容千军。
<br/>　　“什么？我的碧波叠浪掌竟然被破了？”慕容千军顿时一惊，而后脚步猛退。
<br/>　　嗤的一声，火剑在他胸膛上划出一道狰狞的伤痕。随后，剑上的那股势大力沉的力量，直接将他震飞。
<br/>　　若不是他反应快，立刻脚步猛退，恐怕早已被火剑分尸。
<br/>　　“噗。”慕容千军站稳身子后，猛地吐出一口腥血，不可置信地看着萧逸，“怎么可能，萧家最强的不过是玄阶低级武技，怎么可能破我慕容家的武技？”
<br/>　　萧逸冷笑一声，崩山斩乃玄阶中级武技，自然更强。
<br/>　　“躲得了一剑，我看你还躲不躲得了第二剑。”萧逸再次朝慕容千军攻去，摆明了不杀他不罢休。
<br/>　　慕容千军一惊，再次使出了碧波叠浪掌。
<br/>　　轰的一声，仍旧是那个结果，波浪被破，身上又多一条狰狞伤痕，胸膛上汩汩留着鲜血，摆明了已然重伤。
<br/>　　第三剑，慕容千军被震飞后，竟连站起来的力气都没有了，脸色苍白，大口大口地吐着鲜血。
<br/>　　“哼。”萧逸冷笑一声，来到他身旁，一把夺过了他腰间的乾坤袋，看了一眼，三颗妖血炎心果都在里面。
<br/>　　“妖血炎心果还有里面的东西都给你，萧逸，你放过我。”慕容千军忍住伤势，求饶道。
<br/>　　看着萧逸那嗜血的冰冷目光，他毫不怀疑萧逸会杀了他。
<br/>　　“呵。”萧逸冷声道，“如今你的乾坤袋就在我手中，我需要你给吗？”
<br/>　　说罢，萧逸一举手上火剑，狠狠刺下。
<br/>　　如无意外，在重伤的情况下，慕容千军无法抵抗，必死无疑。
<br/>　　“不要...”慕容千军惊呼一声。
<br/>　　恰在此时，远处，一道暴喝声传来。
<br/>　　“小子，给我住手。”
<br/>　　人未至，声音已到。同时，一道恐怖的攻击凌空打来，直直地打向萧逸。
<br/>　　这道攻击虽然从远处打来，但速度奇快，快到萧逸根本反应不过来就被击中了。
<br/>　　‘噗’，萧逸直接被吐血击飞。
<br/>　　“好强，能有如此实力打出这样攻击的，只有可能是先天武者。”
<br/>　　果然，一道人影破空而来。慕容千军见到人影后顿时大喜，“慕容蝎长老，救我。”
<br/>　　另一边，又一道人影破空而来，速度丝毫不逊色于之前的人影。
<br/>　　“萧逸，你没事吧。”
<br/>　　人影来到，竟是四长老。
