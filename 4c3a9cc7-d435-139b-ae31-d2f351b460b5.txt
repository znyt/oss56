<br/>    紫金山下。
<br/>    
<br/>        天空被飞舞腾空的飞剑遮蔽，一根根锁链横空，如同布下一张弥天大网。
<br/>    
<br/>        锁链飞舞之间，一个个大妖被锁链缠住，而后在锁链之上的飞剑拉扯之下化作碎片。
<br/>    
<br/>        金犴站在紫金山上，眼中急的冒出火花来，“这是什么情况，燕山的妖军怎么会精锐到这个地步，这可比那些斩杀沙蝎的燕山军队精锐多了！”
<br/>    
<br/>        不过金犴还没有放弃，他还拥有着十多万大军，金犴挥手道：“飞剑，给我用飞剑！”
<br/>    
<br/>        天空之中，瞬间有着上万柄飞剑横空，朝着血衣剑卫飞去，凌仙看了一眼这些飞剑，不过上万柄而已，对方毕竟不是如同燕山一般全部统一的军队，更像是一个个妖王部落组合在了一起。
<br/>    
<br/>        凌仙轻蔑地笑了笑，“还真是不堪一击！这点攻击，挠痒痒还差不多。”
<br/>    
<br/>        血煞之气瞬间爆发，三万血衣剑卫的血煞之气，如同一朵巨大的红云一般升起，阻挡了上万柄飞剑的袭击，飞剑落在红云之中，便失去了继续飞行的力量。偶尔有几柄飞剑落于血衣剑卫之中，也是被血衣剑卫身上的灵甲所阻挡，根本伤不到他们。
<br/>    
<br/>        凌仙朝着对方的妖军挥手道：“飞剑，杀！”
<br/>    
<br/>        飞剑再次横空，血衣剑卫根本没准备防御，一次剑光掠过，就是地方数千大妖被切碎。
<br/>    
<br/>        鲜血在空中挥洒，落在猎猎狂风中，然后落在地面上，一滴一滴凝聚，汇聚成一条条细小的溪流，整个战场，瞬间遍布鲜血，一头头死去的大妖显出本体，身体之中的血液如同小河般流淌。
<br/>    
<br/>        凌仙看这架势，又是指挥了几轮飞剑袭击，金犴看得两眼发麻，这才打了多久，自己已经损失了两万人马了，这根本就是一场一面倒的屠杀！
<br/>    
<br/>        金犴伸手拿出腰间的令牌，“我以紫金山妖王太子之名下令，出动狂狮部王级军团，给我冲散他们的阵型！”
<br/>    
<br/>        紫金山之中，四十位妖王冲了出来，汇聚在一处，朝着血衣剑卫的一个方阵冲去。
<br/>    
<br/>        凌仙笑了笑，“王级军团了不起啊，今日，便让你们好好看看杀王剑的威力！散！”
<br/>    
<br/>        四十位妖王本想杀尽血衣剑卫之中，没有想到的是，血衣剑卫竟然是直接给他们让出了一条路来，四十位妖王大喜，直接朝着血衣剑卫的中军冲去，就在这时凌仙继续道：“内圈，围杀王级军团，外圈，抵御外部妖军！”
<br/>    
<br/>        四十位妖王还没有弄明白怎么回事，就看到天空之中已经是数千条锁链横空，如同天罗织网一般将他们封锁在了其中，而后，又是数千柄的飞剑袭来，他们的护体法力为之消散，当他们想要继续调动法力之时，却是发现，自己的法力竟然运转变得迟滞起来，就在这时，已经又是数千柄飞剑袭来！
<br/>    
<br/>        “不！”
<br/>    
<br/>        “这到底是什么诡异的灵兵！”
<br/>    
<br/>        “我的法力，我的法力运转变得迟滞了！”
<br/>    
<br/>        “啊！”
<br/>    
<br/>        一柄柄飞剑从他们的胸膛中穿过，从他们的四肢穿过，鲜血从他们的伤口中溢出，同样带走了他们的力量。
<br/>    
<br/>        围拢的血衣剑卫再次散开，继续在凌仙的指挥下开始进攻。
<br/>    
<br/>        金犴看着王级军团冲进了血衣剑卫之中，他本以为，王级军团入场，会给对方造成巨大的骚乱，没有想到的是，一个王级军团，竟然如同泥牛入海一般了无音讯，而现在，更是消失的无影无踪。
<br/>    
<br/>        金犴的手在颤抖，“这燕山的军队，都是什么怪物.......”
<br/>    
<br/>        数千里之外。
<br/>    
<br/>        层峦叠嶂之间，崇山峻岭之中，金毛吼捂着自己胸口的伤口正在逃窜，在他的背后，卓青瑶手握巨阙，巨阙之上泛着寒芒，朝着金毛吼追去。
<br/>    
<br/>        金毛吼看了一眼天空，有着一头青眸玄牛凌空而过，这上面的仙门正道也是奇怪，他不和自己直接交手，但是一旦自己想要逃走，就是凌空一枪直接朝着自己的头打来，将自己再次逼到地面上。
<br/>    
<br/>        刘芒此刻倒是挺轻松的，朝着卓青瑶喊道：“这头老狮子虽然实力不怎么样，但是毕竟有着一丝九头狮子的血脉，不如降服他作为你的坐骑如何？”
<br/>    
<br/>        卓青瑶微微摇头，然后道：“师父说过，这只老狮子，必须死！”
<br/>    
<br/>        刘芒也是没有多话，这毕竟是陈明的安排。
<br/>    
<br/>        金毛吼此刻也是山穷水尽了，朝着卓青瑶嘶吼一声，化身成一头身上三十多丈的双头狮子，两张血盆大口朝着卓青瑶嘶吼道：“燕山大宗主，我好歹也是这十万大山之中一霸，你们燕山如此实力，我就是投靠你们也未尝不可。”
<br/>    
<br/>        卓青瑶笑道：“你这个人，师父说过，喜欢用些卑鄙的伎俩，而且反复无常，师父，只说过让我提着你的头去见他，可没有说让你活着去见他。”
<br/>    
<br/>        金毛吼怒道：“既然你不给我活路，那我也只有拼死一战了。”
<br/>    
<br/>        轰隆隆！
<br/>    
<br/>        一声声巨响在群山之中回荡，惊得周围的野兽四处奔逃，众多飞鸟腾空，大妖们早就已经逃离了他们的战场，只剩下一些妖王还敢在远处远远地看着，“燕山大宗主，在这十万大山之中，恐怕是大君之下最强者。”
<br/>    
<br/>        “没有想到，燕山真的这般按强大，竟然连金毛吼这样称霸一方已久的大妖王都不是对手。”
<br/>    
<br/>        “看金毛吼如今狼狈的模样，只怕今日就要死在燕山大宗主的手中了。”
<br/>    
<br/>        三天之后，一处山丘之上，两颗硕大的狮头从小山上落下，如同两块巨石落地，撞得山石破碎，卓青瑶气喘吁吁地站在失去了头颅的狮身之上，将手中的巨阙在狮身上擦拭，卓青瑶看向周围那些围观的妖王们，“从今日起，就没有紫金山了，只有燕山，这个地方，姓燕山了。”
<br/>    
<br/>        众多妖王看向卓青瑶，目光之中有的只是敬畏，齐齐拱手道：“我等见过燕山大宗主！”
<br/>    
<br/>        “燕山大宗主所向无敌！”
<br/>    
<br/>        消息很快传遍十万大山，称霸紫金山四百多年的金毛吼死于燕山大宗主卓青瑶之手，燕山用他恐怖的实力，再次宣布，谁才是这十万大山之中真正的霸主。
