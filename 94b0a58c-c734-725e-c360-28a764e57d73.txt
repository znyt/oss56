        &lt;h3&gt;第三卷第一百三十章  冷冰冰的凌辰&lt;/h3&gt;
<br/>
<br/>        何少华想着自己之前以为稳稳的拿到了总代的位置，没有想到厂家竟然敢拂了他的面子，当然也不是拂了他的面子而是拂了他舅舅的面子，更是拂了那位的面子。
<br/>
<br/>        在会场上他的脸面就挂不住了，听到厂家宣布凌辰继续为总代，他起身就走了。
<br/>
<br/>        回到酒店，他给他舅舅打了个电话，将这里的情况添油加醋的说了一通。
<br/>
<br/>        “那个凌辰卑鄙的很，也不知道是用了什么手段又拿到了总代。舅舅，你说了今年的总代是我们拿的。”
<br/>
<br/>        “拿不到总代就算了，一级代理是无论如何要拿到的。”舅舅在电话里叮咛。
<br/>
<br/>        “我得罪了那个凌辰，也不知道后期还能不能拿到。”何少华说。
<br/>
<br/>        “不给你总代，一级代理是无论如何都不能丢的，别那么熊。就算是凌辰能拿到总代又能怎么样？这么几年你还不是想怎么干就怎么干？你记得你要一个名额就够了。”舅舅在电话里再次叮咛。
<br/>
<br/>        “好！”何少华心里的那个憋屈啊，简直就是无法形容。
<br/>
<br/>        到手的鸭子飞了。
<br/>
<br/>        但是他还得找凌辰，虽然有一千个一万个不愿意，他还得去。
<br/>
<br/>        和凌辰就在一个酒店，但是就好像是隔着千山万水，走得好累啊。
<br/>
<br/>        敲响了凌辰那豪华的总统套间的门，刘诚之看着垂头丧气的何少华：“何总这是怎么了？昨晚没有睡好？无精打采的？”
<br/>
<br/>        何少华看了一眼房间，房间里还有一些人在和凌辰聊天。
<br/>
<br/>        “何总，请进来吧，凌总正忙着，大家都在交明年的计划。”刘诚之不知道何少华和凌辰之间的过节，但是也知道何少华想争夺总代的位置。
<br/>
<br/>        何少华并没有理刘诚之，直接就坐到了沙发上，刘诚之给何少华倒了一杯茶便忙工作去了，何少华冷眼看着忙碌的众人，心中想着：“等我追到董亚子，整个董家都是我的，你凌辰现在还不是为我打工。”
<br/>
<br/>        这样一想，何少华竟然看凌辰无比顺眼，不过又想起董亚子叫凌辰为：“凌辰哥哥”的时候心里就像是吞了一只苍蝇。
<br/>
<br/>        其实那天见到的那个董巧奕真的是一个性感美女，比董亚子好看了很多，要不是舅舅替他提了亲，他才看不上董亚子呢，唉，要是俩个都能得到就好了。
<br/>
<br/>        何少华沉浸在自己的自娱自乐中。
<br/>
<br/>        各地区的代理商都满意的离开了，何少华还在一脸痴迷的想着董巧奕的美貌。
<br/>
<br/>        “何总？何总？”刘诚之用手在何少华的眼前挥了挥，这家伙抽风了吗？口水都流出来了，不过看那一脸猥琐的样子，又不像是抽风的样子，直愣愣的盯着凌辰。
<br/>
<br/>        刘诚之打了一个寒颤，妈耶，这家伙被我们凌辰的美色所迷，cao，变态！
<br/>
<br/>        我们凌总是雅雅小姐姐的。
<br/>
<br/>        我们凌总和雅雅小姐姐是绝配，你一个臭猥琐男竟然敢肖想我们凌总。
<br/>
<br/>        “凌总，你看他！他......”刘诚之简直不忍直视。
<br/>
<br/>        凌辰也看到了何少华那痴迷的样子，这家伙做白日梦呢。
<br/>
<br/>        “刘诚之，该到饭点了吧？叫上那几个代理商，我们一起去喝几盅。”凌辰吩咐刘诚之。
<br/>
<br/>        “这个人？”刘诚之指着何少华，这个人绝对脑子有病，他们都说了好半天的话，他还是那个样子。
<br/>
<br/>        “喊醒他啊。难道还要我动手吗？”凌辰有些不耐。
<br/>
<br/>        “何总。”刘诚之拍了一下何少华的肩膀，迅速收回手，甩了甩。
<br/>
<br/>        何少华被刘诚之拍了一巴掌，惊醒了，“谁他......”
<br/>
<br/>        突然看见冷冰冰的凌辰，感觉一股子凉气冷飕飕的入了喉钻进了四肢百骸，突然就卡壳了。
<br/>
<br/>        “何总，大家都走了。”刘诚之拍了一巴掌何少华，感觉手都是脏的，想赶快去洗洗，只是老板还站着呢。
<br/>
<br/>        “哦，凌总，我是来看计划的。”何少华回过神来。
<br/>
<br/>        “看，计划？难道何总没有做计划？”只有总代是不做计划的，只是会和厂家一起制定销售计划。
<br/>
<br/>        何少华是奔着总代来的。
<br/>
<br/>        “不瞒凌总说，走之前有人让我直接来拿总代的，没有想到凌总的合同没有到期。”何少华回过神，没有拿到总代没有关系，只是因为你的合同没有到期而已，要不然你以为总代还是你的？
<br/>
<br/>        “呵，那何总就好好等着拿总代吧！”凌辰大长腿一迈出门了。
<br/>
<br/>        刘诚之迅速冲到洗手间，将自己的手洗了又洗，然后迅速跑出去了，留下何少华独自站在豪华的套间。
<br/>
<br/>        “凌总，凌总，我怕！”刘诚之急乎乎的赶上凌辰。
<br/>
<br/>        “怕什么？”凌辰感觉奇怪，一个大男人在酒店里有啥好怕的？
<br/>
<br/>        “那个何少华太不是东西了，他竟然敢肖想你，你是雅儿小姐姐的。”刘诚之气哼哼的说。
<br/>
<br/>        “刘诚之，你这脑袋里想的什么呢？何少华取向正常，再说我也不是你雅儿姐姐的了。”凌辰叹了口气。
<br/>
<br/>        有好多天没有和宁雅联系了，她应该很幸福吧。
<br/>
<br/>        “你什么意思啊？雅儿姐姐那么好，那么漂亮，你竟然说你不是她的了？”刘诚之经常听见凌辰念叨宁雅，而且俩个人打电话，一打就好久。
<br/>
<br/>        “你懂什么？”凌辰今天难得的多说了几句话。
<br/>
<br/>        不是他的错啊，他现在的一切不都是为了雅儿吗？可是宁雅呢？接他的电话的时候和别人“嘤嘤嘤”？
<br/>
<br/>        提起宁雅他心里都是疼。
<br/>
<br/>        “以后不要再我面前提起宁雅！”凌辰的语气冷冽而缥缈让刘诚之心里一惊。
<br/>
<br/>        晚上的聚餐热闹非凡，大家都心满意足，对凌辰是敬佩有加。
<br/>
<br/>        何少华不是说他们家势力很大吗?不是说今年的总代非他莫属吗？这脸打得啪啪啪。
<br/>
<br/>        现在他们都明白了一个道理，越是有能力的人越是低调，比喻说凌总。
<br/>
<br/>        那晚和何少华一起的人，更是羞愧不已，他们跟着何少华一起看凌辰的热闹，可是凌辰当什么事情都没有发生，不但没有为难他们，反而给他们的福利比往年的还要好。
<br/>
<br/>        几个人拿着酒来敬凌辰的酒：“凌总，您大人不计小人过，以后凌总一句话，我们万死不辞。”
<br/>
<br/>        “说什么万死不辞呢，只要大家能赚到钱，我就心满意足了！”凌辰对来敬酒的都来而不拒。
<br/>
<br/>        刘诚之吓坏了，他们凌总其实是不爱喝白酒的。
<br/>
<br/>        上次喝多了睡了一天一夜，别今天又喝多了再睡一天一夜。
<br/>
<br/>        何少华并没有去参加他们的晚宴，他不屑于跟他们在一起，凌辰，咱们走着瞧。
