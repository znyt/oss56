　　林峰带着陈勾直接朝咖啡馆突进。
<br/>　　“这附近丧尸似乎很多啊。”陈勾扫了眼。
<br/>　　“最起码三十多个。”
<br/>　　林峰道；“附近都是小区和一些街道，肯定丧尸多，三十多个而已，我们两人一人一半。”
<br/>　　“林峰，这里居然还有幸存者。”陈勾立即道。
<br/>　　林峰瞧去果然发现有七八个人在清理丧尸。
<br/>　　这群人身手很不错，拿着水管之类的武器，那些丧尸不是对手。
<br/>　　“都是些二阶的黑铁战士怕个毛。”
<br/>　　林峰直接带着陈勾走了过去。
<br/>　　“有人！”
<br/>　　那七人立即也发现了林峰等人。
<br/>　　看到林峰一个人拿着武士刀，一个人拿着精钢长枪都是满脸愕然。
<br/>　　“快给大佬让路！”
<br/>　　七个人立即跑一边去了，还十分警惕的看着林峰，就怕林峰他们会抢劫。
<br/>　　然而林峰对他们压根没兴趣。
<br/>　　抢劫他们有什么意义？
<br/>　　没食物，武器也就是水管，太穷了。
<br/>　　“我去，林峰，居然有人给我们让路，而且你没看到他们几个看我们的眼神，畏惧！”
<br/>　　陈勾笑道。
<br/>　　“这就是实力，这群人不傻，知道我们实力比他们强。别废话了，这个地方看来有不少幸存者。看来是因为这里有超市和药店的缘故啊，也是，这可是末世后的热门景点啊。”
<br/>　　林峰道。
<br/>　　“快抢啊。”
<br/>　　“妈的，这是老子最先看到的水。”
<br/>　　“滚，上面写了你名字吗？还你的。”
<br/>　　“求求你了，给我留一瓶吧，我孩子已经两天滴水未沾了，求求你，只要一瓶就好。”
<br/>　　“滚开吧，你有孩子，我没孩子？”
<br/>　　“求好心人给点吃的啊，求求你们了。”
<br/>　　林峰和陈勾再朝前走后顿时就看到了愕然的一幕。
<br/>　　在一家还算不小的超市前。
<br/>　　这里人来人往，超市内还有不少人在疯狂的抢夺东西，地上还躺着十几个丧尸的尸体还有两个幸存者的尸体。
<br/>　　在超市外，还有几个人正下跪的在那里磕头。
<br/>　　尤其是一个妇女，她还带着一个七岁的孩子，那孩子似乎是生病了，眯着眼，浑身无力的。
<br/>　　“求求你们，给我孩子一盒退烧药吧。”
<br/>　　“自己不会去药店拿？滚开。”
<br/>　　路过的一人直接甩开他的手，扛着一箱子药丢上了车然后和同伴开着车走了。
<br/>　　“没有了，药店里什么都没有了。怎么办啊，我苦命的孩子啊。”
<br/>　　“我去，这尼玛，人间悲剧啊。”
<br/>　　陈勾和林峰都愣住了。
<br/>　　林峰也心中很不是滋味。
<br/>　　但是没办法，他也只是个运气好点的小人物罢了，他也没有指望自己可以当什么救苦救难的观世音和打怪兽的奥特曼。
<br/>　　“走吧，我们去咖啡馆。”
<br/>　　林峰和陈勾走一边绕开，超市内还有人在打架，只是因为一包一块钱的辣条罢了。
<br/>　　“这是老子的！”
<br/>　　“滚尼玛的，明明老子先看到的！”
<br/>　　“去死吧！”
<br/>　　“还敢动刀？来啊，谁怕谁，老子砍死你！”
<br/>　　林峰和陈勾来到咖啡馆，这个咖啡馆内没有什么人。
<br/>　　“你在外面警戒，我进去找东西。”
<br/>　　林峰推开玻璃门进去，扫了眼，的确没有什么丧尸。
<br/>　　估计咖啡馆的员工第一时间就跑了吧。
<br/>　　林峰看了眼收银台，里面的钞票都没了。
<br/>　　“还有煞笔这个时候了还拿钱？莫非是当擦屁股的纸吗？”
<br/>　　林峰无语。
<br/>　　就在这个收银台下一个D级空投安静的躺着。
<br/>　　咔擦。
<br/>　　林峰直接打开。
<br/>　　“又是个背包？”
<br/>　　叮！
<br/>　　拓展物品栏！目前为80格！
<br/>　　除了个背包还不错外，其余就一瓶怡宝矿泉水外加一包绿箭口香糖。
<br/>　　“林峰啊林峰，若是外面那群人知道在你眼中，水不是什么好东西，估计你要被别人用目光杀死了。”
<br/>　　林峰哭笑不得。
<br/>　　林峰把他们全部收入物品栏，来到咖啡馆外面，丢给了陈勾一个口香糖。
<br/>　　“找到了什么？”
<br/>　　陈勾嚼着口香糖道。
<br/>　　“就这玩意，什么都没有。”
<br/>　　林峰耸了耸肩。
<br/>　　“妈的，你们这群人都给老子把东西乖乖交出来！”
<br/>　　忽然。一边开来了三辆面包车。
<br/>　　旋即十几个拿着砍刀的年轻人就从面包车上跳下来。
<br/>　　“你们谁啊，凭什么给你们。”一个背着背包的男人不服道。
<br/>　　刷。
<br/>　　下一刻。这个男人脑袋就开花了，直接被一刀砍断了脖子。
<br/>　　“妈的，还敢废话，谁不服气的站出来？”
<br/>　　十几个年轻人冷笑道。
<br/>　　“你们欺人太甚，我们找到的，为什么给你们！你们这是谋杀！”
<br/>　　“没错！谁怕你，当你是黑社会的了不起啊，大不了鱼死网破！”
<br/>　　“就是，你们这群社会败类，之前末世前也就算了，不和你们一般见识，现在谁怕谁？有本事朝老子砍？看看老子手中的棍子是不是吃素的。老子也是黑铁战士！”
<br/>　　“没错，大不了一起死嘛，来嘛，谁怕谁嘛，反正老子也不想活了！光脚的不怕穿鞋的！！”
<br/>　　一群幸存者楞了一下后个个都怒吼道。
<br/>　　他们一个个都是被逼到了绝路的，早就已经不怕死了，若是怕死也不会出来找东西。
<br/>　　末世之前大家逆来顺受，看到黑社会都绕着走怕惹上不必要的麻烦，但是现在，谁怕谁啊，大家都是一无所有的。
<br/>　　“有意思，居然不怕死，我倒要看看有几个不怕死的。”
<br/>　　一道冷笑，一个身上有一条龙纹身的年轻人冷笑的走了过来。
<br/>　　“大哥！”
<br/>　　其余年轻人各自推开。
<br/>　　“我说怎么都不怕死，原来都是些一阶黑铁战士啊。呦呵，还有个二阶的。不错。记住，我叫广子，认识我的人，都叫我广哥。你们现在有个机会，只要给我跪下来喊我一声广哥，我可以不杀你们，不然动起手来，你们这群人还真不够看。”
<br/>　　几个幸存者都是惊愕的看着这广哥。
<br/>　　“居然是三阶的？这家伙杀了多少丧尸了。”
<br/>　　一群人都面色难看。
<br/>　　“黑社会？社会人？”
<br/>　　陈勾看着那群身上文龙画风的家伙道。
<br/>　　“管我们屁事，走，几个一阶两阶的小流氓罢了，若是来找麻烦，那就是他们找死。”
<br/>　　林峰道。
<br/>　　“站住！你们两个小子莫非还想要溜走不成？不知道我们广哥在这里！”
<br/>　　一个小弟拿着砍刀就是走了过来。
