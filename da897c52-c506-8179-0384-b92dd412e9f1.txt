<br/>　　凤凰族的修士显出巨大的凤凰真身，全身散发出炙热的真火，焚烧苍穹，飞到九天之上，舞动双翅，无数巨大火球冲击而下，又如灭世一般。

<br/>　　“嘿嘿！杀吧！给我狠狠的杀吧，你们杀的越欢，我越是高兴！”
<br/>　　一处虚空之中，罗睺望着三族正在惨烈大战，身后四柄杀气惊天的剑影不断切割空间，形成一道道空间裂缝。

<br/>　　下面的大战如火如荼的进行着，杀戮滔天，而上面，却是龙祖、麒麟王与凤祖三人大战在一起。
<br/>　　三人各具优势，大战到如今都已经有些筋疲力尽了，每人身上都伤痕累累，血迹斑斑。
<br/>　　就在此时，虚空之中，顿时显现出一群人，他们不问青红皂白，遇到三族成员就杀。
<br/>　　他们每一个都是大罗金仙修为，此时不顾一切的杀向三族子弟，三族成员一时间没有反应过来，一下子就被灭杀一大半。
<br/>　　“吼！”
<br/>　　三族子弟在怒吼，在疯狂，不顾一切的要杀了眼前这些人。他们才不管谁谁谁，遇上就杀，他们早已经杀红了眼。
<br/>　　“杀！杀！杀！”
<br/>　　大战变得越来越复杂，从开始三族大战到如今的洪荒大战，已经分不清敌我了，这些人早已经成了杀戮的机器，他们就不是人。
<br/>　　眼见不敌对方，已经有人开始自爆了。

<br/>　　随着一声巨大的爆炸声响起，却是麒麟一族的长老自爆真身，拖着对方同归于尽。
<br/>　　有了第一个，那就会有第二个，第三个……
<br/>　　那声巨大的声响，仿佛引爆了什么，紧接着，一声声巨大的爆炸声在虚空响起。

<br/>　　“轰隆隆！”
<br/>　　爆炸声不断响起，天地间无数修士自爆真身，拖着对方一起陪葬。
<br/>　　三族修士不顾生死，舍生忘死，拼命挥动手中武器，将一个个敌人的身体不断撕开，染血的头发迎风飘扬，厮杀声演奏出一曲亘古的三界悲歌！
<br/>　　整片虚空被搞得支离破碎，到处都是一个个数丈大小的黑洞，光线时断时续，山岳般大小的血爪，无边的火海，灰蒙蒙的世界等恐怖神通不断出现！
<br/>　　就连凰祖都被浑天老祖打伤，最后不得不自爆身躯，拖着重伤之躯，与浑天老祖一起同归于尽。

<br/>　　“轰…轰…轰！”
<br/>　　一阵阵震耳欲聋的响声从天上传了过来，让人不由震撼，三人一次次的对轰着，似乎不知疲倦一般，大地被摧残的不成样子，一片狼藉。
<br/>　　“龙傲天下！”
<br/>　　龙族挥动手中金龙枪，发出他最大威力的一击，轰向对方。
<br/>　　“凤舞九天！”
<br/>　　凤祖也挥动双翅，发出最大一击，打向龙祖与麒麟王。
<br/>　　“麒麟镇世！”
<br/>　　麒麟王亦四脚踏天，巨首仰天，怒吼而来，踏向两人。
<br/>　　轰隆隆！
<br/>　　虚空破碎，演化黑洞，时光扭曲，空间洞穿，日月失色，天地昏暗。无数生灵被化成血雾，洒遍天涯，铺红大地！
<br/>　　一击过后，三人都各自突出一口鲜血，并且似断了线的风筝般，跌落在地，吐血不止。

<br/>　　三人身受重伤，近乎身陨，但仍有几分威严在，只是三人现在脸色都不怎么好看。
