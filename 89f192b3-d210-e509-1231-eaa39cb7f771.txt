　　“八爪鱼”招供了！？
<br/>　　麦克米兰的话，令罗昊和云熙都是感觉到有些诧异。他们没想到“八爪鱼”会在这么短的时间内就把他知道的招出来，因为他所知道的东西，全都是他可以赖以保命的东西，只要他死咬着不松口，他就不会有生命危险；但是他一旦松口吐露秘密，他就会失去价值，很有可能会丢掉性命。
<br/>　　所以“八爪鱼”会在这么短时间内招供，不得不让罗昊和云熙感到意外，甚至怀疑“八爪鱼”给出情报的真实性。
<br/>　　麦克米兰神秘一笑，对罗昊说道：“你不需要怀疑情报的真实性，将军对‘八爪鱼’使用了这个……”
<br/>　　麦克米兰做了一个打针的动作，随后又指了指自己的脑袋，意思是詹姆斯对Parker使用了那种只有一些情报部门才有，直接作用于人的大脑，使人在无意识，类似于被催眠的情况下吐露心中秘密的药剂。
<br/>　　这种药剂对人大脑会造成不可逆的伤害，最严重的情况下，会使人变成白痴！
<br/>　　“所以‘八爪鱼’交代的东西还是可信的。”麦克米兰说道：“而且兰利那边也对‘八爪鱼’交代的情况进行了核实，确认了情报的真实性。”
<br/>　　“需要我帮忙吗？”罗昊对麦克米兰问道。
<br/>　　“不不不。”麦克米兰连连摇头，指了指云熙，说道：“你还是陪着你妻子吧。”说完，就小跑着进入那架停在甲板上的SH-53E“超种马”运输直升机。
<br/>　　“我们的罗队长居然还被人嫌弃了，真是有些出乎人的意料啊。”
<br/>　　云熙双手背在身后，笑吟吟的看着罗昊，一副看好戏的模样。
<br/>　　“女人，你现在胆子是越来越大了，居然连我都敢取笑。”罗昊脸色突然一沉，然后冲向云熙。
<br/>　　云熙嬉笑着逃跑开，可是并没有跑出多远，就被罗昊从背后抓住，抱在怀里，咬着她的耳朵，说道：“胆大的女人，你说我应该怎么惩罚你？”
<br/>　　听到罗昊的话，云熙哪里还能不明白他话中的意思，俏脸一红，说道：“这还是在别人的船上呢，下午的时候就不应该答应你！就算再着急，也得回家啊！”
<br/>　　罗昊突然松开手，在云熙额头上弹了一下说道，“女人，你在想什么呢，思想这么龌龊！”
<br/>　　云熙当即反应过来自己是被罗昊给耍了，磨着牙齿，美眸瞪着罗昊，说道：“罗小昊，信不信回去以后我让你睡两个月的沙发！记住，是60天，按天数计算！”
<br/>　　面对云熙的威胁，罗昊却是不为所动，说道：“云小熙，你不说我还忘了，回去以后我还有一笔账要跟你算呢，到时候谁求谁还不清楚呢。”
<br/>　　云熙疑惑的看着罗昊，显然是想要弄明白罗昊那句话是什么意思，但是罗昊却把话题回避开了，不管云熙怎么追问，罗昊就是不做任何回答，令得云熙郁闷不已。
<br/>　　“看到你们，就想到了我年轻时候的样子。”詹姆斯拿出一盒烟，冲出一根叼在嘴里，然后把烟盒递给罗昊。
<br/>　　罗昊摆了摆手，说道：“我的妻子不喜欢我抽烟。”
<br/>　　云熙这个时候，识趣的跟罗昊和詹姆斯打了声招呼，便转身离开了航母甲板，返回了舰桥。
<br/>　　詹姆斯对于罗昊的拒绝，也没有在意，收起烟盒，拿出打火机把烟点燃，吸了一口，说道：“你是一个优秀的军人，也是一个合格的丈夫。”
<br/>　　“将军，您是想说，这也像您年轻的时候吗？”罗昊对詹姆斯问道。
<br/>　　詹姆斯哈哈一笑，说道：“不，你比我年轻的时候要优秀。如果我年轻的时候像你这么优秀，或许我现在已经是下一届总统的候选人了。看到你们，我想起了当年拿破仑对你们国家所评价的那句话。”
<br/>　　“中国，是一头沉睡的雄狮吗？”罗昊对詹姆斯说道。
<br/>　　“不，是后面那句话。”詹姆斯摇头道：“既然如此，那就让它永远都不要苏醒过来。”
<br/>　　罗昊脸色一凛，有些警惕的看着詹姆斯。
<br/>　　詹姆斯眼睛看着海面，没有去看罗昊，语气平淡的说道：“但其实五角大楼里那些家伙都知道，我们已经失去了让‘雄狮’永远不要苏醒的绝佳机会，只是他们都不愿意承认，自己欺骗自己而已，但事实就是事实，并不是不愿意承认就会被改变的。”
<br/>　　“如果可以选择。”詹姆斯弹飞手中的烟头，看着罗昊说道：“我不想跟你们在战场有第二次交锋，我的祖父曾经在第八集团军服役，是沃克将军手下一名上校，他参加了当年那场战争，跟你们国家的军队交过手，回国之后，他跟我父亲还有还在读书的我告诫，如果可以，不要跟你们国家的军队成为敌人，那是一支可怕到会令人做噩梦的军队，他们或许没有先进的武器装备，也没有强大的后勤保障，但是他们有着强大的意志，在这样的意志之下，他们能激发出强大的战斗力，祖父说，如果是相同的装备条件下，号称‘王牌’的第八集团军根本不堪一击。”
<br/>　　这一老一少，两个国家的军人站在航母甲板上，看着黄昏下的印度洋，沉默着，各自思考着自己心中的事情。良久，罗昊扭头看向詹姆斯，说道：“将军，如果可以的话，我想见一见‘八爪鱼’。”
<br/>　　詹姆斯沉默了一会儿，随后还是点了点头，答应了罗昊的要求。
<br/>　　罗昊在一间船舱中见到了Parker，这个代号“八爪鱼”，被西伦科特极度信任的作战参谋。但是此时罗昊眼前的Parker跟罗昊在军事监狱中抓到他时，所见到的样子完全不一样。
<br/>　　尽管Parker身上没有任何的伤痕，可是他整个人已经没有了精气神，被捆绑在椅子上，目光呆滞的看着头顶的电灯，眼睛隔了很长一段时间才会眨动一下，就连罗昊靠近他的时候，他都没有察觉到。罗昊能感觉到，此时的Parker跟自己之前见到的Parker根本就是两个人！
