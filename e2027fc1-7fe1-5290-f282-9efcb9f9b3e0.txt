　　实际上，到了后来李承乾已经没什么理由可以再推托了，奈之下从宜春宫的大殿中抽了两瓶酒出来，塞给了程小四：“小四，藏好了，别记你那两个哥哥看到，等你爹回家之后交给他。”
<br/>　　“这是什么？”程小四接过玻璃瓶装的白酒，好奇的问道。
<br/>　　“酒，上元时要拍卖的酒。”
<br/>　　“这就是那种仙酒？”程小四眼睛瞪的老大，不可置信的看着李承乾。
<br/>　　小丫头完全没想到，太子哥哥会把仙酒交给自己，让自己拿回家跟老爹炫耀，犹豫了一下之后又把酒还给李承乾：“太子哥哥，我知道那个玻璃大棚已经被皇后娘娘收走了，跟你要蔬菜是为难你，以后我不要了，这酒你还是收回去吧，交给我的话，我爹一定会给喝掉的！”
<br/>　　“没事，这酒你太子哥哥还有好多，送给你就是给你爹喝的，不过酒喝了之后，瓶子千万要给我拿回来！”李承乾嘿嘿一笑，伸手在程小四的鼻子上刮了一下，满不在乎的说道。
<br/>　　大殿里所有的酒加到一起成本还不到十贯，拿出一两瓶送人并不是什么大事。
<br/>　　只是那玻璃的酒瓶不是很多，玻璃工坊加班加点的也就弄出二十来个，如果被老程给匿下，那可是一大损失。
<br/>　　“那好，谢谢太子哥哥！”程小四露齿一笑，开心将两只酒瓶抱在怀里。
<br/>　　上一次跟老程说能搞到昆仑紫瓜，结果好几天没拿回去，总是觉得老爹的眼神不对，现在拿了仙酒回去，估计老爹不会再认为自己吹牛了吧！
<br/>　　“小丫头，一句谢谢就完啦？”李承乾故意调侃道。
<br/>　　“那，那还要怎么样？”小丫头大概也觉得一句谢谢有些不大够用，被李承乾一问顿时显得有些尴尬。
<br/>　　“吃掉哥哥那么多好东西，亲一下不过份吧？”李承乾往前凑了凑，开始调戏小丫头。
<br/>　　“啊？！”程小四呆了一呆，粉嫩的小脸刷的一下变的通红。
<br/>　　纵然是在家里听娘亲说过玉佩定亲的事，而且小姑娘对李承乾也有些意思，但亲一下这样的事情还是有些超过小丫头的理解范围。
<br/>　　这可是光天化日之下啊，身边还有那么多人，几个要好的小姐妹还在不远处的房间里斗地主，自己怎么可能做这种羞人的事情。
<br/>　　小姑娘很纠结，双手抱着两瓶酒扭捏着不知应该如何是好。
<br/>　　亲，太羞人了；不亲，如果太子哥哥不高兴怎么办？
<br/>　　李承乾却不管那些，只是盯着小丫头瞧着。
<br/>　　调戏小丫头的感觉真是太有意思了，呆萌的小萝莉傻傻的，不调戏一下真是对不起这么多年的培养了。
<br/>　　僵持良久之后，小丫头终于狠了狠心，趁着几个小伙伴打扑克没有注意到这边的时间，飞快的在李承乾的脸上啄了一口，然后就抱着酒瓶飞一般的逃开了。
<br/>　　“小四，你怎么了？脸怎么那么红啊？”冲回到小伙伴中间的程小四被长乐几个盯住，纠缠着问道。
<br/>　　“没，没有，我是热的。”程小四努力辩解着，掩饰着自己的已经快要跳出来的小心脏。
<br/>　　“什么嘛，外面明明就是很冷的好吧！”小武第一个表示不信。
<br/>　　“就是啊，小四外面那冷，你怎么会觉得热呢？不会是染了风寒吧？”长乐看看还挂着窗花的玻璃，关心的问道。
<br/>　　“没，没有，刚刚在外面冻的，然后进到房间里一下子这么暖和，自然有些热。”躲避着刚刚从外面进来的李承乾望过来的目光，程小四胡乱的说着，心里只想快点逃离这个房间，快点赶回家去。
<br/>　　“好了好了，都别再玩了，天已经晚了，小四和小雯都要回家，不能再耽搁了！”李承乾看着程小四快要急哭的表情，出面替她解围。
<br/>　　“什么嘛，才过中午不久，怎么会晚！”长乐有些不满意的嘟起嘴巴，漂亮的大眼睛在李承乾与程小四身上扫来扫去。她虽然不知道刚刚外面发生了什么，但很明显，程小四和哥哥之间一定有猫腻。
<br/>　　“现在刚刚过了除夕，家家都有一大堆的事情，你总不能把她们两个一直留在宫里吧，这样她们家里人会着急的。”李承乾随意的编了一个过得去的理由，将长乐打发到一边，随后就安排人准备马车，把两个女孩送出宫。
<br/>　　而等到两个女孩离开之后，长乐和小武立刻就把李承乾赌在房间中，开始了‘审讯’过程：“哥，你到底把小四怎么了？为什么她脸会那么红？”
<br/>　　“就是啊，太子哥哥，小四姐看上去好害羞的样子，你们两个到底发生什么事了？”
<br/>　　李承乾被问来问去，又没办法急眼，最后只能摊了摊手：“我就是送了她两瓶酒而已，她那个样子是感动的。”
<br/>　　“骗人！”长乐和小武的表达方式极度的一致，白眼加一根中指。
<br/>　　李承啧了一声，在长乐和小武每人鼻子上捏了一下：“小姑娘家不要学人家竖中指，当心被母后看到，罚你们回去抄书！”
<br/>　　“哼，大骗子！”长乐撇撇嘴，将头扭向一边。
<br/>　　“行了行了，哥啥都没干，不信你们回头问小四，看看她是不是因为那两瓶酒感动的！”
<br/>　　李承乾死猪不怕开水烫，一口咬定程小四是因为酒的事情被感动，长乐和小武也是一点办法都没有，最后只能勉为其难的相信。
<br/>　　而放开李承乾这边兄妹几个的纠缠，掉头再来看看程小四。
<br/>　　回到家里的程小四脸色已经恢复正常，在家里找了一圈，没发现老程的影子，便回了程夫人的房间，将两瓶酒拿出来，放到桌上。
<br/>　　“囡囡，这是什么？”精致的玻璃酒瓶，颜色清亮的高度白酒，程夫人脸色几乎瞬间就是一变。
<br/>　　“这是酒，太子哥哥送给爹爹的。”程小四依偎到程夫人跟前，指着两瓶酒说道：“这就是过几天要在迎宾楼拍卖的仙酒，太子哥哥说玻璃大棚被皇后娘娘收走了，新鲜的蔬菜今年可能是没有了，所以拿这两瓶酒给阿爹，算是补偿。”
