所以，夏夕绾第一次出现在枢密时，他一眼就认出了她。
<br/>
<br/>        记忆里戴着面纱的女孩已经悄然长大，出落成了现在这样绝色璀璨的模样。
<br/>
<br/>        他找了她整整一年，在时间快要结束的时候，他准备回帝都城的时候，她姗姗来迟。
<br/>
<br/>        陆子羡走上前，来到了夏夕绾的面前。
<br/>
<br/>        夏夕绾诧异的看着他，“是你？对了，那天大罗心经是你从赵立英那里拿走的么，你又帮了我一次，我回去想跟你说谢谢的，但是你已经走了，我这两天也没有看到你。”
<br/>
<br/>        陆子羡看着她，“不用谢。”
<br/>
<br/>        这时陆茵茵走了过来，“子羡，你跟夕绾认识？”
<br/>
<br/>        “子羡？这是你的名字啊？”夏夕绾还不知道他叫什么名字，这是第一次听说，她看向陆茵茵，“校长，我现在在枢密研究院，我们两个人都在药房，还是同事。”
<br/>
<br/>        “夕绾，你现在在李文清院士的枢密研究院啊，那你跟子羡真的太有缘了，我还想趁着这次聚餐介绍你们认识，没有想到你们已经是旧识了，你们相处的怎么样？”
<br/>
<br/>        这个…
<br/>
<br/>        夏夕绾俏皮的眨了一下羽捷，有点为难道，“这个可以说吗，我们没说过几句话，他一般…都趴在桌上睡觉…”
<br/>
<br/>        什么，睡觉？
<br/>
<br/>        陆茵茵抬手就打了陆子羡一下，“你这个孩子，你是不是疯了，你的时间怎么能拿来睡觉，还给夕绾留下了这么不好的印象？”
<br/>
<br/>        夏夕绾没想到陆茵茵会打陆子羡，早知道她就不说了，她仰起小脸望着陆子羡，有点不好意思，“我好像犯错了，告你状了…”
<br/>
<br/>        陆子羡看着夏夕绾，女孩声音轻轻软软的，带着几分歉意和盈盈和笑意，顾盼流转之间像只…小奶猫。
<br/>
<br/>        陆茵茵这个媒婆一直在察言观色，她见陆子羡的目光一直落在夏夕绾的身上就知道这事成了，“夕绾，我家子羡可不是什么瞌睡虫，他大概是无聊，所以睡睡觉打发一下时间的，我家子羡是医学界的天之骄子，20岁就已经是帝都院士了。”
<br/>
<br/>        这么厉害？
<br/>
<br/>        夏夕绾是见过一些帝都院士的，但他们都是老头子了，李文清院士已经算是比较年轻的了，她还真是第一次见陆子羡这种年纪的院士，他看着也才24岁左右。
<br/>
<br/>        她还以为他是进枢密混日子的呢。
<br/>
<br/>        “那真是失敬失敬了，以后请多指教。”夏夕绾勾起了红唇。
<br/>
<br/>        陆子羡看着她，清冷的薄唇也勾出了一道浅浅的弧线，她不记得他了，要不然她怎么会不知道她跟他势均力敌？
<br/>
<br/>        陆寒霆是最后到了，他一出场自然成为了全场瞩目的焦点，大家围着他恭迎了一番，他淡淡的应酬过后就去找陆茵茵了。
<br/>
<br/>        刚转了一个弯，他的脚步蓦然一顿，因为他看到了前方的陆子羡和夏夕绾。
<br/>
<br/>        明黄色的灯光从头顶洋洋洒洒的镀了下来，陆子羡和夏夕绾对面而立，一个是天之骄子帝都院士，一个是天才少女绝丽纤尘。
