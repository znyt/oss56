　　洪荒之中，经过人、妖、巫三族大战，不周断裂，天降大灾，灾难不断，洪荒重新洗牌，势力重新划分。
<br/>　　洪荒暂时陷入了诡异的平静之中，千年时间，如白云苍狗，转瞬即逝。
<br/>　　这平静的千年中，洪荒大地，杀伐不起，争斗不兴，毕竟经过三族大战，三族三败俱伤，都需要时间来休养生息，恢复战力。
<br/>　　其他各族群也因为天降灾祸，损失不少，需要埋头发展，一切平静，却又不同寻常，好似有什么大事要发生。
<br/>　　这天，太古星空，天庭之上，陡然间煞气大涨，杀气汹涌，凶气涌动。
<br/>　　一切是那么的不平凡。
<br/>　　洪荒众生都知道，那代表着妖族帝俊，已经炼制成一柄足以撼动祖巫肉身的宝剑。
<br/>　　大凶之器，屠巫剑！
<br/>　　成剑之时，杀机汹涌，但转瞬间，便内敛于身，只待宝剑出鞘，屠戮祖巫以证其名！
<br/>　　同年，盘古殿中，一样涌起惊天煞气，十二道凶物冲天而起，化为十二道巨大的凶幡，每面幡上都有一位祖巫的法相之形，煞气满满，让人一看就知道那是凶物。
<br/>　　大凶之物，戮妖幡！
<br/>　　大战之后，巫妖二族自然是没有再屠杀人族了，但他们却想到了其他的方法，来炼制屠巫剑与戮妖幡。
<br/>　　如，妖族的屠巫剑，之前活祭人族已经完成了一半，之后他们见识到了弱水的厉害，在其中加上一些弱水，还有太阳之精，星辰之精等珍贵神材，最终经过千年炼制，终于炼成这一柄凶剑。
<br/>　　当然，其中人族魂魄依旧是占了很大一部分。虽然不能再杀人族，但此次灾劫之后，洪荒之中，怨魂可不少，只不过是需要多费一些时间，他们还是收集到了足够的怨魂来炼制此剑。
<br/>　　至于巫族的戮妖幡，那就更简单了。之前屠杀的人族，他们已经炼制出十二柄半成品，之后他们用这十二柄半成品布置出十二都天神煞大阵，用来收拢天地间的无尽杀气与煞气，只要足够的煞气，这十二柄戮妖幡依旧能够成型。
<br/>　　这不，经过千年时间的吸收，十二道路杨帆终于成型了。
<br/>　　人族之地，首阳山最高峰，武宫玄天殿。
<br/>　　玄天看了抬头一眼天庭，之后再看了一眼盘古殿，眼神之中胜过一丝丝冷厉，全身杀气惊天。他口中冷声道：
<br/>　　“我人族的东西可不是那么好用的，本祖迟早要你们还回来。哼！”
<br/>　　说完，他看了一眼山顶那好似屹立万古的九重武塔，这些年经过无数妖族妖王的血祭，再加上一位祖巫的献祭，如今早已威力不凡了，比起三十三层天地玄黄玲珑宝塔也不弱了。
<br/>　　攻防一体，绝对神挡杀神，佛挡杀佛，可镇压诸神。再加上其中有数百妖王与妖神的血祭以及祖巫的活祭，这座宝塔对于巫、妖二族绝对有着天生的克制。
<br/>　　“期待下次大战，你们可不要让我失望啊。”玄天低声喃喃道。
<br/>　　“下次之后，人族必定崛起，称霸天地！”
<br/>　　……
<br/>　　洪荒如往昔一般的静默，然而这平静之下却是激流暗涌，人、巫、妖三族都在暗自准备，休养生息，扩充势力，增加实力，期待后面的大战。
<br/>　　巫族后土部落，祖巫殿。
<br/>　　“宣后羿前来见我！”
<br/>　　一道温润柔和的女声传了出来。
<br/>　　大殿之内，后土眉头紧皱，神情颇为憔悴，眼神之中，充满忧色，好似心情极为糟糕一般。
<br/>　　“是，祖巫大人。”
<br/>　　外面一位巫族大叫道，之后快步跑向远处。
<br/>　　不一会儿，后羿便来到祖巫殿。
<br/>　　“后羿见过后土祖巫，不知祖巫大人有何吩咐？”
<br/>　　后羿身材魁梧，面色有些英俊，腰间围着一张兽皮，背后背着一张大弓，一副青年模样。他走进大殿，单膝跪地，大叫道。
<br/>　　“起来，我不是跟你们说过吗，不必多礼。”
<br/>　　后土回过神来，看了一眼后羿，眼中闪过一丝满意之色，后羿是她后土部落中顶级的大巫，除了她之外最强的巫族，可与与刑天、相柳等堪比了，不过，在上次大战中，相柳大巫已经死了，只有刑天还活着。
<br/>　　后土面上恢复威严，淡淡道：
<br/>　　“我欲外出一趟，此段时间，部落内的大小事件便暂时交与你来打理，记得督促他们修炼，毕竟，我巫族的大敌可不少。”
<br/>　　“是，祖巫大人。”后羿恭敬的回应道。
<br/>　　“嗯，你去吧！”
<br/>　　“是！”
<br/>　　回了一声，后羿便退出了祖巫殿，准备督促大家修炼，毕竟，祖巫的命令不能违背。同时，他也想回去看看嫦娥，自从上次人、巫两族决裂大战，后羿便带着嫦娥回到了巫族。
<br/>　　说来也奇怪，嫦娥虽然记忆有缺，回忆不起过去的东西，但她之后修炼人族的武道，却是进步极快，容颜一点也不显老，依旧如二十来岁的女神。
<br/>　　“哎！”
<br/>　　悠悠地一声长叹，自从人族与巫族大战，祝融祖巫与共工被人族玄天所杀，奢比尸祖巫被武祖玄天镇压（她此时还不知那位祖巫已经被玄天祭塔了），后土的心情就一直不佳，郁郁寡欢，心中对巫族的未来担忧不已。
<br/>　　人族迅速崛起，斩杀无数妖族与巫族，族中强者倍出，而巫族之中呢，本身人数就极少，祖巫更是陨落了三位，大巫、中巫、小巫更是死伤无数，现如今人数更少了。这要如何对付即将的到来的三族大战，而且到时巫族的敌人可就不只是妖族了，还要面对人族的无数强者。一想到此处，她就心忧不已，不知该如何是好。
<br/>　　后土带着心事，独自一人离开了部落，踏上了洪荒大地。
<br/>　　一步走来，走走停停，她看到大劫之下无数的怨魂厉鬼，有人族的，有妖族的，也有其他种族的。想到当初正是因为人族怨魂一事，巫族才会屠杀人族，之后才有祖巫陨落之事，她一时间便心痛不已。
<br/>　　“这平静的大地，不久便会重新燃起战火，到时候，不知又会有多少生灵丧生于战火之中？”一想到不久后，在这片美好的洪荒大地之上，会再次掀起无尽战火，到时候恐怕又会有许多生灵陨落在这洪荒大地，她就心思沉重。
<br/>　　血染苍茫大地，不亦悲乎！
<br/>　　“这些陨落后留下的无数怨魂、冤鬼，厉鬼，又该如何？难道就这么任由他们在这洪荒之中，数年之后自动消散或者化成厉鬼危害苍生吗？”后土眉头紧皱，暗暗想到。
<br/>　　“难道这天地间就没有这些死者的托生之所吗？”
<br/>　　这一刻，好似有一道灵光在后土心中流淌而过，快如闪电，让人抓不住痕迹。
<br/>　　“是什么？”
