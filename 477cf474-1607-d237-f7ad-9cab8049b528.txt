　　第412章
<br/>　　龙椅摆在玉石台阶上，每一块玉石，都是从南方的大山中挖出来的。然后由大船逆流而上，运到长安。
<br/>　　需要挖掘河道，动用上万民夫，平均运送一块玉石所需要的花费，够一座普通小镇吃喝三年。至于玉石本身的价值……好吧，其实没什么价值，因为除了皇族以外，没人用得起。
<br/>　　什么叫皇族，皇族不用最好的，也不用最贵的，皇族只许我有而不许别人有。
<br/>　　如今，这张龙椅就摆在林少羽面前，帝国九万里，他离得最近。
<br/>　　纹以威龙，涂以金漆，九龙出水，吞云吐雾。
<br/>　　林少羽从没想过称帝，自己有多大本事，他还是心知肚明的。从一开始，他打下长安的目的，就是奉程大雷为君。他是这么说的，也是这么想的，接下来还打算这么做。
<br/>　　但这把椅子如今离他太近了，近到触手可及。但这张椅子象征了帝国最高权力，而对于帝国亿万子民来说，莫说坐上去，便是看一眼的福分也没有。
<br/>　　没有人能抵挡这样的诱惑，林少羽也不能。
<br/>　　某一瞬间，他体内的血液滚烫起来，心脏有力的跳动。
<br/>　　从青牛山到扬州，从扬州到凉州，从凉州再到青牛山，最后又回到长安。
<br/>　　他赤手空拳，拉起一支队伍，十万人奉他为主。一路上，他击败了帝国无数名将，攻破了本以为不可能攻破的长安城。
<br/>　　如今，谁提起林少羽的名字，不郑重对待。他已不是青牛山那个一无所有的愣头青，他现在是令天下人胆寒的林少羽。
<br/>　　他有资格坐上这张椅子，哪怕只是坐一坐。
<br/>　　林少羽拾阶而上，一步一步跨上玉石台阶，随着渐渐靠近龙椅，呼吸越来越凝重。
<br/>　　呵，还真不是一件轻松的事情呐。也不知明帝每天走这一步路，是何种心情。
<br/>　　万万人组成的帝国，却只有一个人能够坐在这张椅子上。简单来说，如果谁有机会登上这座金殿，冲明帝磕个头，在帝国都算得上光宗耀祖，何况坐在这张椅子上。
<br/>　　今天，林少羽在这张椅子上坐下来。
<br/>　　还真是硬得不舒服呐，林少羽下意识直起肩膀，大殿空空荡荡，一无所有。
<br/>　　面前，却有九万里江山。
<br/>　　视线慢慢拉高，穿过大殿的屋顶，来到皇宫的上空。起义军攻破宫门，对溃败的鱼龙卫赶尽杀绝。宫中的宫女、太监、公主、皇妃等等诸如，面临着一个个杀人魔头。精致的屋子被撞开，养尊处优的公主被扒光衣服摁在地上……
<br/>　　视线再拉高，整个长安城内有战斗，有逃亡，有火光，有趁乱为非作歹者。孩子的头颅被挑在空中，纵马招摇而过。财主的金库大门被砸开，金银珠宝，翡翠玉石不计其数。那些一个个锦衣玉食，高贵的如同生活在云巅的千金小姐，如今被狠狠摁在泥土里，卑贱不如猪狗。
<br/>　　城外山峰上，立着两道人影，一老一少，一僧一道。
<br/>　　小道士红尘了看着这一幕，那些女人的哭声、父母的悲号似乎可以透过夜幕传到这里。
<br/>　　“他们怎么可以这样，他们怎么可以这样！”小道士咬牙切齿。
<br/>　　“看仔细了，这就是乱世，乱世人不如狗，本就是这样。”青叶佛缓缓道。
<br/>　　“原来乱世……”小道士呆呆道：“这么可怕呐。”
<br/>　　……
<br/>　　凉州城，城主府。
<br/>　　李行哉歪在罗汉床上，听宋伯康讲述有关长安城的情报。两地相隔太远，如今凉州城什么情况，其实宋伯康也不知道。但仅仅靠猜测，李行哉也能猜出个大概。
<br/>　　“如今的凉州城是一片乱局，只等着殿下出兵，主持大局。”
<br/>　　李行哉呵地一声笑，道：“那什么时候能够出兵，该不会又要拖个把月吧。”
<br/>　　宋伯康抬起头：“三天后。”
<br/>　　李行哉一怔：“怎么快！”
<br/>　　“国难当头，匪贼做乱，就连三天，臣都觉得太慢了。”宋伯康长叹一口气。
<br/>　　筹备粮草，点起兵马，十万凉州兵兴师动众，直扑长安城。宋伯康的迫切程度，连李行哉都觉得惊讶。如此重要的出兵，他甚至都没准备任何祭天仪式。
<br/>　　筹备用了三天，但路上却需要更多时间，大概有月余才能抵达长安附近。这支十万人的部队，以李行哉为首，人人头缠白巾，号称为明帝复仇。但所有人都明白，李行哉只是一个幌子，就连李行哉都明白这件事。
<br/>　　他每天在军营中乱转，与士兵同吃同住，共同搬运粮草，脚踩污泥，浑身臭汗也毫不在意。
<br/>　　李行哉不是不能吃苦，他吃得苦头甚至比许多普通士兵都多。大家一起干活，一起骂脏话，一起讨论女人……
<br/>　　对于李行哉的用意，宋伯康不是不懂。甚至一开始就预料到了，不过双方谁也没有挑明。有时行军途中遇到一些问题，诸人谁也没有办法，反而是李行哉能够高屋建瓴，给出可行的法子。
<br/>　　便是领军的戴白都对李行哉刮目相看，这明显不是一个废物呐。
<br/>　　李行哉流浪天下，考察各地的风土人情，在蛤蟆寨又主要负责商队。那时，他的目的可不是做个大财主，他的目的一开始就是谋天下。
<br/>　　宋伯康昭告天下，帝国六王子要杀贼复仇，号召天下忠君爱国之士，共同杀敌。同时，李行哉依旧小心谨慎经营着自己的事情。
<br/>　　他此时也明白，到长安城后，自己怕是好和曾经的哥哥弟弟争一争。宋伯康是敌人，但现在是朋友，而想要争这天下，只靠一个六皇子的幌子是没用的，必须有自己的人手。
<br/>　　帝王要有海纳百川的胸怀，什么朋友敌人，什么爱恨情仇，能为我所用，便是工具。
<br/>　　一月后，凉州兵赶到长安城，李行哉也见到了哪位崔相大人。他竟然没死，真是令李行哉心底大呼可惜。
<br/>　　崔相见到李行哉时，未语先泪，李行哉也是面带悲色，二人谈了很久，都要除贼保江山。
<br/>　　至于长安城中的林少羽……
<br/>　　他们已在各路诸侯的联手攻击下被击溃，就连十日都未撑得下去。
