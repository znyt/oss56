        &lt;h3&gt;第三卷第四百零五章  工作交接&lt;/h3&gt;
<br/>
<br/>        陈子乔接到了调令，她调到了下属单位，一个比较清闲的的单位，没有什么实权也没有什么经济往来，适合养老。
<br/>
<br/>        而来和她进行工作交接的人是夏颜。
<br/>
<br/>        陈子乔叹息了一声，老马真的是宠爱夏颜，这么多年一步步将夏颜往起提，年轻轻轻的能到这个位置真的是极少有人，夏颜是特列了。
<br/>
<br/>        “陈姐，你这年纪轻轻的去那么闲的部门，这不符合你的风格啊。”夏颜在工作交接完毕之后和陈子乔聊开了。
<br/>
<br/>        “孩子上学呢，两个都是毕业生，我得盯着点。”陈子乔微笑着说。
<br/>
<br/>        “陆二叔最近有和你联系吗？”夏颜关切的问陈子乔。
<br/>
<br/>        “陆二叔？”陈子乔不明所以，她和陆家老二的关系只有老马知道。
<br/>
<br/>        “是啊，二叔对你那么好，不可能不和你联系吧？”
<br/>
<br/>        “我不知道你说的陆二叔是谁，我也不曾和任何人有非常好的关系，工作交接完了，如果没有别的事情我就走了。”
<br/>
<br/>        “蔡少芬是我干妈，她们妯娌之间的关系是非常好的，陈姐现在从这里出去，你能保证以后能够安稳幸福的生活吗？”
<br/>
<br/>        “人不犯我我不犯人，人若犯我我必敬之，我呢，不像我们家子昂一样有着一颗圣母心，大家相安无事最好，如果我过得不安稳大家都会不安稳的，你说呢夏总？我现在在您的手下工作，请多多关照，子乔过得安稳，夏总也会安稳的。”陈子乔转过身微笑着看着夏颜。
<br/>
<br/>        “那是当然，你是二叔的人，我当然会照顾好你的。”夏颜暗含讽刺。
<br/>
<br/>        “我是谁的人不重要，重要的是我底子硬，正儿八经科班出身，比有些冒牌货要好，至于你的二叔我压根不知道是谁，夏总大可不必看谁的面子照顾我，别为难我就好！”
<br/>
<br/>        “你……”
<br/>
<br/>        “夏总，祝你工作顺利，万事如意！”陈子乔转身头也不回的离开。
<br/>
<br/>        夏颜是老马的人也是陆家的人，之前有些尾工也进行了交接，有些签字盖章的陈子乔一直都没有动，现在全部交给夏颜，她深深的吐了一口气。
<br/>
<br/>        自从三姐陈子澜提醒她之后，有些事情她就不经手，要经手也是有人同意了她才签字，她不敢说她是明哲保身，最起码不要让自己越陷越深。
<br/>
<br/>        陈子昂送孩子出国，只有三姐陈子澜知道，别人是不知道的，陈子乔将工作交接之后，给陈子昂打了个电话，电话无人接听，然后给陈子昂发了个消息。
<br/>
<br/>        陈子乔交接工作之后，感觉一身轻松，她开车去看了她的父母亲，父亲已经出院在西城陈子昂的家里住着，原来他们是住不习惯的，现在住这里去医院方便。
<br/>
<br/>        陈元良心境很好，身体恢复得很好，连大夫都说陈元良超过了他的期望值，冯睿离开了，俩个老人还亲自去送了一程，白发人送黑发人，心中自是哀伤了一阵，不过俩个老人对生死已经看得很开，而他们多冯睿的死因也没有怀疑，只是安慰陈子建和陈天天说：“生死有命，富贵在天，冯睿走了，但是活着的人还要继续，你们要好好的活着，冯睿在天之灵才能安息。别弄着牵牵盼盼的，她走得都不安生。”
<br/>
<br/>        陈子乔提着水果去陪着父母亲坐了一会，便提出说要去看看陈子澜，明天过来给爸爸妈妈做饭吃。
<br/>
<br/>        “子乔，你上班就不要来来去去的了，我们现在都很好，饭自己也能做，况且还有阿姨在，有什么事情我给你们打电话。”冉灵卉将陈子乔送到门口。
<br/>
<br/>        “我最近调工作了，比原来要轻松很多，以后有时间陪你们了，我回去将家里收拾收拾，你和我爸一起去我家里住。”陈子乔开心的说，以前都没有时间陪父母亲，想想真的是很后悔，辛苦半辈子，其实都是为了别人。
<br/>
<br/>        “你没有啥事情吧？”冉灵卉听说陈子乔掉工作了，有些担心。
<br/>
<br/>        “都好着呢，俩个娃不是要考学了吗，我求着人家给我找了一份闲职，这样好照顾俩个孩子。”
<br/>
<br/>        “那就好，你去看看你三姐早点回家吧。”冉灵卉看着陈子乔开车离开，摇摇头，孩子们都大了，各有各家，只要他们平平安安就好。
<br/>
<br/>        陈子乔到陈子澜病房的时候，陈天天也在，也不知道陈天天和陈子澜在说什么，陈子澜一天凝重。
<br/>
<br/>        “三姐，天天今天咋有时间来？”陈子乔看着陈天天，最近陈天天说是要写论文，忙得都看不见人。
<br/>
<br/>        “我今天没有课，我就过来看一下三姑，四姑，你咋也来了？”
<br/>
<br/>        “今天开心，把手中的工作终于交出去了，以后我都会很轻松的了。”陈子乔将买的水果放在桌子上，顺手拿了三个去小厨房洗。
<br/>
<br/>        “接你工作的是谁?”陈子澜问陈子乔。
<br/>
<br/>        “夏颜！”
<br/>
<br/>        “夏颜还真是飞黄腾达，这几年疯了一样的往上升啊。”
<br/>
<br/>        “呵，西城疯传夏颜和马总，夏颜还厚颜无耻的说她认识蔡少芬，蔡少芬是她干妈，掩耳盗铃。”
<br/>
<br/>        “蔡少芬是谁？”陈天天不知道蔡少芬是谁。
<br/>
<br/>        “陆战君的老婆。”
<br/>
<br/>        “陆战君？”
<br/>
<br/>        “哎呀，你个小孩子问那么多干什么？就是北京一个老总，很厉害的一个人物！”陈子乔手中削着水果。
<br/>
<br/>        “陆家的女儿陆玉是你小姑前男友张函的妻子，说句难听的话，要不是陆玉，你小姑可能就会和张函在一起。”陈子澜看了一眼陈天天。
<br/>
<br/>        “夏颜的干妈还真多，我妈也曾经是她的干妈，这又搭上了北京的老总。”陈天天嗤笑一下。
<br/>
<br/>        “有些人只要能用得上，你让她叫爷都行。”陈子乔也笑着说。
<br/>
<br/>        “你们俩个人无聊不，来看我呢还是来讨论夏颜？”陈子澜笑骂着陈子乔和陈天天。
<br/>
<br/>        冯睿的房子留给了陈伟冯，陈子昂在西城为陈子建买了一套房，和她离得不远，也是江氏集团开发的楼盘，这样方便陈子建照顾俩个老人，但是陈子霖家里产业也需要人照顾，二秀和陈子建就又回到了泉城上湾。
<br/>
<br/>        陈天天回到了陈子霖的家里，一如既往地叫着陈子霖爸爸，严慧芝为妈妈，和弟妹们一起恢复到日常的生活中，上学放学写作业，只是陈天天再也没有了往昔的欢快，偶尔也会去公司，偶尔也会和易倾秋一起出去聚餐。
<br/>
<br/>        陈子乔和陈天天从陈子澜处一起出来，陈天天准备回家，想了一想，她开着车到了冯睿家里，曾经冯睿给过她钥匙。
