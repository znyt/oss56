<br/>　　挥手之间一掌轰杀九劫古帝，这代表着楚风眠的实力，至少已经是达到了半步天人境的境界。

<br/>　　楚风眠这个名字从大罗天星域之中响彻而起都没有多久的时间，现在楚风眠转眼之间，都已经是从一个小家伙，成为了媲美半步天人境的存在。

<br/>　　楚风眠也懒得理会这些人的震惊，他的目光看向了日炎殿主跟青玄殿主，冰冷的开口道。
<br/>　　今日这些人，只有两条路，臣服，或者死。

<br/>　　那日炎殿主，青玄殿主听到楚风眠的话，毫不犹豫便是化为了一道遁光，直接向着远方飞了过去。

<br/>　　看着逃走的两道遁光，楚风眠眼神冰冷无比，他不愿意大开杀戒，可既然不愿意臣服，那也就只有死路一条了。
<br/>　　轰！
<br/>　　楚风眠的掌心之中灵力凝聚，突然一掌向着天空之上打去，这一掌瞬间跨越了千万里的空间。
<br/>　　猛然之间轰击在了那日炎殿主，青玄殿主的身躯之上。

<br/>　　这两位五行天的殿主，五行天的九劫古帝，在楚风眠的一掌之下，身躯被凌空打爆。
<br/>　　他们的精血，全部都被吞天秘术所吞噬，化为了神树的给养。

<br/>　　从楚风眠背后的天涯剑匣之中，一道道剑光突然之间飞了出来，所有想要逃走的五行天弟子，无一例外，都是被剑光所斩杀。

<br/>　　不过还是有一些五行天弟子，已经是明白了眼前的局面，看向楚风眠，也不在反抗，立刻跪倒在了楚风眠的面前。

<br/>　　有第一个，就有第二个，很快不少的五行天弟子，都是选择了臣服，甚至是有些五行天的长老，古帝级别的人物，都是选择了臣服。

<br/>　　在这些选择臣服的古帝殿主面前，还有一位九劫古帝，五行天的林天殿主，也都是选择了臣服，跪在了楚风眠的面前。

<br/>　　这林天殿主，身为九劫古帝，也是五行天的高层了，应该有资格知晓宝库的所在。
<br/>　　“是。”
<br/>　　林天殿主听到楚风眠的话，便是无比恭敬的起身站了起来，化为了一道遁光，带着楚风眠向着五行天内部飞去。
<br/>　　看起来现在这林天殿主，对楚风眠已经是彻底服从了，听从了楚风眠的命令。
