　　可介绍上，并没有标明这块砖头的用途，刘哔问了系统，也没能得到任何回复。
<br/>　　名称：灵砖
<br/>　　种类：特殊物品
<br/>　　用途：未知
<br/>　　介绍：似乎只是一块普通的砖头，上面带着一些特殊的波动，不知道有什么作用。
<br/>　　将灵砖和之前兑换出来的鬼脸面具收入空间戒指，刘哔关闭了系统，目前为止，他也终于消化完了这场战斗的收获。
<br/>　　他的战斗力也得到了一个质的飞跃。
<br/>　　看了一眼房间内的落地钟，现在的时间已经是凌晨两点多钟了，没有照明工具，刘哔打算在这里住一宿再离开。
<br/>　　马上就要离开了，他想趁着这不到一个月的时间，找一处落脚地，然后将解封紫女的任务完成。
<br/>　　至于九叔传承的任务，应该是不可能完成了，他现在的状态，根本不适合在九叔面前出现。
<br/>　　计划完接下来的行程，刘哔盘坐在床上修炼了起来。
<br/>　　不得不说，丧尸并没有睡眠的需求，利用夜晚的这些时间，他的修炼速度比别人快了三分之一有余。
<br/>　　再加上他的天赋本就不差，紫女给传授的功法前期本来就进展神速，以至于许多修炼多年的修士实力都不及刘哔。
<br/>　　一夜很快过去，身处这样一个环境，刘哔自然也不敢进入过深的修炼状态，当第一缕晨光照进屋子里的时候，他就终止了修炼，下床走了出去。
<br/>　　对于要去什么地方落脚，他的心里早有了计划，之前在九叔家中的时候，他就有一个猜想，就是不同电影的位面，是否可以连接起来。
<br/>　　就如同在《僵尸先生》和《僵尸叔叔》这两部电影，在原著中就出自同一个世界，那《僵尸叔叔》里面的剧情，在这个世界中又是否会继续展开？
<br/>　　假使这样，趁着另一个世界中的剧情还未展开，他强行去做一些改变，又将会起到什么样的效果。
<br/>　　心中有这样的打算，他也早早从文才那里套出了四目道长的住址，虽然距离这里并不相近，但以他现在的脚程来说，也只需要两三天的时间即可到达。
<br/>　　辨认了一下自身所处的位置，刘哔即刻启程，向四目道长居住的地方行去。
<br/>　　……
<br/>　　一路上夜伏昼出，虽然已经在可以规避些什么，但还是发生了一点小小的意外。
<br/>　　大概第二天夜晚，刘哔没找到留宿的地方，被迫无奈，只能在荒郊野地凑活一晚，半夜的时候，倒是招来了一些不干净的东西。
<br/>　　不过，这些鬼物的实力都不算很强，没花费多少工夫，他就将这些鬼物悉数消灭了。
<br/>　　之后一路顺畅，第三天的下午，刘哔来到了一处山脚下。
<br/>　　这座山叫做大泽山，说是山，却不怎么巍峨，反倒是道路崎岖无比，山上的植物倒也很是茂盛。
<br/>　　似乎是刚刚下过雨，山路有些泥泞，时不时还会碰到几汪积水，挡住了原本就不宽敞的山路。
<br/>　　但有飞行能力的刘哔，并没有花费多少力气，就来到了半山腰的地方。
<br/>　　前方传来‘哗哗’流水的声音，让他停住了脚步，这几天忙于赶路，他还没腾出时间好好洗漱一下，此时探到水源，他自然忍不住想要停下洗把脸了。
<br/>　　稍稍改变了了一下方向，刘哔拨开山上茂密的植物，看到了一处水源清澈的小湖，一群通体乌黑的乌鸦，正在湖边饮水嬉戏着。
<br/>　　刘哔挥动翅膀，轻盈的飞到湖边，用手鞠起一捧清水，洒在了自己脸上。
<br/>　　夏末的水有些冰凉，几天没有清理的脸庞遇水，有些不适，却也不想一开始那样如同死皮一般了。
<br/>　　直至此时他才察觉到，自己的体质似乎发生了一些改变，已经可以轻微感应到一些知觉了。
<br/>　　清脆的落水声惊动了远处的乌鸦，待看到刘哔之后，纷纷挥动翅膀聚集了过来，‘呜呜哇哇’的叫着，落在他背后的黑色翅膀上。
<br/>　　似乎，这些家伙是把他当成了同类了。
<br/>　　心中有些好笑，但刘哔并没有驱赶它们，而是任凭其在自己翅膀上啄着上面的沙石。
<br/>　　旁人都视乌鸦为不详，认为它们是魔鬼的化身，见到必定会有不幸发生，但刘哔对此，却没有一点忌讳。
<br/>　　甚至他有时候就很难理解，只因为子虚乌有的传言，就如此不待见它们，甚至滥杀，这样的做法，倒更像是魔鬼。
<br/>　　仔细将脸庞和头发梳洗了一遍，刘哔还未来得及将头发拧干，就感应到附近有人过来了。
<br/>　　据文才所说，这个山上就只有四目道长师徒，以及一个老和尚居住，怕被人看见真身，刘哔立即收起了翅膀。
<br/>　　按照原著里的实力对比，四目道长应该和九叔是一个等级的，另一个老和尚，也相差不到哪去。
<br/>　　筑基期的修士，他现在应对起来并不困难，但也没有百分百的把握可以将其留住。
<br/>　　毕竟修士和僵尸不同，到了筑基这个级别，每个修士都有自己的保命手段，先前的鬼修老头，要不是自己先让他感染了丧尸病毒，在其虚弱的时候又趁机出手，恐怕也根本不可能击杀他。
<br/>　　别说筑基，就是那个鬼修少年，如果当时不是抱着必死的决心，他也很难将他击杀掉。
<br/>　　虽然现在实力有了一个大幅度提升，但刘哔也不敢有丝毫大意。
<br/>　　如果来人是四目道长，让他发现了自己的真身，而他又没有将其留住，那引来的麻烦，可就大了。
<br/>　　将一切准备好，刘哔站在一块长满斑驳的青石旁，拧着自己未干的长发，四周脚边，一群乌鸦环绕，莫名显得有些诡异。
<br/>　　因为之前的战斗，刘哔之前衣物已经破碎，此时他身上的，是从鬼修宅院搜索来的粗布麻衣。
<br/>　　这套衣物本来是属于那鬼修少年的，对他来说并不合身，却像是紧身衣一般，将其修长的身材给勾勒了出来。
<br/>　　一切准备妥善之后，刘哔将目光投向了远处，前方被茂密的植被遮掩，根本看不清后面的情景。
<br/>　　他很想知道，来的这个家伙，到底是谁。
