        &lt;h3&gt;第七十一章  寒伯安出国考察&lt;/h3&gt;
<br/>
<br/>        寒伯安打开车门，大长腿一跨，坐在驾驶室里，顺手拿起一支雪茄，用手捋捋，天气太热，雪茄放在车上都快晒爆了。
<br/>
<br/>        他在美国留学四年，对这里已经是轻车熟路了，而且寒家本来也有公司在这里，他既是来看看寒家的分公司，也是考察新的项目。
<br/>
<br/>        这一次考察非常满意，在心中已经有了一个大体的轮廓，每一个新生的事务都有一个孕育期、初生期、成长期、他们的这个事情没有三至五年是下不来的，传统的覆盖网点需要大量的人力来进行管理，如果再加上像物流上运用的app那样的操作，可以及时进行沟通，也能让管理处于新的水准。
<br/>
<br/>        郝景文已经将他们的设想方案给了陈子昂，不知道陈子昂有何设想？
<br/>
<br/>        也许是江俞轩的原因，他们愿意给陈子昂一个尝试的机会，但是他们的创业是只许成功！
<br/>
<br/>        寒家、郝家、江家设计的产业不一样，但是他们几家老人却是很好的朋友，相对来说他们年轻的一辈也是好朋友。
<br/>
<br/>        他将他们三个人聚在一起，没有利用家族的资源，拓展了一个全新的业务，对他们来说就是一种全新的锻炼，他们不想躺在家族的功劳簿上。
<br/>
<br/>        他们更需要新鲜的血液来完成他们这一代人的使命。
<br/>
<br/>        每个时代都有每个时代的特色，每个年龄段的人都会背负着不同的理念和使命，他们的这一代人是时代的宠儿，也是时代的缔造者。
<br/>
<br/>        国家强盛，百花齐开，只要你抓住机会，便能成就一番事业，寒伯安正是看中了这一点，才另辟捷径。
<br/>
<br/>        他拿着烟放在鼻子低下嗅了嗅，他已经有了很多年的吸烟历史了，他很喜欢雪茄的味道，轻轻的打着火，深深的吸一口，满足的靠着他的座驾。
<br/>
<br/>        汽车的冷风里弥漫着雪茄的味道。
<br/>
<br/>        安男此次也和他一起，不过安男却是去了他们曾经上学的地方，他在学校附近给安男定了酒店后，他开着车跑了好几个地方，和几个汽车租赁服务巨头聊了聊国外的现状和需要改进的地方。
<br/>
<br/>        “虽然全国都有这个行业，但是每个地区的消费水平不一样，我们也会设置很多的消费等级，但是服务态度是不会改变的。”某一巨头说。
<br/>
<br/>        “我们的价值体现在服务上，价格一般不存在太多差异，因为会有统一的收费标准。”另一个巨头说。
<br/>
<br/>        他们口中都一致的强调服务，他们将来该如何做好服务呢？
<br/>
<br/>        寒伯安掐灭了雪茄，发动汽车。
<br/>
<br/>        他需要去体验一下别人的服务态度。
<br/>
<br/>        在他们目前的物流行业里，他们注重的是服务性和实效性，从某一点来说还是和国际接轨的。
<br/>
<br/>        寒伯安将车开到安男住的酒店，他本来还想上去看看安男回来了没有，但是想了一想还是转身离开，他喜欢一鼓作气的将事情做完。
<br/>
<br/>        寒伯安打电话租了一辆车，司机穿得就和绅士一样文质彬彬，他并没有目的地想去，不如就随性而走？
<br/>
<br/>        穿过异国风情的街道，他不时的和司机聊着天，司机对着这个黄皮肤的人能够说一口流利的本土话感觉很好奇，便一路走着一路聊着。
<br/>
<br/>        每个行业的规则都不一样，寒伯安也从司机的口中知道了一些规则，他心里在想：“去其糟粕，留其精华。”
<br/>
<br/>        寒伯安回到酒店的时候，安男还没有回来，外面的夕阳如火，他拿出相机，伸出手，拍了一张两指捏着夕阳的照片，火红色的太阳合着周围的云彩，都被他捏在了手中。
<br/>
<br/>        安男披着齐腰的如瀑布一样柔软的长发，踩着轻盈的脚步打开了房间的门。
<br/>
<br/>        “欢迎回来。”寒伯安变戏法一样拿出一支鲜艳生活的玫瑰花。
<br/>
<br/>        “睿琪。”安男没有接过寒伯安手中的玫瑰花，直接是扑过来，环上寒伯安的脖子。
<br/>
<br/>        “我以为你还得一会回来。事情忙完了吗？”安男温柔的声音像清泉流响。
<br/>
<br/>        “差不多了。”寒伯安环着安男的温软一握，带着安男坐在沙发上。
<br/>
<br/>        安男容貌绝佳，性格温软，不但是学校里的校花还是学霸，他们已经毕业几年了，这次回来是因为安男有个学术交流会，他也刚好来考察，所以就一起来了。
<br/>
<br/>        安男什么都好，生活和工作上都很自立，就是和前女友一样爱黏人，上次听了父亲的一席话之后，寒伯安恍然大悟。
<br/>
<br/>        在和女孩子谈恋爱期间，她黏着你，不愿意和你分开，那是她一种爱的表现，也可以说是一种隐形的占有欲，她希望你是她的，或者说她希望你心里一直都装着她。
<br/>
<br/>        如果结婚了，有孩子了，她的心思有可能被孩子分走，或者说她全部的心思都在孩子的身上，你想她黏你，她也未必有时间或者有精力去黏你。
<br/>
<br/>        从那次谈话之后，寒伯安改变了心态，他对安男很用心，虽然他不曾对安男说过一生一世一双人的话，但是在心底，他已经将她当做要相守一生的人。
<br/>
<br/>        “累吗？”寒伯安拂去安男脸上的秀发，露出安男精致的小脸，丹凤眼盛满一潭碧波，天然的红嘟嘟的樱桃唇散发着成熟的魅力。
<br/>
<br/>        “你呢？”安男反问。
<br/>
<br/>        “我不累。”
<br/>
<br/>        “我也不累。”
<br/>
<br/>        “不累，我们吃一点?”寒伯安眼里的安男绝美的容颜。
<br/>
<br/>        “现在吗?”安男看着外面美丽的夕阳，寒伯安很帅不错，可是外面的美景稍纵即逝，如果不去看会不会留下遗憾？
<br/>
<br/>        “你看，夕阳照耀下的城市笼罩在金色的朦胧里。”
<br/>
<br/>        “天涯海角，世界上任何美景都及不上你的美丽。”
<br/>
<br/>        安男那盈盈秋水里盛满火红的夕阳还有寒伯那帅气的小小的人儿，她的脸微微的红了，寒伯安的情话是这个世界上最动人的旋律。
<br/>
<br/>        安男展颜一笑：“伯安的情话，总是会人让人甜到心底。”
<br/>
<br/>        “吃饱了，你会发现整个世界都是甜的，包括我也包括你。”
<br/>
<br/>        安男那一头柔软的秀发遮住了她羞红的脸：“伯安，你就是我心中的太阳，无论何时都是光芒万丈。”
<br/>
<br/>        夕阳余晖下，一对璧人相携而行，景色如此美好，岂能辜负当下，他们踏着余晖，金色撒在身上，漫天卷起甜的蜜，安男柔软的小手放在寒伯安宽大的掌心，是的，整个世界都是甜的，包括他和她。
<br/>
<br/>        玉人相携，美景向往，寒伯安眼里只有安男，安男眼里也只有寒伯安，他们都是彼此眼里最美的风景。
