　　第429章
<br/>　　这一场战斗，程大雷的确取得了胜利，甚至可以说是完胜。
<br/>　　通过这一战，令他对五支队伍的战斗力有了判断，至少这证明持续三年的训练并不是无用功。而也借此，试探出凉州兵的战斗力：不弱，但不算强。
<br/>　　但战斗，哪有不死人的。
<br/>　　己方也为此付出了人命，有不少弟兄牺牲在战场上。牺牲是再所难免的，新兵初上阵，可能连武器都握不紧。今天，能以微小的伤亡取得胜利，还要多亏平常的训练。
<br/>　　刚才收拾战场，抢夺战利品，同时，也是为死去的兄弟收尸。
<br/>　　如秦蛮、关鱼等人都不太将这些当会事，在这个乱世，人命没有程大雷想象的那么贵，而身为一员大将，将人命当做数字是必须的素质。
<br/>　　可是，程大雷没有这种素质。
<br/>　　他一直在提醒自己，将心脏磨练得更坚硬些，想要在这个乱世活下去，活得好，柔弱是行不通的，必须坚硬起来。比这个乱世更坚硬，才不至于被乱世的车轮碾碎。
<br/>　　当程大雷还在返回蛤蟆城的路上时，陆洛已经带着残兵败将，逃回了凉州城。
<br/>　　虽然是文官，但陆洛有颗武将的心。这次高飞虎大摇大摆带人过境凉州，陆洛不可能不管。管就要管个彻底，将对方全歼在凉州境内。
<br/>　　宋伯康带人去了长安，留下陆洛管理凉州城，军政大权他一把抓。也想趁着这件事，好好露一次脸。
<br/>　　结果没能露脸，反而摔了个狗啃泥，屁股都掉在地上了。
<br/>　　“路大人！”手下的笔吏过来汇报：“宋大人回来啦。”
<br/>　　“啊！”陆洛咧开了嘴。今天刚吃了败仗，还想着怎么用春秋笔法，把这件事遮过去，怎么这时候宋伯康竟然回来了。
<br/>　　没有法子，陆洛只好硬着头皮去见宋伯康。刚走入大厅，他一张脸就搭了下来，委屈着脸道：“宋大人，您可算是回来了，那程大雷实在欺人太甚！”
<br/>　　“住口！”
<br/>　　宋伯康一张脸黑得像锅底，发生什么情况，他已经知道了。他怎么可能不知道呢，刚刚回来，便有人向他汇报此战的情况。凉州城损兵折将严重，经此一战，凉州元气大伤。
<br/>　　“我临走前便与你说过，让你坚守城池，轻易不可动兵，你为何违抗我的命令？”宋伯康冷声道。
<br/>　　“大人……”陆洛的脑袋垂下来：“有队伍从我凉州过境，我怕他们不坏好意，总不能任他们大摇大摆的过去……”
<br/>　　陆洛越说声音越低，到后来渐渐消失。
<br/>　　宋伯康看了他一眼，没有说什么。他当然明白陆洛心中的想法，如果连手下人在想什么他都不明白，那他岂不是成了李行哉那样的废物。但他也没办法如何陆洛，毕竟他身后是江南陆家，动他就是不给陆家面子。当然，经此一事，陆洛的政治生命就算是结束了，这对他来说，却是最重的惩罚。
<br/>　　“你……好自为之吧。”宋伯康叹了口气。
<br/>　　见宋伯康没有再说什么，陆洛也松了口气，他试探着问：“大人，您怎么回来了，难道长安的情况又有变化？”
<br/>　　“嗯，的确有了些变化。”宋伯康道：“我这次来，是需要押送一些粮草进京，另外也看看凉州的情况，哼，没想到你竟惹出这种事。”
<br/>　　陆洛不敢答言，这次的事情毕竟是他莽撞了。他等了一会儿，试探着问：“长安那边发生了什么？”
<br/>　　夜静无声，大殿里亮着一盏昏黄的烛火，二人压低声音，开始密谈起来。
<br/>　　……
<br/>　　经过一昼夜的奔波，程大雷的部队返回了蛤蟆城。当走入蛤蟆城时，看着眼前雄伟的城关，张大的嘴巴一时无法合拢。
<br/>　　城墙耸立，来回有巡逻的兵丁。人人腰杆笔直，两腮圆润，这起码说明，在蛤蟆城吃饱饭是不成问题的。而在如今的帝国，每天都能吃上饱饭，已是一件极其不容易的事情。
<br/>　　看来大当家在这里，很是财大气粗么！
<br/>　　城里的街道收拾的很是干净，房屋整齐排列，高飞虎已经不是吴下阿蒙。他能从街道、建筑、每个人脸上洋溢的骄傲神情，感受到一种生气。
<br/>　　这是一个正茁壮成长的地方。
<br/>　　“大当家，你这里搞得很是不错么？”
<br/>　　程大雷打着哈欠：“先吃饭，后睡觉，其他事情，等醒了后再聊。”
<br/>　　一路奔波，又打了一场硬仗，其实每个人都筋疲力尽。打仗，当真不是人干的活儿。
<br/>　　留守山寨的刘悲已带人治疗伤员，处理弟兄们的遗体，对伤亡战士的家属给予一定抚恤。有人在这场战斗中立了大功，一家人喜气洋洋。也有人的父母兄弟死在战场上，随之而来的便是几个家庭的不幸。
<br/>　　但在这个乱世，这种不幸却又是无法避免的。
<br/>　　和珅安排高飞虎的手下住进军营，又安排老弱妇孺住在民居中，同时安排好这些人，并不是一件简单的事情。而他们现在最需要的，是食物和良好的睡眠。
<br/>　　至于程大雷自己，则回到城主府，简单吃了些东西后，也沉沉进入梦乡。
<br/>　　这一觉一直睡到第二天天亮，程大雷迷迷糊糊从床上醒来，就听到外面街道上传来一阵阵喧哗声。
<br/>　　他披衣下床，走出城主府，看见街道上挤满了载歌载舞的人群。
<br/>　　有人大口饮酒，醉倒在大街上，有人站在高台上，大声唱着不成曲调的小曲。更多人则是聚在一起吹嘘，战斗中自己又杀了几个人，可以领到多少奖励。但同时也有人一言不发的喝酒，眼眶通红。
<br/>　　程大雷立刻明白了他们在做什么，这是庆祝胜利的欢宴。几乎每次打赢仗，他们都会聚在一起饮酒作乐，放肆发泄心头的恐惧，也有人借着酒精，怀念逝去的弟兄。
<br/>　　程大雷一出来，便立刻被人围住，以高飞豹为首的一伙人，堵住他要他喝酒。程大雷捧起一弯酒一饮而尽，然后赶紧找机会脱身。
<br/>　　这几个憨货都是无酒不欢的主儿，程大雷若再不走，非得被他们灌成一堆烂泥。
<br/>(本章完)
