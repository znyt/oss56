　　当时间到达3点钟的那一刻，在一片欢呼声中双方正式进入到了ban选画面。
<br/>　　这一局白队在蓝色方，而boom战队则是在紫色方，上来这一边香蕉在叶帝的指挥下直接将大树这个版本比较强势，而且香蕉又不喜欢玩的英雄给ban掉了。
<br/>　　大树这个英雄到了后期实在是太肉了，真要是打到后期，有着这个前排在，那对于他们的威胁还是很大的。
<br/>　　当然了这最关键的还是香蕉不适合这类英雄，或者说香蕉在玩那些暴力型的上单获得的收益要远比这些坦克型上单的收益大。
<br/>　　双方轮流ban人，很快便是各自将需要针对的英雄给ban掉了。
<br/>　　白队这边ban掉了：大树、蜘蛛还有沙漠皇帝这几个英雄。
<br/>　　而对面这是将：盲僧、机械先驱还有妖姬这个三个英雄给ban掉了。
<br/>　　既然不想打后期，那么叶帝他们这边自然是不想要沙皇这样的英雄留在场上的，只要这个英雄在场，那么让他平稳的度过中期，他们这边想要上高地可就难了。
<br/>　　而对方的ban人呢，则纯粹是为了针对叶帝和夜雪狂歌两人，虽然香蕉之前的那场训练赛确实震惊了很多人，但是还是有些战队认为叶帝还有夜雪狂歌的威胁要比香蕉大的。
<br/>　　在双方ban人结束后的一刹那，都没用叶帝提醒，香蕉这边直接秒锁鳄鱼这个英雄，要是有着鳄鱼这个英雄他都不选的话，那还真就不是他了。
<br/>　　“香蕉秒锁鳄鱼，这个英雄可是他最擅长的英雄之一，这一局boom战队没有针对香蕉ban这个人，这真的是大失误啊。”
<br/>　　“没错，香蕉现在的状态非常的完美，感觉这一场boom战队很可能要在香蕉这个点上吃亏了。”
<br/>　　解说席上小木和悠悠两个解说在那里分析着，看到香蕉选到了鳄鱼这个英雄大家都还是非常开心的。
<br/>　　几乎在确定下这个英雄的同一刻，场下的观众便是已经欢呼了起来，而直播间里的观众则是在那里频繁的刷着666的弹幕。
<br/>　　“66666666666666，真是佩服boom战队的勇气，他们脚软敢放香蕉的鳄鱼，这是要提前结束比赛么。”
<br/>　　“在国内夏季赛后半程的时候，香蕉的鳄鱼几乎都已经没有登场机会了，他的鳄鱼真的是太恐怖了，不过还是要感谢boom战队，如果不是他们，这一局恐怕还看不到香蕉玩鳄鱼呢。”
<br/>　　“这一局估计香蕉要在上路杀通关了，拿到鳄鱼白队的胜率已经在9层了。”
<br/>　　“boom战队说你们傻呢还是说你们傻呢，真以为你们是那个天下第一？”
<br/>　　看到香蕉选下了鳄鱼这个英雄，直播间里的观众纷纷叫好，当然了也有很多观众在那里嘲讽boom战队，说他们太自大了。
<br/>　　双方飞快的选人，很快阵容便是确定了下来。
<br/>　　白队这边选择了：上单鳄鱼、打野螳螂、中路妖姬、下路则是ez和娜美这两个英雄。
<br/>　　而boom战队这边选择的则是：上单皇子、打野狮子狗、中路狐狸、下路则是女警和布隆这两个英雄。
<br/>　　“这一局boom战队这边的选人很好啊，感觉白队的阵容有些被对方给克制了。”看着双方的选人，小木开口说道。
<br/>　　“没错，狐狸这个英雄克制妖姬大家都是知道的，在妖姬施放w技能位移过去的时候，一个职业水准的狐狸是完全可以利用自己的e技能魅惑来打断对方的这个位移的，然后在释放一套技能打出伤害，所以狐狸是一个counter妖姬的很好人选。”
<br/>　　“另外boom战队这边的下路布隆这一手拿的也是非常的好，布隆的盾可是能够挡住娜美还有ez的大招的，所以布隆这个点倒是真的有些麻烦。”悠悠跟着开口说道。
<br/>　　“当然了道理虽然是这么说，不过那是建立在双方的实力在55之间的，虽然这个boom战队有实力，但是想要和世界顶尖级别的荣耀白队来比，他们还是有着很大的差距的，所以有的时候阵容并不能决定一切，就像是这一局，我感觉很可能白队是想要快节奏结束比赛。”小木继续开口说着。
<br/>　　调试好符文天赋后，度过了等待时间，很快双方便是进入到了游戏中。
<br/>　　进入游戏后双方都没有选择一级团什么的，白队这边是不想，而boom战队则纯粹是不敢。
<br/>　　至于叶帝他们不打一级团，这也很简单，一级团的时候布隆这个英雄实在是太猛了，并且周围还都是草丛，这对于狮子狗这个英雄实在是太有利了，在对方有着这样两个英雄的前提下，他们就算是实力在高，恐怕在一级的时候也打不过对方。
<br/>　　毕竟一级的时候，大家根本就没什么能够秀的手段。
<br/>　　而boom战队虽然知道自家的这个阵容一级很强势，但是毕竟白队的名头摆在那里呢，他们还真怕自己这一波入侵被叶帝他们给埋伏到了，到时候赔了夫人又折兵，那可就有意思了。
<br/>　　本来他们对于这场比赛就没有太多的信心，在前期在丢掉人头的情况下，那还不如直接认输投降，这一局他们唯一的期望就是能够拖到后期，如果那样的话，他们可能还有一丝丝机会取胜。
<br/>　　1分40秒双方比赛正式开始，开局都是正常开局，打野在下路的帮助下选择了从下路的buff起手。
<br/>　　叶帝和对方的中单也是在那里和平的补着小兵，并没有做出太多的动作。
<br/>　　上路这边香蕉也是在那里安静的补着小兵，同时还利用走位轻巧的避过了皇子的一个q技能，鳄鱼真正强势的时候是在三级，到了那个时候他可就进可攻退可守了。
<br/>　　只要有红怒，那么他完全可以利用e技能上前去消耗对方一套，然后在利用自己的二段e技能返回来，对方拿他是不会有丝毫办法的，如果打的在强势一些，说不定在前期就能单杀对手。
<br/>　　而另一边皇子，在看到香蕉的鳄鱼身上红怒过半的时候，便是立刻猥琐了下去，虽然鳄鱼还没有到达三级，但是有着红怒的鳄鱼的伤害还是非常的高的，他可不敢让香蕉这样消耗下去。
