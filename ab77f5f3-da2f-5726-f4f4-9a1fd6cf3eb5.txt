　　一千二百九十七滴，一千二百九十八滴……
<br/>　　楚风眠的背后，远古战龙虚影疯狂的咆哮着，每时每刻都在汲取这灵泉中的灵力。
<br/>　　正好像是一头饥饿猛兽入了羊群一般，开始疯狂吞噬了无数灵力，转为了远古战龙精血。
<br/>　　不一会，楚风眠身上的远古战龙精血，已经是足足转化了一千五百滴。
<br/>　　但是就算是如今这种力量，还是不足以让楚风眠冲破神力境的境界，楚风眠只是感觉到，他的身躯，仿佛都是要炸开了一样。
<br/>　　好像是无法承受如此庞大的力量，但是却也无法打破瓶颈。
<br/>　　若是一直如此下去，很有可能楚风眠便是落得一个突破失败，直接陨落的下场。
<br/>　　“好在我之前已经是做好了准备，不然贸然突破的话，还真有可能失败。”
<br/>　　楚风眠察觉到了这其中危险，心中暗自感叹一声，也不托大。
<br/>　　直接便是将那天元神丹给拿了出来。
<br/>　　想要冲破境界，缓缓凝聚力量是不行的，只能是将楚风眠自己撑死，只有强大的灵力冲击，才是冲破这瓶颈。
<br/>　　那天元神丹，被楚风眠拿出，顿时楚风眠大口一张，这天元神丹便是落入到了楚风眠的口中。
<br/>　　“轰！轰！轰！”
<br/>　　继续是无数爆炸，从楚风眠的身躯之中传来，这每一次爆炸，都带有着无数灵力的冲击。
<br/>　　顿时楚风眠的身躯，都给炸裂开来，身上的皮肤，都开裂了，无数精血流露了出来。
<br/>　　天元神丹，在加上如此之多的灵力，换做给一个生死境武者，都会将他直接撑破。
<br/>　　但是如今楚风眠的身躯，已经是经过天晶淬炼，坚韧无比。
<br/>　　楚风眠心神一动，脸上露出几分坚毅之色。
<br/>　　“恢复！”
<br/>　　楚风眠长啸一声，他身上的伤痕，全部恢复过来，无数精血，也开始蒸发，全部化为了精粹的灵力，再一次的凝聚到楚风眠的身躯中去。
<br/>　　楚风眠的身躯，一次次的破碎，重组。
<br/>　　每一次的破碎重组，楚风眠的身躯，仿佛都是变的坚韧了一分，他身上的灵力，也逐渐开始转化。
<br/>　　“两千滴！”
<br/>　　楚风眠背后，那远古战龙虚影之中，已经是足足凝聚出了两千滴远古战龙精血，这简直是将楚风眠的实力，又翻了一倍。
<br/>　　但是就算如此，楚风眠也感觉到，这神力境的瓶颈，居然还未打破。
<br/>　　任由楚风眠的力量暴涨，但是这瓶颈，却是始终无法打破，就差最后的临门一脚。
<br/>　　“轰隆！轰隆！”
<br/>　　一股股灵力，从楚风眠的身上爆发出来，是将整个洞府，都给震响。
<br/>　　也幸好楚风眠之前布置了无数阵法，禁制，才能这股动静全部压制住，不然圣山震动，只怕是整个北邙学院的强者，都会被引过来。
<br/>　　“还不够！”
<br/>　　楚风眠咬牙说道，猛然之间，在他的手心中，一枚龙丹，突然出现。
<br/>　　正是楚风眠捣毁那祭坛，得到的龙丹。
<br/>　　楚风眠手心中，一股火焰点燃，这龙丹便是瞬间融化，化为精粹的龙力，被楚风眠一口吞噬进去。
<br/>　　龙丹入口。
<br/>　　楚风眠身上的力量，简直是如同火山喷发一般，积攒了许久的力量，全然都被调动起来。
<br/>　　为了突破神力境，楚风眠不知道是压制了多久的力量，这压制了多久，如今爆发出来的力量，才越是可怕。
<br/>　　楚风眠的力量，在这一瞬之间，已经是达到了巅峰。
<br/>　　天元神丹跟龙丹两股力量，同时爆发而出，楚风眠背后的远古站龙血虚影中，足足是凝聚出了两千九百滴远古战龙精血。
<br/>　　“两千九百九十八……两千九百九十九！”
<br/>　　远古战龙虚影，凝聚精血的速度，开始慢了下来，但是楚风眠却是精神紧绷到极点。
<br/>　　因为楚风眠清楚，这是他最后的机会，若是这次失败，可能楚风眠的下场，便是陨落。
<br/>　　“三千！”
<br/>　　“轰！”
<br/>　　在第三千滴远古战龙精血，凝聚而出的一刻，楚风眠身上的力量，简直是一下子便是发生了翻天覆地的变化。
<br/>　　楚风眠身上的灵力，都在熊熊燃烧这，但是这燃烧，却是并非在消耗这楚风眠的力量。
<br/>　　反而是这一股无色灵力，在燃烧之后，全部便是蜕变成为了金色灵力。
<br/>　　这是一种蜕变，楚风眠身躯的力量，在这达到临界点的一刻，全然转变。
<br/>　　“神力境！成了！”
<br/>　　楚风眠的眉头，终于是舒缓了起来。
<br/>　　在楚风眠的身边，一层层空间，突然便是变的脆弱无比，楚风眠挥手之间，都可以点破一处空间。
<br/>　　这并非是因为楚风眠的力量变强，而是因为他的空间法则，达到了一种截然不同的地步。
<br/>　　本来在北邙学院中，因为空间的强大，楚风眠已经是无法突破空间，在虚空中穿行了。
<br/>　　但是如今楚风眠踏入神力境，便是可以再一次的打破空间。
<br/>　　“神力境，果然奥妙无比，跟御风境，简直便是天壤之别。”
<br/>　　圣者之下皆蝼蚁的道理，只有现在楚风眠，才可以明白。
<br/>　　这两者间的力量，绝对是天差地别，真正能够以御风境抗衡圣者的人物，都是名震九域的天才，才能做到。
<br/>　　楚风眠感觉到，他如今的力量，至少是要比起曾经，要强大了十倍。
<br/>　　要让昨日的楚风眠出现，现在的楚风眠，几乎可以一掌镇压，毫无悬念。
<br/>　　“借助这圣山的力量，正好可以大肆吞噬灵气，不然等到离开后，便是还需要寻找机会积攒灵力，如今这最好的机会，不能浪费。”
<br/>　　楚风眠再次闭眼，立刻稳定下来。
<br/>　　再一次的催动起战龙决的力量，令远古战龙虚影，再一次凝聚。
<br/>　　如今神力境的瓶颈打破，楚风眠便是不在需要任何压抑，可以肆无忌惮的吞噬这灵力。
<br/>　　顿时远古战龙虚影，甚至都是一下子顺着阵法，潜入到了圣泉之中去。
<br/>　　开始远远不断的吞噬灵力，凝聚远古战龙精血。
<br/>　　“这圣泉中的灵力，如此庞大，我也可以从中夺取灵力来，将灵力凝聚，积攒财富！”
<br/>　　楚风眠的心神，猛然一动。
<br/>　　他的剑意催动，一股剑意，便是悄然顺这阵法，潜入到了地底灵泉之中去。
