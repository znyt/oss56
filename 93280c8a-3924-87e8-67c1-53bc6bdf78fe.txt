　　两位总殿主，看着画面中的流转，脸上惊色愈发浓郁。
<br/>　　萧逸看着画面，同样心中一阵惊骇。
<br/>　　时间回溯这一手段，他听说过，但还是第一次见。
<br/>　　而且，观面前画面，猎妖殿总殿主手掌轻动间，竟在随意操控画面的快慢、定格等等。
<br/>　　更简单来说，时间的回溯、流速、定格，皆在他一念之间。
<br/>　　要知道，即便是修为深厚的老牌传奇强者，也顶多只能做到时间回溯，随后回溯快便快，慢便慢。
<br/>　　而总殿主手中，却是或快或慢，乃至随意定格，整个时间回溯如臂指使，掌控得淋漓尽致。
<br/>　　猎妖殿总殿主，真不愧是当世传奇，果真厉害。
<br/>　　当然，萧逸心头也是一阵庆幸。
<br/>　　他向来谨慎，即便是进入八殿的闭关室，那般堪称绝对安全的地方，也每每先打下屏障禁制。
<br/>　　可现在看来，在那些真正的传奇人物面前，特别是三殿总殿主这样的当世传奇，他的那些屏障禁制，根本只是个笑话。
<br/>　　也幸好他谨慎无比，并未松懈下摘下面具。
<br/>　　否则，此时便麻烦了。
<br/>　　数分钟后。
<br/>　　猎妖殿总殿主大手一挥，散去了光幻画面。
<br/>　　这时，三位总殿主，同时看向萧逸。
<br/>　　三双苍老的眼眸中，同时迸发出炙热的目光。
<br/>　　萧逸愣了愣，心头微微有些发毛。
<br/>　　越是与这些总殿主接触得多，萧逸便越对他们的手段而感到惊骇。
<br/>　　被这样三位当世传奇凝视的感觉，怕是等闲之辈直接就会受不住。
<br/>　　不过，萧逸心头同时也一阵疑惑。
<br/>　　三位总殿主，这般神色变化是怎么回事？
<br/>　　退一万步来说，那三份奖励是他的，他是否炼制光，是否吸收光，那也是他的事。
<br/>　　三位总殿主，紧张这个做什么？
<br/>　　他真的吸收完也罢，没吸收完只是说谎或者随口说说也好，这都是他的事。
<br/>　　三位总殿主，没必要过问。
<br/>　　萧逸心头，愈发感到不妙。
<br/>　　他并不知晓三位总殿主之前在殿内聊了些什么。
<br/>　　但他，似乎已有了些许猜测。
<br/>　　这时，三位总殿主，仅是凝视着萧逸，并未说什么。
<br/>　　但他们脸上，却毫不犹豫地尽皆写满了惊色。
<br/>　　事实上，也不怪他们之前如此惊讶，甚至直接便断定萧逸说谎。
<br/>　　数个时辰前，他们才初见萧逸，然后遣萧逸去闭关室。
<br/>　　可数个时辰后，萧逸便回来了。
<br/>　　这是什么概念？
<br/>　　他们在殿内的议事都还没有结束呢，萧逸就已经结束闭关重新出现了。
<br/>　　三位总殿主聚首，自然是有不少事情要商议的。
<br/>　　原本他们打算，商议完后，便各自回总殿。
<br/>　　等数天后，或者更久，再回来猎妖殿总殿等萧逸出关也不迟。
<br/>　　可现在，才数个时辰，他们的商议都还没有结束，萧逸便已经出关了？
<br/>　　别人闭关，哪个不是动辄半月、数月，乃至更久。
<br/>　　就算是再怎么可怕的天骄妖孽，好歹也闭关个数天罢。
<br/>　　现在，直接便是数个时辰就结束？
<br/>　　也难怪三位总殿主那般惊讶，甚至直接断定萧逸说谎。
<br/>　　“额。”这时，萧逸还是率先打破了沉默。
<br/>　　“三位总殿主，不知唤易某来何事？”
<br/>　　“自然是有事的。”猎妖殿总殿主，满脸赞赏笑容。
<br/>　　“呵呵。”药尊殿总殿主，展颜而笑。
<br/>　　炎殿总殿主，和颜悦色地频频点头。
<br/>　　“小子，跟我回炎殿，数年内，我保证你能在控火一道踏入传奇。”
<br/>　　“诶。”药尊总殿主摆摆手，“先随我回药尊殿，不出三年，我保证你成为大陆最年轻的传奇炼药师。”
<br/>　　传奇控火武者？
<br/>　　传奇炼药师？
<br/>　　猎妖殿总殿主看了二人一眼，“你二人是不是忘了，这里是我猎妖殿总殿？”
<br/>　　萧逸皱了皱眉，沉声道，“三位总殿主，唤易某来，若有事，不妨直说。”
<br/>　　猎妖殿总殿主踏前一步，道，“你可听说过古帝之墓？”
<br/>　　“知晓。”萧逸点点头，“古帝之墓，乃大陆最为盛大的盛事之一。”
<br/>　　“算算时间，还有两月左右边开启了。”
<br/>　　虽然不知晓总殿主问这个做什么，但萧逸还是如实回答。
<br/>　　“嗯。”猎妖殿总殿主点点头，取出一块令牌，道，“这是我猎妖殿总殿的令牌。”
<br/>　　“你持之，便算直接拥有了我们猎妖殿的名额。”
<br/>　　“两月后，持令牌前往古帝之墓，参加盛事。”
<br/>　　“还有…”药尊殿总殿主打算插话一声。
<br/>　　猎妖殿总殿主看了眼，笑笑，道，“还有，药尊殿与炎殿的名额也给你。”
<br/>　　“你本就身兼三殿身份，如今便代表三殿总殿去参加古帝之墓的开启。”
<br/>　　“这…”萧逸皱了皱眉。
<br/>　　“这便是总殿主给我的任务？”
<br/>　　之前总殿主所说，让他先不必急着走，先去闭关一趟，之后有任务给他。
<br/>　　便是关于古帝之墓开启？
<br/>　　只是，他本来就要代表修罗殿前往，如何能再接这一任务？
<br/>　　“恕易某拒绝。”萧逸摇了摇头。
<br/>　　“对于古帝之墓，在下并没兴趣。”
<br/>　　“总殿主该知晓，易某更乐于去击杀邪修，或者去剿杀妖兽。”
<br/>　　代表三殿去参加古帝之墓盛事，那是万万不可能的。
<br/>　　至于借口，现今这个借口不错。
<br/>　　对猎妖师而言，猎妖乃是天职。
<br/>　　萧逸搬出这个借口，应该便可推辞掉这个任务。
<br/>　　猎妖殿总殿主闻言，却并不介意，反倒点头一笑。
<br/>　　“好，很好。”
<br/>　　“古帝之墓，各方天骄妖孽趋之若鹜，妄图得里头重宝传承。”
<br/>　　“各个名额，更方更是因此争执不已，乃至频生杀戮。”
<br/>　　“你倒好，压根未将此事放在心上；不为名利心动，不为浮华而动摇，心守本质，宁愿剿杀邪修妖兽。”
<br/>　　“很好，很好。”猎妖殿总殿主，一连说了两个‘很好’。
<br/>　　“这小子，心性确实不错。”炎殿总殿主，头一次认可了‘心性’之事。
<br/>　　“那古帝之墓的事，便就此作罢。”猎妖殿总殿主笑笑。
<br/>　　“反正那只是走个过场罢了，你先接令牌吧。”
<br/>　　“是。”萧逸点点头，踏前一步，伸手接过令牌。
<br/>　　总殿主如此好说话，自然是最好的。
<br/>　　虽然对于那句‘走个过场罢了’有些疑惑，但只要不是让他代表三殿去古帝之墓，那其他的事，倒也好说。
<br/>　　孰料，恰在此时，总殿主却笑意盈盈道，“数月后，等总殿这边准备好事宜，你便准备回来参加仪式。”
<br/>　　“仪式？”萧逸皱了皱眉。
<br/>　　“嗯。”总殿主点点头，“仪式过后，你就开始接触三殿总殿事务，我们三个老家伙磨砺你一番，之后就可以正式接任总殿主之位了。”
<br/>　　“接任总殿主之位？”萧逸先是一愣，随后吓了一跳。
<br/>　　原本伸向令牌的手，连忙收回。
