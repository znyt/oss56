　　轩辕剑，是传说中轩辕氏掌控的一件法器，以氏族为名，传闻当年上古之战，黄帝就是靠的这把剑斩杀的蚩尤，当然，有关轩辕剑的的一切，跟上古之战一样，都是传说。
<br/>　　不过轩辕剑的名字，的确是存在于各大门派的典籍中。
<br/>　　九段光的无上法器。
<br/>　　张无生沉吟了一下，道：“轩辕剑是轩辕山的镇山之宝之一，为何愿意拿出来送人？”
<br/>　　王道乾听见这话，说道：“想也知道，轩辕氏有把握最后选出来的人是星月奴那几个弟子之一，把轩辕剑给他，等于左手换到右手，还博了个好名声，哼哼，算盘打得不错。”
<br/>　　在悬空观一战之中，王道乾非常跳，跟他们师徒三人作对，不过那毕竟是大是大非，虽然有点心机，让人心里不爽，但在面对法术公会这个问题上，他还是站在大众的立场上，当下望着玉玑子冷哼一声，说道：“我说，法术公会的事，你跟着瞎掺和什么，给你什么好处了？”
<br/>　　“我……”玉玑子说不出话来。
<br/>　　王道乾道：“实话说，我跟道风，跟叶少阳，都不对付，但我绝对不会为了个人恩怨去出卖法术界……这里没人帮你说话，也没人拦得住那个家伙，你……自求多福吧。”
<br/>　　王道乾说完，别人也都转过头去，不理玉玑子。
<br/>　　玉玑子目光从众人脸上扫过去，突然看到了慈心师太，刚叫了一声，慈心师太就厉声说道：“玉玑子你可别乱说，我只答应你一起对付茅山，我可没跟法术公会勾结，这种事贫尼我做不出来！”
<br/>　　玉玑子怔了半晌，咬了咬牙，“好好好，你们都是正人君子，就我一个是小人，来吧，道风，想怎么样你来吧！我辈修士，何惧一战！”
<br/>　　“这话从你口中说出来，真的侮辱了这句话。”
<br/>　　道风朝他一步步走过去。
<br/>　　玉玑子脸上视死如归的模样并没有保持太久，最终还是泄气了，问道：“你刚说要死怎么样，要活又是怎么样？”
<br/>　　此言一出，身后几位宗师脸上又出现了嘲讽和好笑的表情，道风却是一点都不意外，他早就知道玉玑子贪生怕死，不然的话，在悬空观那一战之中，他也不会早早地跑掉，然后潜伏这么久了。
<br/>　　“要死，就来一战，要活，你终生不许踏入茅山一步，否则我必杀你。”
<br/>　　玉玑子心中一块石头落了地，长吁了一声，刚要说点“青山绿水”之类的狠话，道风头也不转地说道：“你要走就走，再多说一个字，听好，一个字，我立刻让你血溅五步！”
<br/>　　玉玑子当场怔住，嘴巴张了张，然后……竟然真的一个字都没有说，咬着牙，冲道风点了点头，朝山下走去。
<br/>　　他的背影很狼狈，内心一片失望，他本来是想借助各大门派的力量，逼迫叶少阳失去茅山掌教的名分，再借助法术公会给茅山一个大大的难堪，结果……没想到道风这个疯子来了，更没想到道风居然敢杀了悦然……一败涂地。
<br/>　　也许，一切只有寄望于不久之后的龙华会了，机会还是有的，毕竟道风的身份，没资格参加龙华会，而且他与法术公会结下仇，星月奴不可能放过他。
<br/>　　道风的风之谷在鬼域，不可能开来人间，仅凭他一个人，又怎么能对抗的了法术公会？
<br/>　　想到这些，玉玑子心中又有了少许的安慰，道风，走着瞧……
<br/>　　玉玑子走后，道风转头望着张无生等一干宗师，说道：“今日之事，是我一人造成，与茅山无关，诸位有什么事继续商谈，小苏，你好生招待诸位客人，必须怠慢，但若有人像玉玑子一样捣乱，立刻找我。”
<br/>　　道风说完，给了他一枚玉符，只要用法力激活，自己立刻就能感应到。
<br/>　　苏钦章手捏着玉符，开心地笑起来，“大师兄，有你在，我就什么也不怕了。”
<br/>　　突然又想起什么，问道：“大师兄，你怎么知道今天山上有事？”
<br/>　　道风没有回答，嘱咐了他几句，转身要走。张无生上前一步，说道：“道风，我师叔找你有点事，让我通知你，本来我还在想不知道上哪找你，不曾想在这遇到你，你得空最好去见一下老人家。”
<br/>　　他的师叔，自然就是道渊真人了。
<br/>　　道风微微皱眉：“找我何事？”
<br/>　　“只说是与少阳有关，具体什么事，连我也没有说。”
<br/>　　道风心中一动，点了点头，把苏钦章叫到一边，交代了几句，立刻就离开了。
<br/>　　张无生等一干宗师，重新回到大殿，坐下喝了几杯茶，也就一起走了。苏钦章恭恭敬敬地送他们离开。
<br/>　　这些宗师，很多都上了年纪，平时都各自清修，连门派的事都很少过问，这次一起下山，前来茅山会面，看似仓促，但根本上还是因为空界最近的动乱，让人间的各大门派都有了一种深深的危机感，生怕这场战争波及到人间。
<br/>　　因此，这帮长期不走动的老家伙们才聚到一起，上茅山其实是次要的，主要是见见面，讨论一下，二来也商量一下对付法术公会的办法。
<br/>　　人间法术界，没有门户之见，抵制法术公会，只是不想让轩辕氏来掌控人间，在他们看来，轩辕氏是异族，一旦让他们掌控人间，谁也不知道将来会发展成什么样子。
<br/>　　人间法术界，与普通人组成的社会之间一直保持的平衡，很有可能被打破，而各大门派都有一种担当和使命，就是维护这种平衡。
<br/>　　“星月奴几百年前就开始收弟子，有的据说已经证道，最近收的这一批弟子，就是这七个孩子，这次龙华会，法术公会必然出手，扶持他们上位！”下山的路上，大伙讨论起来，张无生说道。
<br/>　　静慧师太道：“这七人……现在应该是六人了。”
<br/>　　大伙想起之前道风斩杀悦然的那一幕，都有点心惊肉跳。张无生喃喃道：“星月奴花了十几年时间培养的弟子，被道风杀了，她必然不会善罢甘休，法术公会与风之谷之间，恐会爆发一场争斗。”
<br/>(本章完)
