        冷静到明明很心痛，却装作若无其事。
<br/>
<br/>        其实，是她让他学会了，什么叫隐忍。
<br/>
<br/>        傅夜沉回到自己的房间，站在通风口下点了根雪茄，他不可能让她长期住在这里，虽然房子里什么都不缺，但毕竟这里是地下室，永不见阳光。
<br/>
<br/>        “咳——咳——”身后突然传来两声清咳，打断了傅夜沉的思绪。
<br/>
<br/>        他侧过身去时，见童筱颖光着脚丫子站在自己的房门口，连忙将雪茄按灭在一旁的烟灰缸里，疾步走过去将她打横抱起，落坐在床边。
<br/>
<br/>        “这里的地面很凉，万一感冒了怎么办？”明明是训斥她的话，却透着难以隐藏的关心。
<br/>
<br/>        童筱颖直来直往地明说：“是你房间里的烟味呛到我了。”
<br/>
<br/>        “对不起，我抱你回房间。”
<br/>
<br/>        傅夜沉刚准备将童筱颖抱起来，童筱颖却伸手抓住了他的衣襟，弱弱地说道：“你可不可以陪我睡？我一个人睡不着。”
<br/>
<br/>        在这之前的每个夜晚，她的梦里都有这个男人的面孔。
<br/>
<br/>        如果，这个男人就陪在自己身边，她是不是就不会再做梦呢？
<br/>
<br/>        童筱颖双眼期许地凝望着傅夜沉。
<br/>
<br/>        傅夜沉微微一笑，应了声：“好。”顿了顿后，他又接着说道，“我先抱你回房，然后去洗掉这一身烟味。”
<br/>
<br/>        童筱颖能感觉到这个男人对自己的体贴，更确切地来说，是他对她的好。
<br/>
<br/>        她说要他陪着她睡，他的做法倒是十分矜持。
<br/>
<br/>        童筱颖侧身躺在床上，傅夜沉靠着床头坐在了床边。
<br/>
<br/>        他身上不再有她不喜欢的烟味，而是沐浴乳的清香。
<br/>
<br/>        童筱颖伸手抓着他的衣角，睡眼迷离地问道：“你为什么对我这么好？”
<br/>
<br/>        “因为，我爱你。”傅夜沉缓缓地回答，但此刻，身边的她却已经睡着了。
<br/>
<br/>        替她掖好被子，关了灯，他回到房间里是，看到烟灰缸里的雪茄，又将抽屉里剩下的雪茄，以及烟灰缸一起扔进了垃圾桶中。
<br/>
<br/>        半年的时间，让他染上了烟瘾，但筱颖似乎对烟味很敏感，所以，他还是决定把烟戒了。
<br/>
<br/>        此时，放在口袋里的手机振动了起来，他直接接听电话后，一句话也没说，却皱起了眉头。
<br/>
<br/>        挂了电话后，傅夜沉换了身衣服，离开了这里。
<br/>
<br/>        翌日，清晨。
<br/>
<br/>        一辆私家车渐渐偏离了原来的正轨，夏阳枭就坐在这辆私家车的驾驶座后座上，当他意识到这不是去安防局的路时，整个人的面色立马变得格外铁青。
<br/>
<br/>        他紧张地从衣摆的腰袢上掏出手枪，直接抵在正在开车的男司机的后脑勺上，命令道：“停车！”
<br/>
<br/>        “干爹怕什么？”傅夜沉摘掉了脸上的墨镜，一边淡定地开车，一边若无其事地问。
<br/>
<br/>        夏阳枭有些恼火：“我叫你停车！”
<br/>
<br/>        傅夜沉勾唇一笑，踩了个急刹车。
<br/>
<br/>        “嘭”地一声，夏阳枭一头撞到了驾驶座的椅背上。
<br/>
<br/>        傅夜沉随即丢给了他一台手机，里面正自动播放着一段视频。
<br/>
<br/>        视频里，一个黑衣人手持着匕首，正抵在他夏阳枭的父亲的脖子上。
<br/>
<br/>        夏阳枭的父亲年迈已高，坐在轮椅上也是半死不活。
<br/>
<br/>        面对此情景，夏阳枭十分淡定地冷哼：“你傅夜沉现在的行为，可真对不起你以前所穿过的那身军装。”
<br/>
<br/>        “我这行为确实对不起，但是干爹你昨晚因为抓不到筱颖而派人绑架我母亲，威胁我把筱颖交出来，是否又对得起你这局长的身份？”傅夜沉慢条斯理地反问。
<br/>
<br/>        夏阳枭凝了凝眉：“我听不懂你在说什么。”
<br/>
<br/>        “干爹何必再装下去？这里除了我两就没有其他人。这半年的时间里，冒险者集团几乎快被我夷为平地了，干爹心里一定不好受吧！”傅夜沉话中有话地阐述。
<br/>
<br/>        “小沉，无凭无据的话，最好不要乱说。”夏阳枭嗤笑，抵在傅夜沉脑后的手枪，毫不懈怠。
<br/>
<br/>        傅夜沉面不改色地笑了笑：“干爹的本事确实很大，把自己藏得如此深，一面当着两道通吃的老大，一面又教我什么是正道，一副正义凛然的样子。”
<br/>
<br/>        “小沉，如果你爷爷知道，你绑架了我父亲……”
<br/>
<br/>        “如果我爷爷知道，您绑架了我母亲，您觉得我爷爷会怎么做？”傅夜沉直接打断了夏阳枭的话。
<br/>
<br/>        夏阳枭依旧坚持：“我没有绑架你母亲！”
<br/>
<br/>        “干爹是敢做不敢当吗？”傅夜沉哑声质问。
<br/>
<br/>        夏阳枭不得不转开了话题，坦白道：“筱颖是我女儿。”
<br/>
<br/>        “呵——”傅夜沉嗤之以鼻。
<br/>
<br/>        “你明知道我在找女儿，而且你也早就知道筱颖是我女儿，但你却一直将她藏起来，跟我派出的人绕弯子，就是不让我的人找到筱颖，阻止我和筱颖父女相认。小沉，你这是居心叵测。”夏阳枭斥责道。
