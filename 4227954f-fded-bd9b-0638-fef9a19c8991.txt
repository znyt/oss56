　　第690章
<br/>　　帝国十三州一百单八城，每天都有许多事情发生。每个人也都有每个人的困境，毕竟是乱世，就连帝国天子都逃不过桎梏，何况是普通人。
<br/>　　但总得来说，程大雷的日子还过得去，总之是一切向好。
<br/>　　这一夜他安稳睡去，第二天便有丫鬟上门。苏樱从将军府挑选的女人今天被送进来，对于这种事，程大雷不好显得太过关心。但进进出出，多了许多女人，端茶递水的丫鬟越来越标致，程大雷总能察觉到一些变化。
<br/>　　他是悄悄问过柳芷才知道，苏樱这次还算蛮对得起他，挑选了五十余人，都是那群女俘中最水灵的一批，人人知书达理，皆是大户人家出身。另外，还有一支十几人组成的乐班，歌舞俱佳，人人身怀绝艺，类似茶道、乐器、诗词歌赋。
<br/>　　程大雷不得不感慨，如今自己也算个体面人了，家里都养得起班子。
<br/>　　当然，仅仅这五十余人还是不够一个城主府的配置。但程大雷贴身的没有几个人，除了他自己之外，满打满算也就是苏樱、樊梨花、柳芷，还有个银眸以及黑牛，也不需要多少下人伺候。
<br/>　　在这批女眷中，最显眼的便是张白玉主仆四人，她们身在凉州，心可未必在此地，目前尚未归顺，大概属于戴罪之身。所以四人都是穿着素白长袍，也没有涂脂抹粉的资格。
<br/>　　程大雷窝在屋子里看这四人劈柴挑水洗衣烧火，却也是一道靓丽的风景线。
<br/>　　除了程大雷挑选的这批外，将军府剩下的其他人，也被程大雷手下一帮文武给瓜分了。这些事，程大雷心里明白，但并没有细问。
<br/>　　无论如何说，都不是太过体面的事。
<br/>　　程大雷可以将这些事不放在心上，拍拍肩头，像肩上的灰尘一般随手拍去。但对崔白玉而言，她绝不可能轻易放过这件事。
<br/>　　每一天，崔白玉都生活在水深火热中，将她当一个杂役使唤，对她而言确实不算什么。她担忧的是，那噩梦一样的事会发生。
<br/>　　幸运的是，程大雷至今还未拿她侍寝。不幸的是，她知道这一天迟早会发生。这对她简直是一种煎熬，噩梦终会到来，只是不知何时会到来。
<br/>　　死的确会一了百了，痛苦的是，她求死的勇气早已被耗尽。明知生不如死，却没勇气再提起刀。
<br/>　　到了这一夜，崔白玉回到房中，摊开双手，本来十指不沾阳春水的纤纤玉手如今已磨出茧子。这更令她悲从中来，满腔委屈却无处诉说。至于小鹿三女，也未必好到哪里去。
<br/>　　老实说，她们的住宿条件还是不错的，毕竟城主府的房子是多的，如今也住不满，程大雷没有必要给她们特意找个柴房。而在饮食上也没有亏待她们，自然和从前没法比。
<br/>　　房门突然被推开，小蝶立在门口打量了四人一眼，满目鄙夷。
<br/>　　“喂，出来，大当家要见你们。”
<br/>　　崔白玉心中一激灵，该来的终于来了。她下意识问：“她，她要做什么？”
<br/>　　“去了你就知道了，问那么多干什么。”
<br/>　　小蝶的态度绝对说不上好，对这样的貌美女子她很难能够心平气和。冷目扫过，见四人依旧不动弹，她催促道：“怎么，还要我用轿子抬你们走么？”
<br/>　　四人面面相觑，都没有主意。最后无可奈何的起身，刀山火海终究是要闯的，她们随小蝶离开屋子。
<br/>　　程大雷如今的办公地点在城主府的书房，这原先是宋伯康的地方，装潢摆设自然不是蛤蟆城能比的。首先宽敞得紧，宋伯康平素歇的不是炕，而是一张檀木床。
<br/>　　崔白玉到的时候，见程大雷正与刘悲议事。看到崔白玉进来，程大雷立刻闭上嘴巴，道：“这事情明天再谈，你先回去。”
<br/>　　刘悲答应一声，离去时多看了崔白玉一眼。崔白玉自然知道他是谁，罕见的，他没有露出那副男人都会有的猪哥像，眉目中甚至有几分厌烦。
<br/>　　这便已是不容易做到的事情，崔白玉暗暗吃惊，这程大雷手中还真是藏龙卧虎。
<br/>　　程大雷的目光已落到她身上，程大雷没有刘悲那样的涵养，见这幅美貌不能不动心。
<br/>　　崔白玉的心情一直在打鼓，若程大雷真打算要了自己的身子，自己该如何办？如今自己已经是这恶贼嘴边的肉，自己又能怎么办？
<br/>　　小鹿三人同样有这方面的担忧。
<br/>　　程大雷咳嗽一声，吓得四人心中一颤。程大雷目光也收了回来，差点把正事忘了。
<br/>　　啪，一份材料拍在桌上，程大雷冲崔白玉招招手：“你过来看看，对于这三人你了解多少？”
<br/>　　崔白玉心惊胆战，好在程大雷不是让自己脱衣服，她战战兢兢的过去，拿起那份材料时一怔。
<br/>　　是关于武威、浩甲、金城三座城池的情报，崔白玉只是翻看第一页，就立刻明白程大雷在做什么。
<br/>　　他竟是在做自己当初做过无数次的事情，透过这些情报，分析自己的敌人。达到知己知彼，百战百胜的目的。
<br/>　　浩甲钱通玉，武威白空集，金城孙别山，崔白玉当然了解这三人。也就是在一瞬间，崔白玉明白了程大雷目前的困境。
<br/>　　程大雷摊子铺的太大了，他其实并没有吞下凉州城的实力，却阴差阳错成了凉州城的主人。目前兵力不足，内部空虚，只要一场小小的战争，就可以将他外面的繁华打破。
<br/>　　崔白玉的双手颤抖起来，极力压抑着情绪。这些事自己早该想到的，可是也是身在局中，只顾着自身安危倒是疏忽了自身之外的事。
<br/>　　自己并不是没有希望，只要传一道消息出去，告诉武威三座城的任意一人……
<br/>　　她不敢再想下去，免得被程大雷发现。
<br/>　　而此刻，程大雷目光已落在小鹿三女身上。却也是环肥燕瘦，各有千秋，只是平时跟在崔白玉身边，显不出她们。
<br/>　　都是十七八岁的年纪，青春郁葱，程大雷悄悄开启首领光环，多少能刷一波好感值。
<br/>　　“如何，在这里还住得惯么？”
<br/>(本章完)
