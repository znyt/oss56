        傅氏集团大厦，总裁办。
<br/>
<br/>        傅昼景刚走不久，季思妍便登门拜访，傅夜沉还以为她是为了哥哥季运恒而来，谁知她一开口的第一句话，便是询问他的弟弟傅昼景的事情。
<br/>
<br/>        “夜沉哥，我联系不上阿景，你可不可以帮我联系他？”季思妍手指交叠，声音暗哑，双眸里布满了血丝。
<br/>
<br/>        自从，她利用自己的关系，栽赃嫁祸“童筱颖”，把“童筱颖”送入看守所后，傅昼景就再也没联系过她了。
<br/>
<br/>        她有去地利置业大厦找他，甚至他的私人别墅，她也去过，就连他的助理赵斌，也不知道他这段时间的去向。
<br/>
<br/>        傅昼景就像失踪了一般，从她季思妍的世界消失了。
<br/>
<br/>        傅夜沉不禁感到意外，狐疑地看着季思妍，问道：“你找阿景做什么？”
<br/>
<br/>        “我、我……”季思妍支吾着欲言又止，两眼泪水充盈，顿了顿后，忍不住声泪俱下，“我怀了阿景的孩子！”
<br/>
<br/>        “你……”傅夜沉瞬间瞪大了眼睛，哑然失色。
<br/>
<br/>        季思妍挪步上前，低头从自己的手提包里拿出了一张B超早孕检查报告，递给了傅夜沉。
<br/>
<br/>        傅夜沉剑眉微蹙，深邃的黑眸微微眯了起来，眼神里透着一股犀利的寒气。
<br/>
<br/>        季思妍抬眸对上傅夜沉的目光，被他这冷酷的气场给怔住后，莫名其妙地有点儿心虚起来。
<br/>
<br/>        她拿着B超早孕检查报告单的手，在微微颤抖。
<br/>
<br/>        傅夜沉沉默了片刻后，才开口说道：“季大小姐，跑到我面前来撒谎，可不大适合你的身份。你肚子里是否有我们傅家的血脉，我会带你去医院里重新核实！”
<br/>
<br/>        季思妍瞄了傅夜沉一眼后，心虚地聋拉着脑袋，并将手里的B超早孕检查报告单胡乱地塞回自己的手提包。
<br/>
<br/>        “若无其他事，季大小姐请回！”傅夜沉见季思妍一副做贼心虚的样子，当即便下逐客令。
<br/>
<br/>        季思妍顿时急红了眼，鼻子一酸，再次潸然泪下地哭诉：“夜沉哥，我撒谎是我不对！但是，我真的很想找到阿景！我想求他不要离开我！我是真的很爱他！夜沉哥！我求求你，好不好？求求你，告诉我阿景的下落！”
<br/>
<br/>        “阿景已经结婚了，这事你不知道吗？”傅夜沉凝眉反问。
<br/>
<br/>        他现在不得不怀疑季思妍和傅昼景之间的关系了。
<br/>
<br/>        而事情到了这个地步，季思妍开始大言不惭地说道：“我知道阿景已经结婚了，但是，我还是他的初恋情人！”
<br/>
<br/>        “你这话是什么意思？”傅夜沉眉头深锁。
<br/>
<br/>        季思妍哭丧着脸，哽咽着回答道：“阿景跟我，他……婚内出轨……”
<br/>
<br/>        “你！”傅夜沉顿时额冒青筋，恼火得一句话也说不上来。
<br/>
<br/>        季思妍此时此刻，打算破罐子破摔。
<br/>
<br/>        她为傅昼景付出了身体，也付出了金钱。
<br/>
<br/>        结果……
<br/>
<br/>        她落得个人财两空，她又岂能就此善罢甘休？
<br/>
<br/>        而此时此刻，傅夜沉只觉自己一阵心慌意乱。
<br/>
<br/>        又或许，这只是她季思妍的片面之词，但是，她季思妍这样诋毁自己，有损自己清誉，是为了什么？
<br/>
<br/>        倘若季思妍说的是真的……
<br/>
<br/>        看样子，筱颖并不知道阿景婚内出轨的事情。
<br/>
<br/>        “阿景他出国了……”良久后，傅夜沉才淡淡地开口，“他们去了澳洲。”
<br/>
<br/>        阿景出国了，带走了本应该是属于他的筱颖……
<br/>
<br/>        “澳洲？！”季思妍顿时眼前一亮。
<br/>
<br/>        只要有了这条消息，她就不怕找不到阿景了。
<br/>
<br/>        季思妍忽然喜上眉梢，破涕为笑，向傅夜沉欠了欠身，手里甩着包包，转身离去。
<br/>
<br/>        傅夜沉随之起身，走到了落地窗前，仰望着蔚蓝的天空。
<br/>
<br/>        他英俊的脸庞从玻璃上映现出来的样子，有几分隐忍与矜持，更多的是令人捉摸不透。
<br/>
<br/>        总裁办的门外，刚刚季思妍前脚刚进门，连华生后脚就到了。
<br/>
<br/>        只是，季思妍一直在追问傅夜沉有关于傅昼景的事情，他便只好候在门外，等他俩把事情说完，他再进来。
<br/>
<br/>        所以说，刚刚季思妍的那番话，他连华生是一字不漏地听得一清二楚。
<br/>
<br/>        “阿沉，你为什么不直接帮那位季大小姐找到二少爷，然后干脆将计就计，默认那位季大小姐肚子里怀有你弟弟的孩子的谎言。与她同仇敌忾，让你弟弟回到那位季大小姐身边。而童筱颖，自然而然地就归你了！”连华生一边走了进来，一边笑吟吟地说道。

<br/>        傅夜沉微微侧过头，俊颜神色淡漠了几许。
<br/>
<br/>        他看着眉飞色舞的连华生，淡淡地问道：“你什么时候到的？”
