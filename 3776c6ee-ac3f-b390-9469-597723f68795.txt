　　北邙宫。
<br/>　　在北邙学院之中，真正知晓这北邙宫存在的弟子都寥寥无几。
<br/>　　北邙学院之中，唯有实力已经是超越了生死，踏入了生死台阶的武者，才有资格踏足之地。
<br/>　　在这北邙宫之中出现的一道道身影，都是在北邙学院之中地位极高的老古董，他们才是真正北邙学院的高层。
<br/>　　北邙学院，传承万年时间，虽然真正的底蕴无法是跟七大宗门相比，却也是已经超越了大部分的宗门去，在这北邙宫中的身影，足足是有着上百位之多。
<br/>　　在整个北邙学院中，居然足足是有着超越百位超越生死的人物。
<br/>　　其中坐在北邙宫最上方的几位老者，每一尊的气息，更是恐怖到了极点，这些老古董，都是已经不知道活了多少岁月的人物。
<br/>　　端坐在虚空之中，一动不动，但是单单身上散发出来的威势，都足以令人震撼。
<br/>　　而在下方的许多身影，其中的几人，便是都是楚风眠熟悉的存在。
<br/>　　曾经的那夜家的夜牧，也是赫然坐在下方，以及那北邙学院的副院长，夜家的夜众星，也都坐在下方，不过此人的地位极高。
<br/>　　在他的身边，其他的那些身影，都是不由的低下了头，表示尊敬。
<br/>　　“楚风眠从归墟天中活这出来了，根据情报，此人在归墟天中，大肆杀戮，得到了无数宝藏！甚至是那七万年前的庞然大物，云羽天宫的传承，都被此人得到！我们北邙学院的弟子中，许久未出现过如此人物了，尤其是那云羽天宫的传承，更是一座巨大无比的宝藏，我们北邙学院想要崛起，这云羽天宫的传承我们必须要得到！”
<br/>　　其中一道身影，冰冷的开口道。
<br/>　　这一道身影，正是北邙学院的一尊老祖之一斗荒老祖，算是在北邙圣域之中，经常出现的老祖，掌控现在北邙学院的种种情报。
<br/>　　楚风眠从归墟天中发生的种种消息，可以瞒过其他人，却是很难瞒过像是北邙学院这样的庞然大物，毕竟任何一方强大势力，其中的情报网都是比起想象中的还要恐怖。
<br/>　　不止是北邙学院，现在关于楚风眠从归墟天中得到云羽天宫传承的消息，七大宗门，也一样知道，不止是七大宗门，像是玄天宗等宗门，如今相比也都得到了情报。
<br/>　　这斗荒老祖的话音刚落，又一尊身影，站了起来，言语尖锐的开口道。
<br/>　　“根据消息，此子在归墟天中，大肆杀戮，七大宗门的弟子，几乎是都被此人全部斩杀，就连玄天宗，炎阳门的弟子，都被此子斩杀！此人简直是疯狂到了极点，给我们北邙学院带来的极大的麻烦，不如将此子擒下，将其交给七大宗门！”
<br/>　　这尊身影，便是夜牧，之前他没有杀了楚风眠，一直怀恨在心，如今看到楚风眠归来，他便是第一个想要杀了楚风眠。
<br/>　　不过楚风眠从归墟天之中的所作所为，已经是传到了学院高层哪里，就算是身为夜家弟子的他，也不敢擅自出手轰杀楚风眠。
<br/>　　只能够是等到学院做出的决定，再来对付楚风眠。
<br/>　　“夜牧的话不错，我们北邙学院，一向低调，不去招惹七大宗门，但是此子实在是太狂妄了，大肆屠戮七大宗门的弟子，惹了一堆的麻烦，将来七大宗门怪罪下来，对我们学院也不利，我同意将此子交给七大宗门。”
<br/>　　又是一尊身影开口道，是一尊名为天南的老祖，他一开口，便是对那夜牧的支持。
<br/>　　“若是将楚风眠交了出去，那他所得到的那么多的机遇，还有云羽天宫的传承怎么办？”
<br/>　　又一尊端坐在上方的身影，身为北邙学院的副院长之一的邓灵老祖开口道。
<br/>　　“这楚风眠，在归墟天中，就连那剑祖荒，鹿战神，荒圣君三人都被此子斩杀，其中那荒圣君，据说已经是超越了生死，达到了碎命之境，实力已经是不亚于一般的老祖一般的人物。”
<br/>　　“这样的天才，我们北邙学院千万年来，都没有碰到过，如果现在白白交出去，不是让我们北邙学院错过了一个崛起的好机会吗？更何况此子还是玄北的弟子，对我们北邙学院也算忠心，倒不如我们先赔偿一些财富给七大宗门，保下此子如何？”
<br/>　　这北邙学院的副院长，邓灵老祖，却是说出了不同的意见。
<br/>　　“邓灵副院长，此子是个祸害，心性狂傲至极，留在北邙学院中，只能够是给我们带来灾祸，哪里可以培养！”
<br/>　　夜牧老祖尖锐的声音再度开口道。
<br/>　　“更何况，此子不过是运气好，多得到了些机缘罢了，我们只要是将此子擒下，先将他的机缘全部逼问出来，在抽出他的血脉，足以再度缔造出一个天才来，倒时候在将此子交给七大宗门去，也足以平息七大宗门的怒火。”
<br/>　　“不错，这楚风眠虽然是玄北弟子，却是突然之间冒出来的，之前我们都没有听说过玄北还有这样一个弟子，更何况此人的实力实在是太过诡异了，进展太快，区区数个月的时间，就从御风境突破到了生死境的地步，这种速度简直是闻所未闻，我怀疑此子，是七大宗门派来的奸细！”
<br/>　　坐在宫殿上方，一直没有开口的那夜家的副院长，夜众星突然开口道。
<br/>　　“七大宗门的人，一直想要吞并我们北邙学院，这事诸位的心中应该都知道吧，不过一直以来七大宗门的人都没有找到机会，这一次，这楚风眠很有可能，便是七大宗门派来的奸细，他的实力，应该也是以一种特殊的手段压制下来的，不然的话不可能进展如此之快！”
<br/>　　“无论如何，此子不能不查！就算是我们学院要决定培养此人，也必须要此人知根知底才行，让他将一切都坦白出来，在做定论！”
<br/>　　夜牧老祖急忙开口道。
<br/>　　“就由我将他擒下，带到这北邙宫中来如何？在众目睽睽之下，此人到底是什么来历，一眼便之！”
<br/>(本章完)
