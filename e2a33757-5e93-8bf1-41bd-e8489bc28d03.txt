　　接下来叶帝这边又打了几局排位赛，好在是再也没有遇到香蕉他们，要不然的话估计叶帝这心里就真的要崩溃了。
<br/>　　几局下来叶帝这边大杀四方，心情一下子好了起来：“我就问问，还有谁！”
<br/>　　这个时候他倒是忘了自己刚刚被香蕉他们制裁的那一局的排位赛了。
<br/>　　等到时间差不多到达5点钟的时候，听到叶颖那边已经催促着他们吃晚饭，叶帝这边便是没有在继续下去。
<br/>　　“今天的直播就到这里了，希望大家看的开心，感谢大家一直以来的支持，我们下次见。”说着叶帝便是关掉了自己的直播。
<br/>　　而直播间的观众虽然有些不舍，不过还是刷起了88的弹幕。
<br/>　　“北北他们的比赛怎么样？”来到餐厅后，叶帝开口问道。
<br/>　　“发挥神勇，直接将EG战队给斩落马下了。”秦霄竖着大拇指说道。
<br/>　　“可以啊，这群小子还是很有潜力的嘛！”听着秦霄的话，叶帝这边也很是开心。
<br/>　　接下来几天大家依旧是如同往常一样正常的训练，至于那天休假的时间，大家倒是都没有早早的使用出来，看样子都想拖到后面用。
<br/>　　一转眼的时间便是来到了周六，这一天是B组的荣耀蓝队和雷霆之战，这也是被所有人公认的B组半决赛之战，谁赢谁将进入半决赛。
<br/>　　叶帝一行人这一天也都是跟着来到了现场观战，虽然蓝队很强，但是雷霆战队同样也是一个实力非常强劲的对手，叶帝几人也想从这一场比赛中观察一下双方的战术。
<br/>　　时间才刚刚到达12点钟，现场便是已经坐满了两队的粉丝，仔细一看的话，雷霆战队的粉丝倒是也占据了将近半座之数。
<br/>　　“不愧是老牌战队，雷霆这边粉丝还是非常多的。”柳清月有些感叹的说着。
<br/>　　“是啊，之前在华夏电竞没有崛起的时候可都是雷霆战队在这里撑着，所以他们有这么多粉丝倒是也不奇怪，这一段时间雷霆战队的实力进步很快，不知道蓝队会怎么应对。”叶帝跟着开口说道。
<br/>　　当镜头给到叶帝几人的时候，现场轰然爆发出一阵巨大的欢呼声，不论蓝队粉丝还是雷霆战队的粉丝，都在那里热情的欢呼着。
<br/>　　“哥这个人气真的是太大了，感觉只要是玩英雄联盟的都是哥的粉丝。”萝卜一脸艳羡的说着。
<br/>　　他自然是不会以为这些欢呼声是给自己的，如果说蓝队的粉丝会为自己欢呼还可能，可是那还包括着雷霆战队的粉丝啊。
<br/>　　能让这些战队粉丝同时欢呼的，估计当下英雄联盟职业战队中，也就只有叶帝有这个待遇了，他用他的个人魅力征服了所有人。
<br/>　　“人家那是看清月才欢呼的，你们不要想多了，现在清月的人气可是很大的。”叶帝笑着说道。
<br/>　　听着他的话，一旁的茄子几人也都是跟着调侃了起来，当然了大家心里对于叶帝的话却是不信的。
<br/>　　虽然如今柳清月有着火神一姐的称号，但是和叶帝相比，她的人气还是要插上很多的。
<br/>　　“今天的比赛你们怎么看？”夜雪狂歌开口问道。
<br/>　　“蓝队应该能够3：0吧，虽然小伟他们在进步，不过蒋念他们的进步同样也很大，估计还是之前那个局面。”柳清月开口说道。
<br/>　　“我也是这么想的，之前蓝队就3：0击败过一次雷霆，感觉这一次还会是那样，有着秦哥在场上指挥，想要让他们失误，实在是太难了。”香蕉跟着开口说道。
<br/>　　“就算是3：0，估计这几局比赛打的也是会相当的精彩，雷霆战队这边肯定会使出浑身解数的。”叶帝笑着说道。
<br/>　　当时间到达一点钟的一刹那，这一场世界都在关注的比赛终于是正式开始。
<br/>　　双方轮流ban选，很快各自的阵容便是确定了下来。
<br/>　　雷霆战队这边选择了：上单武器、打野挖掘机、中单冰女、下路则是女警和扇子妈。
<br/>　　蓝队则是选择了：上单诺克、打野皇子、中单沙漠皇帝、下路则是大嘴和牛头这对组合。
<br/>　　“这一局有意思了，上路居然选出了武器和诺克这两个英雄，看样子现在在这些顶尖战队中，他们真的是有意让上单回归到这种暴力流派中，而让打野承担一个坦克的位置。”看着双方的选人，米粒大声的开口说着。
<br/>　　“没错，估计这一局小刀的皇子是要纯肉出装了，当然了武器和诺克这两个英雄也都是半肉型的，所以前排双方都是不缺的，不过我比较期待的还是蓝队这边的下路组合，大嘴和牛头，这个阵容可是好久没见了。”林雅楠跟着开口说道。
<br/>　　“牛头这个英雄自从大招被削弱后，一级大招已经不能够顶住防御塔的伤害了，加上在前期就是被动挨打的一个辅助，所以在那次版本更新之后，使用牛头的辅助倒是越来越少了，今天霄神拿出这个英雄倒是真的让人有些期待。”米粒继续说着。
<br/>　　场下看着双方的选人，叶帝他们这边也都是在你来议论着。
<br/>　　“蓝队这个阵容，到了后期真要是被飞刀的皇子多大中几个人，那老林的大嘴可就真的要爆炸输出了。”
<br/>　　“这个阵容打到后期，雷霆战队想要赢还是很难的，不过这一局我更期待的还是上路的对决，诺克这个英雄可是小南非常擅长的一个上单的。”
<br/>　　“这才刚刚开始，蓝队这边便是放出了半个大招，他们真的是打算火力全开了啊。”
<br/>　　在众人的议论声中，双方的比赛已经是正式开始，这一局比赛在前期双方打的都非常的谨慎，并没有爆发出什么人头来，就算是下路的林萧和秦霄两人，也都是在那里发挥的游刃有余。
<br/>　　“只要大家稳住发育，其实ad根本就没有前期后期之分，大家谁都杀不了谁，所以以后的下路还是选那种后期能力强一点的好。”看着下路的对决，叶帝开口说道。
<br/>　　事实上也正是这样，各个战队的打野基本上都没有照顾下路的习惯，而下路双方实力在差不多的情况下，想要在线上单吃对手，基本上是不可能的。
<br/>　　所以只要这么发育下去，基本上打的都是后期。
