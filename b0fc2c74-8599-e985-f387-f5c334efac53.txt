　　叶少阳和翠云则要继续往江西的东北方前进，去往鹰潭地区。
<br/>　　在这破房子里，叶少阳二人打开行李，检查了一下，带来的干粮大部分都被水泡了，不能再吃，剩下的肉干之类的，最多也只够两个人吃三天。
<br/>　　原本按照计划，他们一天就该进入县城，采购一番之后，雇马车前往龙虎山，但是现在……没钱了，连吃饭都是问题，去鹰潭还有几百里，如果靠双脚走过去……就算叶少阳不怕浪费时间，也会饿死在半路上。
<br/>　　因此，眼下最要紧的事是想办法弄钱。
<br/>　　翠云一个乡下妇女，到了外地，自然是一点主意都没有了，只顾望着叶少阳发呆。
<br/>　　叶少阳更是没什么主意，毕竟自己是穿越过来的，在这个世界缺乏生活技能，唯一的特长，也就是法术了。
<br/>　　“我们去这里。”叶少阳指着地图上一个用毛笔圈起来的地方说道。翠云不识字，但图还是可以看的，凑上去看了一眼，皱眉说道：“这不是万大伯说的义庄的位置吗，你要去那干啥啊？”
<br/>　　“赶尸。”
<br/>　　叶少阳坚定不移地吐出这两个字，听在翠云的耳朵里，当场就懵了，怔怔地看着叶少阳，一句话说不出。
<br/>　　“赶尸赚钱。”叶少阳补充了一句，耸了耸肩道，“别的我什么都不会，我就只会这个了。”
<br/>　　翠云这才明白他是认真的，急忙说道：“赶尸啊，你以前赶过？”
<br/>　　“没赶过。但是应该没问题。”
<br/>　　叶少阳挠了挠头说道。赶尸术，是苗疆巫术的一种，与道门法术略有关系，但并不算是真正的道术，对叶少阳这种茅山派出身的来说，更是旁门左道，叶少阳也不懂其中的诀窍，但他所掌握的三百八十六道符篆之中，至少有五六种是能够控尸的，就算现在法力不济，用灵符吸收天真地秀，驱赶几具尸体还是不在话下。
<br/>　　翠云不知道这些底细，听叶少阳说的这么勉强，登时担心起来，急忙说道：“我知道你是有些法术的，但你又没赶过尸体，那些尸体都会动的，想想都吓人啊，万一出事了咋办，不成不成，对了你力气大，不如去附近县城卖力气做点啥子赚钱……”
<br/>　　“出苦力肯定不行了。”叶少阳说道，“现在流民这么多，最不缺的就是苦力，就算找到活干，给的钱太少，得干多久才能把盘缠赚出来？赶尸的话，虽然赚钱也不多，估计走一趟也就够了。”
<br/>　　翠云各种不想让叶少阳去，但是叶少阳保证绝对不会出事，翠云拗不过他，只好听他的了。
<br/>　　叶少阳看了下地图上的记号，从这里到公堂义庄有差不多一百里路，走起来的话，估计也就是一两天，两人身上的干粮也足够撑到那里。
<br/>　　叶少阳不再耽搁，招呼翠云这就赶路。一路上，翠云一直在为被抢走的三根小黄鱼耿耿于怀，情绪很低落，总觉得是自己的责任，把叶少阳的老婆本给丢了，对此叶少阳也只能各种安慰她，心里对于翠云的善良，又是增添了几分感动。
<br/>　　对于被抢劫的经过，叶少阳偶尔也会记起，想起那个叫“小三”的家伙，身上不知道佩戴了什么非同寻常的法器，百鬼不侵。
<br/>　　一般普通人身上，是绝对没有这种强度的法器的。
<br/>　　不过相比这件事，更加让叶少阳念念不忘的，是那个小三的所作所为：三言两语就劝了自己的长官和同伴们去当土匪，面对逃走的同伴，可以毫不犹豫地开枪杀人。
<br/>　　这是一个城府极深而又心狠手辣的人，如果他将来不死，叶少阳相信，他绝对会成为一个了不起的人物，甚至成长为一个枭雄。
<br/>　　叶少阳很好奇他大名叫什么，更好奇他当上土匪之后的人生经历。不过这个跟自己没有任何瓜葛了。
<br/>　　因为刚下了一场雨，泥土湿滑，走的很慢，两人一直走到天黑，循着灯火，走进了一个位于深山中的小村子，翠云前去借宿，开门的是个老大娘，看他们只有两个人，还是一男一女，很爽快地让他们进去了。
<br/>　　后来叶少阳才知道，这一带虽然是深山，却是这一代山脉走向的一个节点，经常会有徒步进山的人会从这个小村里路过，在这里借宿，当地人也算是习惯了。
<br/>　　这家人是一对老夫妻，叶少阳和翠云到的时候，两人正在生火做饭，翠云很自然地上前去帮忙，跟大娘套着近乎，很快就熟络起来，之后四个人一起吃饭，自从进山之后，叶少阳二人就没吃过柴火灶煮出来的饭，此刻喝着热腾腾的米粥，真是一种莫大的享受，比平时吃过的什么东西都香。
<br/>　　叶少阳满意地咽下一口粥，看了翠云一眼，发现翠云也在看自己，两个人都明白对方的想法，相视一笑。
<br/>　　一边吃饭，翠云一边跟老夫妻聊天，得知这家人姓刘，有三个儿子，一个去当兵了，一个去了广州上学，只有一个在山脚下不远的县城做小买卖，偶尔会上山看望他们。
<br/>　　被问起自己的说，翠云谎称自己跟叶少阳是姐弟俩，来江西是投亲的，因为遇到了土匪，不得已只要步行到这边来。老两口一听，更是觉得他们可怜。
<br/>　　叶少阳向他们打听起附近的情况，老两口表示，这个村子叫做思故村，之所以有这么个奇怪的名字，还要从村子的祖先说起，这个村子的祖先，据说是一位明朝时候的将军，因为避祸，才带着一干亲族和部下躲到这里来，开辟山石，自己种地，自给自足，过着好像桃花源一样的生活，后来大明朝亡了，将军的后人们也都习惯了山里的环境，一直繁衍下来。
<br/>　　因为有客人到来，再加上翠云嘴甜，哄的老两口很是开心，刘老头兴奋地拿出了自己泡的五毒酒，要让叶少阳尝尝。
<br/>　　叶少阳一听名字就知道不是什么好东西，果然，在刘老头打开酒缸之后，叶少阳看了一眼，在里头发现了蝎子蜈蚣之类的毒物，心中一阵发怵，不过又不好薄了刘老头的面子，只好硬着头皮接住他递来的酒杯。
