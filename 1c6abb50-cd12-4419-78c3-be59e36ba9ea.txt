　　因为他知道，
<br/>　　修真界本就是弱肉强食，尤其是在底层，这种事情几乎随处可见，他根本管不过来。
<br/>　　而且，就算叶尘，将这爷孙俩，救下来了又如何？
<br/>　　他能帮他们一时，却不能帮他们一世，在这个残酷的修真界，弱小本身就是一种原罪！
<br/>　　更何况，迷罗星海马上就要开启，他还有更重要的事情要做，不能在这个上面浪费时间。
<br/>　　“爷爷！！”
<br/>　　一旁的那名瘦弱少年，看到爷爷的惨状，顿时大吼了一声，继而两眼竟然变成了赤红色，连头发也隐隐泛红！
<br/>　　继而，少年猛然站起身来，竟然将押着他的那名恶狼帮弟子，直接弹飞开来！
<br/>　　“嗯？”
<br/>　　原本都已经准备默默走开的叶尘，看到这突然的一幕，不由得定在了当场，眼眸之中露出一抹惊愕之色，
<br/>　　“狂魔之体？难道这小孩，竟是狂魔一族的后裔？”
<br/>　　狂魔一族，曾经是修真界赫赫有名的一个种族！
<br/>　　据说，凡是拥有狂魔血脉之人，一旦因为极度愤怒，机会进入暴走状态，实力能够在瞬间，陡然提升百倍，乃至更高！
<br/>　　后来，狂魔一族中的高人，又苦心钻研出了一门逆天神通，可以让拥有血脉之人，随心所欲的掌控自己的情绪，从而达到随时暴走的地步，从而令整个狂魔一族的实力，得到了飞速暴涨！
<br/>　　这一点，倒是和叶尘的逆生百转有些相似，但不同的是，逆生百转修炼起来十分困难，也就叶尘这种重活过一世的人，才能完美施展。
<br/>　　而对于狂魔一族的人来说，却只是他们的天赋神通，根本无须修炼，轻轻松松就能将自身的实力，提升数百倍！
<br/>　　你说变态不变态？
<br/>　　据传说，在狂魔一族最鼎盛的时代，差一点，就统一了整个修真界！
<br/>　　只可惜，后来，或许是因为那门神通的缺陷问题，血魔一族的血脉，开始变得越来越稀薄，能够施展那门逆天神通的狂魔族人，也越来越少，这才慢慢的没落了，彻底退出了舞台。
<br/>　　记得前世，叶尘有一段时间，对狂魔一族这门逆天神通，十分的眼热，耗费了无数心血，终于在一个狂魔族强者的古墓中，找到了那门神通。
<br/>　　然而可惜的是，经过叶尘一番仔细研究，发现那门神通，只能是拥有狂魔一族血脉之人，方才可以施展。
<br/>　　这个结果，一度让叶尘耿耿于怀了许久，一直到后来，得到了逆生百转这门神通，这才算好受了些。
<br/>　　就在叶尘思绪纷飞之时，处于暴走状态的那名狂魔少年，此时已经又打飞了两名恶狼帮弟子。
<br/>　　其余一众弟子，看到突然像变了一个人的少年，顿时不由得连连倒退。
<br/>　　“好小子！”
<br/>　　为首的那名恶狼帮弟子，同样是一脸惊色，不过他毕竟是筑基境修为，在眼下这个小星球，也算是高手了，自然不能被一个刚刚踏入炼气境的小子给吓住，当即飞身扑了过来，狠狠一拳砸出。
<br/>　　那狂魔少年，同样也是一拳砸出，
<br/>　　轰隆！
<br/>　　两人的拳头，在空中狠狠相撞在一处。
<br/>　　蹬蹬蹬！
<br/>　　狂魔少年顿时倒退了四、五步，而为首的那名恶狼帮弟子，同样倒退了两步，顿时满脸的骇然之色！
<br/>　　要知道，他可是正儿八经的筑基境中期，而对方呢，不过才仅仅只有炼气境一重，而且肉身也如此孱弱，竟然几乎和他打了一个旗鼓相当？
<br/>　　这简直就像是在做梦一般啊！
<br/>　　另外一边，原本已经奄奄一息的白发老者，看到孙子的变化，苍老的脸上，但是露出一抹狂喜之色，竟像是回光返照一般，脸色居然红润了不少，
<br/>　　“太好了！太好了！没想到我狂魔一族的血脉，终于又重现世间了，而且还是我的孙子！哈哈哈！咳咳咳……”
<br/>　　然而很快，老者便高兴不起来了，
<br/>　　恶狼帮众弟子反应过来之后，纷纷朝那少年围攻了过来。
<br/>　　那少年，虽然拥有狂魔血脉，并进入了暴走状态，可是毕竟底子太薄，而且根本不懂任何武技，只知道犹如无头苍蝇般乱打一气，很快就被恶狼帮众人死死压制，
<br/>　　“这小子有点邪门！老大应该会感兴趣，抓活的！”
<br/>　　为首的那名恶狼帮弟子，冲着大喝了一声。
<br/>　　而那老者，看到孙子在众人的围攻下，已经开始节节败退，顿时大急，连忙大吼起来，
<br/>　　“虎子！快跑！你快跑！！”
<br/>　　“爷爷！”
<br/>　　那少年原本还以为老者已经死了，此时听到老者的呼喊声，顿时大喜过望，当即回头望去，竟然都忘记了抵挡。
<br/>　　“就是现在！”
<br/>　　恶狼帮众弟子见状，顿时抓住几乎，纷纷朝那少年飞扑过来。
<br/>　　“啊！！”
<br/>　　那少年恢复了几分理智之后，身上的狂暴状态也顿时弱了几分，顿时吓的双手抱头，蜷缩在了地上。
<br/>　　“唉！”
<br/>　　远处地上的老者，看到这一幕，顿时重重叹了一口气，彻底绝望。
<br/>　　然而，
<br/>　　就在这时，
<br/>　　砰砰砰！
<br/>　　伴随着一连串响声，那十多名恶狼帮弟子，几乎同时飞了出去。

<br/>　　这个突然出手之人，自然正是叶尘。

<br/>　　少年惊愕的望了一眼远处，那十几具恶狼帮弟子的尸体，缓缓抬起头，眼眸深处，透着浓浓的畏惧之色，结结巴巴的开口道。
<br/>　　作为修真界最底层的存在，少年对于强大的修真者，有着天然的畏惧心理。
<br/>　　叶尘微微一笑，点了点头。
<br/>　　“噗通！”
<br/>　　少年惊愕过后，当即跪倒在了地上，向叶尘砰砰磕了几个响头，
<br/>　　“多谢大人救命之恩！恳求大人也救救我爷爷吧！我爷爷他，他快要死了！”
<br/>　　叶尘瞥了一眼远处的那名白发老者，并没有立即答应，即便那老者真的死了，他也能让他起死回生，
<br/>　　“想要我救你的爷爷可以，但是你必须答应我一个条件！”
<br/>　　叶尘不急不缓的开口道。
