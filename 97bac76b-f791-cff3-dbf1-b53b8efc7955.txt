　　依靠气魄光膜腐蚀地面，苏重一路下行。
<br/>　　开始的时候很顺利，光膜释放出灼热的侵蚀能量，坚硬的岩石如同枯木，一碰就变成了粉末。更在在气魄的强大力量下，直接压进了周围岩壁之内。
<br/>　　可随着不断深入，巨大的压力，让岩石变得越发坚硬。气魄光膜的侵蚀速度在变慢，虽然只有一丝，但苏重明确感觉到了速度在变慢。
<br/>　　苏重不理会，继续向下前进。可速度却越来越慢。开始的时候，他一个小时能下降数千米。可到了现在，他一个小时，只能向前移动不到一百米！
<br/>　　机械的释放气魄侵蚀岩石，拓宽通道。地下黑暗一片，苏重根本感觉不到时间的流逝。
<br/>　　自从有了生命虹吸，进食已经不再是必要。没了三餐提醒，苏重对时间的感官越发模糊。不知道过了多久，他突然停了下来。
<br/>　　嗤嗤嗤！
<br/>　　一阵白烟冒起，苏重眉头一挑。气魄光膜的侵蚀，竟然遇到了阻碍？
<br/>　　“之前的岩石虽然坚硬，但却挡不住气魄侵蚀。即便能抵抗，却根本无法阻挡。现在这块石头，竟然能阻挡气魄？”苏重眼睛一眯，伸手摸向漆黑岩石。
<br/>　　嗤！
<br/>　　苏重眉头一挑，好高的温度！收回手，手掌上，黑红色纹路弥漫。竟然激发了身体动防护？
<br/>　　嗡！
<br/>　　气魄猛然扩散，所过之处，岩石全部华化作黑灰，快速挤进四周岩层内。
<br/>　　身周空间迅速扩大，眼前出现了一块带着微弧的黑色墙壁。
<br/>　　不远处，庞大的世界树根须，直直扎进岩壁之内。两者紧紧的嵌合在一起，没有一点儿缝隙，就好像长在了一起。
<br/>　　四周看了看，整个黑色墙壁深入四周岩壁，不知道延伸到哪里：“好大的一面墙，想要绕路根本不可能。”
<br/>　　“熔岩肯定就在这后面！”
<br/>　　走到世界树根须前，苏重伸手按在粗粝的根须之上。
<br/>　　生命虹吸！
<br/>　　嗡！
<br/>　　一道绿色能量猛然被拖拽出来。庞大的生命能量迅速融入苏重体内。松开手，苏重停止继续吸收世界树的生命能量。
<br/>　　他能够感觉到整个世界树，那磅礴的生命力，好似爆发大海一般。之前他发动生命虹吸，不管是多么久远的古树，几个呼吸就能被他吞噬干净能量。
<br/>　　“加强版的生命虹吸，竟然只能撼动一根根须？”苏重甚至能感觉到世界那种有一种模糊的本能，一种束缚吸收周围能量的本能！
<br/>　　“怪不得能够吸收岩浆，生命场真是太强大了。强悍的生命力场，约束着自身的生命力，同时吞噬周围的一切能量！”苏重感觉，自己和世界树很像。
<br/>　　他有道种操控自身生命力，世界树同样有着自己的生命场。他有生命虹吸吸收能量，世界树同样有着强大的根须和枝叶吸收能量。
<br/>　　“或者说，这是强大的个体生物的共性！都有着一种不断向内凝聚，让自己底蕴更加强大本能！”
<br/>　　想要像吸收其他树木那样吸收世界树，根本就不可能！
<br/>　　“只有突破了对方的生命场，才能彻底吸收世界树。和随手可得的岩浆相比，这样的效率太低了。”
<br/>　　苏重摇头放弃了吞噬世界树的想法。
<br/>　　“虽然不能完全吞噬，但侵蚀一段根须还是可以的。”伸手一拍粗粝根须。震劲猛然爆发。
<br/>　　砰！
<br/>　　眼前的根须瞬间化作粉末。可惜，粉末化只是蔓延了十多米就停了下来，露出了灰白色的木质部分。
<br/>　　“只能破坏眼前这一小段了。”苏重感叹着世界树生命场的强横。不刻意爆发的情况下，生命虹吸也就只能吞噬根须的一小部分。
<br/>　　呼！
<br/>　　一股热风陡然吹来，苏重衣服被刮的咧咧作响。
<br/>　　在近乎封闭的地底，竟然挂起了灼热狂风。转头看去，刚才还嵌在黑色岩壁内，严丝合缝的根须，露出了一条缝隙。
<br/>　　隐隐的红光从裂缝中射出，热风就是从那里吹来。
<br/>　　不再犹豫，苏重大步走进缝隙。七拐八拐，眼前陡然一亮。
<br/>　　一个巨大无比的岩浆湖出现在苏重脚下。一眼看去，竟然看不到边界！
<br/>　　这是一个鸡蛋样的地下空间，下方全是不断冒泡的岩浆。四周岩壁黑漆漆一片，全都是被岩浆不断融化凝固起来的坚硬岩石。其构造之复杂，即便是顶级实验室的尖端设备也无法复制。
<br/>　　半弧形的顶部，无数黑色石柱子垂下，插入下方岩浆湖。心里一动，苏重仔细看去。立刻发现，那哪里是什么石柱，根本就是世界树的根须！自己身边就有一根，直直插来脚下岩浆内！
<br/>　　“怪不得世界树根须那么难以吞噬，就连岩浆热量都能承受，可见木质之坚硬，岂是那么好是破坏？”
<br/>　　“这可真是个修炼圣地！”这么一大片岩浆湖，可比枯枯戮山的火山熔岩大多了！
<br/>　　苏重双眼放光，直接一跃而下，竟直接跳进了岩浆湖内。
<br/>　　嗤嗤嗤！
<br/>　　几乎在进入岩浆湖的瞬间，全身衣服毛发全被灼烧干净。几乎本能的，气魄光膜骤然爆发，将苏重完全笼罩。皮肤上，黑红色鳞甲状纹路浮现，覆盖全身。眉心处火焰印记，越发鲜明，不断摇曳。
<br/>　　嘶！
<br/>　　苏重倒吸一口灼热空气，尽管有气魄保护。可灼热高温的侵蚀，实在是太疼了！
<br/>　　只是片刻，皮肤就已经被高温灼伤，苏重甚至闻到了焦肉的味道！
<br/>　　道种猛然震颤，一股凉爽的气流从体内升起。被灼烧的皮肤肌肉，被快速回复。
<br/>　　黑色焦匹皲裂脱落，露出白嫩皮肤。可下一刻，刚恢复的皮肤立刻又被高温灼伤。
<br/>　　“给我吸！”
<br/>　　道种狠狠一颤，巨大的吞噬力量骤然产生。身周鲜红色岩浆猛然黯淡。紧接着又被周围岩浆热量炙烤成红******龙七段猛然发动，吞进体内的能量迅速送到身体各处，不断改造这身体。
<br/>　　苏重要紧牙关，整个人全身肌肉不断颤动。皮肤底下好似有小老鼠在爬动，鼓起一个个肉瘤，在身体之内不断窜动。
<br/>　　这已经不是强化，而是完完全全的改造！苏重甚至能听到隐隐的骨骼碎裂声！那是骨骼破碎重塑的声音！
<br/>　　体表的黑红色纹路猛然一亮，黑色更深沉，红色越发鲜亮。远远看去，就和地下的岩浆湖面似得，一片斑驳。
<br/>　　叮叮叮！
<br/>　　脑海之中，技能面板的数据急剧变化。
<br/>　　魔龙七段后面的经验值数据，几乎以肉眼可见的速度不断攀升。
<br/>　　一刻钟的时间，就已经飙升了一百多个点！
