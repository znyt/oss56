　　叶少阳点点头。这时候车来了，他上车走人。
<br/>　　回到家里，卫生间开着灯，叶少阳好奇地进去，见是瓜瓜在摆弄热水器，陈悦站在边上，见到叶少阳，有点尴尬。
<br/>　　“我娘想洗澡，但她用不好这个，我帮她弄一下。”瓜瓜解释。
<br/>　　“哦。”叶少阳转身进屋，突然想到什么，又摸到卫生间，问陈悦：“你不是说你除了自己的身世，别的常识什么的都想起来了？”
<br/>　　“是的，这些楼房、汽车什么的，我都见过的，但我没用过这种东西。”陈悦指着热水器说道。
<br/>　　热水器都没用过？
<br/>　　连自己以前茅山上都装有太阳能热水器，难道她住的地方比茅山还落后？简直不敢想象。
<br/>　　躺在床上，叶少阳又试着联系周静茹，结果微信不会，打电话手机也关了。
<br/>　　一想到离别的时候她挥手的样子，叶少阳心里就特别难受，只好去吐纳，借这个来排解内心的烦闷，吐纳了几个周天，情绪的确缓和了很多。
<br/>　　叶少阳开始思索将来的事情……不知不觉就想到了陈悦，第一件事，当然就是先弄清她的身份，叶少阳思前想后，还真想到了一个办法。
<br/>　　第二天一早，他电话联系谢雨晴，提出了自己的想法。
<br/>　　“你想通过她的户籍资料找她？可是她未必叫陈悦，就算这是真名，这名字也太普通了，不知道全国各地有多少。”谢雨晴解释。
<br/>　　“这么说，没办法了？”
<br/>　　谢雨晴想了一下，说道：“可以试试指纹。我们国家近十年一直在做指纹采集，大部分人都录了，你应该也录过吧。”
<br/>　　叶少阳想了一下道：“好像在茅山的时候，当地派出所找上山录过。”
<br/>　　“是啊，现在第一次办身份证都会采集指纹，虽然也有些很多人没录，不过可以试一试。你这就把她带来吧。对了你们吃饭没有。”
<br/>　　叶少阳表示刚起床。谢雨晴说自己也没吃，让他带着陈悦，一起去吃大肉包子。
<br/>　　叶少阳去敲响陈悦的房门，陈悦早就醒了，叶少阳表达了自己的计划，陈悦也想弄明白自己的身份，欣然同意，立刻就要跟他出门。
<br/>　　“这不行，你得换个衣服。”叶少阳指了指她身上的古装，穿着倒是好看，但穿越感太强，出去怪怪的。
<br/>　　“我没衣服。”陈悦道，她是空手回来的。就身上这一套衣服。
<br/>　　“我有啊。”
<br/>　　“我总不能穿男人衣服。”
<br/>　　“谁让你穿男人衣服了。”
<br/>　　芮冷玉的衣服，装了满满几个柜子，叶少阳打开让她自己挑，结果睹物思人，站在一旁发起呆来，决定自己要立刻去见芮冷玉，顺便去找道风。
<br/>　　陈悦挑了一些衣服，去房间换了出来：一件高领毛衣，下面是牛仔裤和运动鞋，外面罩着一件米黄色的风衣。
<br/>　　叶少阳是第一次看到她穿现代衣服，还真的挺不错，又拿了一条芮冷玉的围巾给她，陈悦自己照着镜子，说道：“这是你女朋友的衣服吗。”
<br/>　　“对。”
<br/>　　“她很有品位。”陈悦称赞道。
<br/>　　在包子铺见到了谢雨晴。
<br/>　　“你们点餐，我点过了。”谢雨晴拿出一个手提袋，扔给叶少阳，“太沉了，你自己拿着吧。”
<br/>　　“什么东西？”
<br/>　　“天冷了，给你买了件大衣，你先试试，不合适你自己去换。”
<br/>　　叶少阳心头一暖，当即拿出来换上，是一件黑色大衣，穿着还挺合身。
<br/>　　好久没吃大肉包子，叶少阳要了三笼，和一碗豆浆，那边陈悦说自己不吃荤腥，要了韭菜鸡蛋的包子。
<br/>　　“你为什么不吃荤？”谢雨晴很好奇。
<br/>　　“不知道，我记忆中，好像就没吃过肉。”
<br/>　　“你不会是……”谢雨晴本想说“尼姑”两个字，但看到她一头茂密的长发，忍住没说。
<br/>　　大肉包子还是那么好吃，叶少阳跟谢雨晴边吃边聊，两笼包子下肚，吃饱喝足，几个人一起来到警局。
<br/>　　谢雨晴让叶少阳和瓜瓜在她办公室呆着，自己带着陈悦去查指纹。
<br/>　　叶少阳在她办公室无聊呆着，又尝试给周静茹发微信，还是没有回应，看了下她的朋友圈，倒是更新了一条：
<br/>　　是一张照片，照片是别人拍的，一个姑娘的背影，站在海边，双臂高举成一个“V”字，
<br/>　　，下面配着一句文字：
<br/>　　祝福自己，重新启程。
<br/>　　叶少阳仔细辨认了下，从背景看还真是周静茹。
<br/>　　看到这照片，叶少阳心里有点酸，但也为周静茹高兴，至少是个新的开始。
<br/>　　放下手机，叶少阳走到窗前去站了一会，突然身后有人推门进来，叶少阳回头一看，是一个年纪挺大的男警官，有点面熟，好像哪里见过？
<br/>　　“叶少阳，你怎么在这。”警官微微一笑，他自然看不到瓜瓜的存在。
<br/>　　他认识我？
<br/>　　叶少阳愣了一下，突然想起他的身份，靠，这不是谢雨晴他爹吗，本市的警察局长，之前见过一面的。
<br/>　　“啊，谢伯伯，你……你咋来了。”
<br/>　　“我找雨晴有点事，你坐你的。”谢局长自己走到谢雨晴的位置上坐下，问叶少阳去哪了。叶少阳表示有个朋友要做指纹鉴定，谢雨晴带她去了，自己在这等着。
<br/>　　谢局长也没有往深了问，打量着叶少阳，跟他闲聊了一会，问起近况什么的，叶少阳也就随便回答，正聊着，谢雨晴推门进来，一眼看到他爹，愣了一下，把陈悦拉进来，一边问他爹什么事。
<br/>　　“还是那件事，你考虑怎么样了，名额比较紧，你给个结果，我好去操作。”
<br/>　　谢雨晴表情有点不自然，道：“回头我去找你说吧，我……还要考虑一下，再说我现在有正事呢。”
<br/>　　谢局长起身，跟叶少阳打了招呼，邀请他没事去家里坐坐什么的，然后走人了。
<br/>　　“什么名额？”等他走后，叶少阳好奇地问谢雨晴。
<br/>　　“没什么，说了你也不知道，说正事。”谢雨晴看了一眼陈悦，道：“指纹比对了，还真的找到了她的身份！”
<br/>　　叶少阳和瓜瓜都是一惊。
