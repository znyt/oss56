<br/>    血腥味逐渐蔓延，鲜红色的血水融汇在九幽寒泉之中，蔓延到四方。
<br/>    
<br/>        无尽的黑暗之中，一座小山忽然被轰开，躲藏在其中的人王们抬头，是一只王级军团，足足有着四十人，他们都带着挎腰魔刀，身着黑衣。
<br/>    
<br/>        他们想要逃走，却是发现，他们的去路，已经被对方这一只王级军团封死，“我们已经没有退路了！”
<br/>    
<br/>        “现在剩下的，唯有死战！”
<br/>    
<br/>        “我就是死，也不会让这些肮脏的魔崽子碰到我的身体！”
<br/>    
<br/>        领头的王级军团统领冷眼看了一番这些人王，淡淡道：“全杀了，并没有重要目标！”
<br/>    
<br/>        四十柄长刀飞舞，朝着这些人王飞去，这些人王已经看到了自己的死亡近在眼前，长刀纵横飞舞之间，魔气已然笼罩全场。
<br/>    
<br/>        就在这时，无数柄灵剑自黑暗之中飞来，化作一条剑龙，穿梭在王级军团之中，一个魔王接着一个魔王化成碎片，王级军团统领看向周围，“是谁？”
<br/>    
<br/>        陈明的声音在黑暗之中响起，“一个完整的王级军团，还真是一条大鱼啊！”
<br/>    
<br/>        王级军团统领看向陈明，大喝道：“你这是要和我魔宗为敌？”
<br/>    
<br/>        陈明笑道：“人都杀了，还说这些没用的做啥？”
<br/>    
<br/>        王级军团统领道：“以阁下的实力，我可以给你一次机会，效忠我魔宗，我魔宗此次前来的魔君一定会放你一条生路，而且还有着魔宗宗主之位等着你！”
<br/>    
<br/>        陈明道：“你说的一点诱惑力都没有，而且我也不需要考虑，在下，是他们所有人的希望！”
<br/>    
<br/>        王级军团统领道：“你会后悔的，魔君已经知道了关于你的消息，你就等待着魔君的镇压吧！”
<br/>    
<br/>        陈明挥手，万道天竹舞动，这一位王级军团的统领终究是倒在了数百剑之下。
<br/>    
<br/>        陈明看向那些惊魂未定的人王们一眼，认真道：“好好活下去。”
<br/>    
<br/>        道完，陈明已经重新托起卓青瑶，离开了这里。
<br/>    
<br/>        人王们终于是冷静了下来，看向陈明刚刚离开的身影，“刚刚那一位，是宗师吗?”
<br/>    
<br/>        一位人王精神振奋道：“是的，刚刚那位就是宗师，在凶兽潮来临之际，便是宗师带领四大天才斩杀了龙鲲！”
<br/>    
<br/>        “宗师果然没有放弃我们！”
<br/>    
<br/>        本来已经是心存死志的人王们重新燃起了求生的欲望，“你们不要忘了，有一句话流传在我们所有人之间，当天劫到来之时，便是我们重见天日之时！”
<br/>    
<br/>        “我们要活下去，活到我们可以反攻的那一刻！”
<br/>    
<br/>        “宗师会给我们所有的希望！”
<br/>    
<br/>        陈明斩杀了一支王级军团的消息在人王之间流传，他们开始相信，他们有着重见天日的那一天，因为宗师，正在经历达到大君的最后一步，杀人证道！
<br/>    
<br/>        无尽的黑暗之中，升起了一抹曙光，那一抹曙光的名字，叫做陈明。
<br/>    
<br/>        越来越多的人王得知了这个消息，并成为在无尽的黑暗中，继续在恐惧中活下去的希望。
<br/>    
<br/>        “我们终有一日会重见天日，而那一刻，我们会在宗师的带领下撕碎所有的魔宗杂碎！”
<br/>    
<br/>        所有人王不安之时，只要说出这句话，大家都会安静下来。
<br/>    
<br/>        人王们得知了这个消息，魔宗自然也是知道了这个消息，解千愁伸手将斩草握住，然后问道：“怎么回事，这些仙门正道，为何会龟缩起来，按照仙门正道的做法，他们不是应该集结起来到我们面前送死吗？”
<br/>    
<br/>        一个王级军团统领道：“在我们杀掉的那些人王之中，每一位人王死前都曾嘶吼过一句话。”
<br/>    
<br/>        解千愁有些感兴趣了，“什么话？”
<br/>    
<br/>        王级军团统领道：“他们说，天劫到来之日，就是他们重见天日之时！”
<br/>    
<br/>        解千愁愣了愣，“天劫？人王的天劫，有人要突破到大君了吗？”
<br/>    
<br/>        王级军团统领道：“是的！根据消息，在这些人王之中，有着一位炼丹宗师，已经是到了最后一步。宗主，我们是不是将派出去追杀正道的王级军团全部撤回来？”
<br/>    
<br/>        解千愁笑了笑道：“撤回来，开什么玩笑？”
<br/>    
<br/>        “一位大君而已，没有什么值得惊慌的，一位新晋大君而已，能够有多强？要知道，我们在这里的时间不多，最多只有三年，时间再长，雪山宗就会有所察觉了。继续追杀下去，就让他成为大君，带领着所有的人王前来送死吧！”
<br/>    
<br/>        王级军团统领领命，解千愁发出一抹轻蔑的笑容，“天劫到来之时，便是他们重见天日之时，哈哈哈哈哈，还真是可笑啊！孩子的游戏就让孩子们去玩吧，告诉魔宗七大弟子，他们可以动手了！”
<br/>    
<br/>        陈明行走在黑暗之中，身后是卓青瑶，让卓青瑶跟着紫霞仙子她们，陈明实在是不放心，“师父，你已经杀了五十四人了。”
<br/>    
<br/>        陈明道：“还是太少了，继续找吧。”
<br/>    
<br/>        又是三天过去，陈明感受到了一抹魔气，“魔气还在，追过去看看！”
<br/>    
<br/>        追了十里，陈明可以看得到，“在前方，有着二十个黑衣人，先把他们杀了吧！”
<br/>    
<br/>        二十个人王而已，陈明冲上前去，龟口之中吐出两千四百柄灵剑，瞬间组成了一支大阵，在对面，一个头顶有着长着两根弯角的年轻人哈哈大笑道：“陈明是吗？我焦木天可是等了你很久了！”
<br/>    
<br/>        陈明看了对方一眼，魔气彭拜的程度已经是无限接近大君，扫了一眼危险指数，1500+100。
<br/>    
<br/>        这也是一位悟道榜上的天才，而且排名还不低，至少是前五。
<br/>    
<br/>        焦木天手中出现一柄绿色的矩尺，看向陈明，“听闻你便是那个有望突破到大君的人，我倒要看看，你这个臭王八能够有多强？”
<br/>    
<br/>        卓青瑶怒道：“师父，他骂你！”
<br/>    
<br/>        陈明无所谓的笑笑，然后道：“将死之人，何必跟他多费口舌？”
<br/>    
<br/>        陈明浑身法力爆涌，同时开启，两千四百柄灵剑朝着对面的二十黑衣人直接飞了过去！
<br/>    
<br/>        只听见一阵噼里啪啦的响声，对面已经只剩下焦木天一个人还站着。
