　　握着金色光团，慕清霄身躯紧绷，脸色前所未有的苍白，显露出一丝病态。
<br/>　　旋即，一抹金色血迹自嘴角流淌而下，随后滴落在布满灰尘的地面上，黑色勾玉消失，眼中却充满了笑意。
<br/>　　见慕清霄流血，小医仙脑子里变得空白，脸色也随之一白。
<br/>　　下一秒，不顾周围的光团与目光，迅速冲到他的身边，纤纤玉手轻轻的抓着他的胳膊，娇躯轻微的颤抖起来。
<br/>　　来到他的身旁，小医仙却手足无措起来，只能担忧的看着他。
<br/>　　他可是斗皇，一路下来，那么多危险都奈何不了他，怎么就突然受伤了？
<br/>　　小医仙美眸中泪光闪烁，慕清霄就是她的全部，若是出点意外，她真的不知道该怎么办才好。
<br/>　　抬起左手，抹去嘴角的血迹，慕清霄露出一个牵强的笑容，道：“别担心，小伤而已，养几天就好了。”
<br/>　　嘴上虽这么说，但他的心却一下子沉到了谷底，只有他自己知道，身上的伤势有多么的严重！
<br/>　　刚才的一击，堪比高阶斗宗，恐怖的冲击力，全都被他一个人挡了下来，否则屋内除了他自己，其他人全都得死。
<br/>　　好在经过多番雷劫的洗礼，他的肉身已经远比其他斗皇强的多，估计斗宗肉身都没有他来的强。
<br/>　　饶是如此，他的修为也一下子掉到了元婴级别！
<br/>　　直接从化神八重，掉落至元婴，简直就像从天堂坠入地狱，当然，以他凤凰血脉的恢复力，根本不是人类能够比拟的。
<br/>　　除了渡劫的时候，真是第一次受到如此严重的伤势，想要恢复，恐怕需要静养一段时间了。
<br/>　　虽然在这一击下，深受重伤，但慕清霄依旧很兴奋，如此恐怖的能量，其中的东西该有多么的珍贵？
<br/>　　他的脸色不见恢复，小医仙哪能不知道是在安慰自己，眼泪如珍珠一般，顺着脸颊滑落而下，无声的掉落在地面。
<br/>　　轻轻将她揽入怀中，慕清霄叹息一声。
<br/>　　“真的没事，静养一段时间就好了，我什么时候瞒过你？乖乖听话，别在哭了，在哭心都要碎了。”
<br/>　　见她眼泪止不住的往下落，慕清霄捏了捏她小巧的鼻子，道：“怎么，不相信我？”
<br/>　　小医仙轻轻靠在他的胸膛上，不敢靠的太紧，生怕弄疼他，见他依旧安慰自己，眼泪也慢慢止住。
<br/>　　但她隐隐能猜到，伤势恐怕是极其严重了。
<br/>　　旋即，慕清霄脸上的笑容消失，眼底闪过一丝黑色神火，瞥了一眼站在四周的五名学员与古薰儿，语气冷漠，道：“怎么？你们想要这东西？”
<br/>　　他的实力现在虽然直接退到元婴期，却也不是区区几只蝼蚁能撼动的。
<br/>　　俗话说，龙游浅水被虾戏，但虾米终究只是虾米，跟龙比起来，完全不是一个层次上的生物。
<br/>　　不对，他是凤凰。
<br/>　　虽然不能杀了他们，但直接打的半死不活丢出去，对他来说也不算难事。
<br/>　　只是一个不经意的眼神，众人就感觉一股死亡的气息缠在了他们身上，使得众人自心底升起一股浓浓的恐惧。
<br/>　　特别是古薰儿，当初在萧家所察觉到的压迫感，直接迎面而来，使得她脸色一白，娇躯猛地一颤。
<br/>　　周围五名男学员后退一步，脸色惨白，眼中的贪婪之色也消失不见。
<br/>　　他们能通过选拔赛，成为这一届的前五名，自然不是蠢货，知道什么东西能动，什么东西不能动。
<br/>　　而面前的青年，无论是他怀中的女人，或者是手中的宝物，都是万万碰不得的！
<br/>　　“哼。”
<br/>　　冷哼一声，慕清霄收回目光，气息也收敛起来，拍拍小医仙的香肩，目光盯着手中的金色光团。
<br/>　　光团依旧在手心中挣扎，慕清霄开始尝试着查探其中的物品，却愕然的发现，根本无法穿透能量层，查看其中的东西。
<br/>　　简直日了狗，身受重伤不说，连其中的东西都无法查探，脸上肌肉狠狠抽了抽，难道放回去，做梦吧？
<br/>　　让他修为大跌，哪有放回去的道理，既然将此物放在这里，可不就是让他们取走的么？
<br/>　　当然，将此物放在这里的人，是否如此想的，那就不得而知了。
<br/>　　无法打开，也就是说其中的东西无法拿走，慕清霄心下一动，松开手掌的一瞬间，其中的金色光团也随之消失不见。
<br/>　　见他手心中的光团消失，屋内五名男学员心底都涌上一丝快意，他们刚才尝试过，由于能量团的缘故，东西根本无法直接放进纳戒中。
<br/>　　也就是说，金色光团重新回到光团大军中了。
<br/>　　想到此处，他们都幸灾乐祸起来，当然，也只敢在心中想想，并不敢表现在脸上。
<br/>　　有的时候，人性就是如此恶劣，得不到的，也不愿意被别人得到。
<br/>　　唯独站在几米外的古薰儿，心中满是困惑，因为金色光团所散发出来的能量波动，在一瞬间消失了，完全无迹可寻。
<br/>　　瞥了一眼慕清霄，偌大的屋内没有一丝金色光团的气息波动，难道真的消失了，又或者......
<br/>　　深吸了口气，慕清霄面色如常，从他的脸上看不出任何一丝的情绪。
<br/>　　此时，他正望着储存空间的一个黑色盒子，愣愣出神，刚才的金色光团并非消失，而是被他装进来储存空间中。
<br/>　　他本也是抱着尝试的心态，没想到真的成功了。
<br/>　　进入储存空间的瞬间，能量也瞬间消失，显露出其中的物品，正是一个四方形的黑色盒子。
<br/>　　盒子看不出是何种材料制的，看上去极为普通，但其上却传来一股沧桑古朴的气息，估计有些年代了。
<br/>　　慕清霄心中一喜，尝试了一下，依旧无法开启，上面有一层强大的禁止，想要抹除，但他现在暂时办不到。
<br/>　　他怀疑这盒子中的东西，可能是迦南学院那半圣院长藏在这里的，至于为何会窜出来，这就不关他的事情了。
<br/>　　不过，如此恐怖的能量，后者恐怕就不曾想过，这东西会落在别人的手中，半圣藏在这里的，绝对不是普通物品。
<br/>　　不管其中有什么东西，对于慕清霄来说，都是捡到宝了。
<br/>　　而且，能量团能收进储存空间中，简直就是作弊啊！
