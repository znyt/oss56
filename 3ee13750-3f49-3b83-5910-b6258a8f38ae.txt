　　龙隐部队特种作战基地。
<br/>　　战术中队在对西伦科特的行动中，阵亡三人，多人受伤，其中伤得最重的是徐旭东；不过这小子也的确命大，送回国的时候，医院已经让人做好徐旭东变成植物人的准备，可是这小子在床上躺了一个多月醒了过来。但是他还能不能继续在战术中队服役，还要通过后面的测试才能确定。
<br/>　　不过徐旭东目前在陈怡的陪同下，正在有计划的进行康复训练，看他的样子，显然是不甘心就这么离开战术中队。这就是部队神奇的地方，加入的时候，可能是因为各种各样外部的原因导致的，所以刚成为其中一员的时候，可能会排斥，可是等到真正融入到其中之后，却又不再舍得离开这里。
<br/>　　至于其他人在接受治疗后也是恢复的很快，已经开始进行康复训练，恢复情况很理想。战术中队中的绝大多数人，包括罗昊在内，距离枪支从作战部队退役的年限还剩下一年到两年的时间，所以赵天宇和云岳早就已经计划着对战术中队的人员补充和替换。
<br/>　　从战备中队、红旗学校，还有各大军区选拔出来符合条件的队员，也已经被送出去进行强化训练，这次那些战术中队后背队员除了“地狱火”四个阶段的训练外，还增加了一个地方，那就是“炼狱”！
<br/>　　他们会被分散到“炼狱”世界各地几个军事训练基地，接受这个世界上最残酷、最血腥的训练，云岳要让他们知道，即便是训练也是会死人的，而且这个死亡率要远远高出龙隐部队选拔的死亡率，在“炼狱”甚至每一天都会看到有队员在训练过程中死亡，被送出“炼狱”训练基地。
<br/>　　这种残酷是直击内心的，会让人真正感觉到，死亡真的就在自己身边，可能下一秒子弹就会打穿自己的脑袋，让自己血溅当场。
<br/>　　在那些战术中队新队员回来之前，罗昊他们还是主力队员。云熙从红旗学校返回龙穴后，就把从硬盘中发现的东西告诉了赵天宇和云岳，包括那张残缺，没有任何标注的地图也是拿了出来。
<br/>　　赵天宇、云岳、朱致阳三个人对那张小丑和玛丽科娃的合影感到有些意外，这三个都是老狐狸，他们的嗅觉何等的灵敏，自然很快就察觉到了照片的价值。
<br/>　　只要抓住玛丽科娃这根线，就有很大概率找到小丑！至于那张地图，赵天宇、云岳、朱致阳三个人都是没有任何头绪，他们跟罗昊一样，也只能确定只是一处山脉，可是地图最关键的一块缺失了，而且地图上没有任何的标注，世界上山脉有那么多，仅仅只是凭借一副残缺的地图，根本不可能确认地图上的山脉到底是在什么地方。
<br/>　　“无常，立刻到我办公室来一趟！”
<br/>　　赵天宇拿起桌子上的对讲机，通过对讲机让慕容飞立刻到自己的办公室报到。要说对玛丽科娃这个女人，最熟悉她的人就是慕容飞！
<br/>　　“报告！”
<br/>　　接到赵天宇命令之后，慕容飞很快就小跑到赵天宇的办公室。
<br/>　　“进来！”
<br/>　　赵天宇招呼慕容飞进入办公室后，把自己手中的平板电脑递给他，问道：“照片上这个女人还认识吗？”
<br/>　　“我还以为是什么事情呢，就是让我看照片啊……”
<br/>　　慕容飞接过赵天宇递给自己的平板电脑，当他看到平板电脑中的照片时，脸色立刻一变，脸上的笑容也是消失不见，眼睛中冷冽之色不断闪烁，咬牙切齿的说道：“玛丽科娃！”
<br/>　　尽管事情已经过去了很多年，可是在看到照片的时候，慕容飞还是第一时间就认出了照片中的女人是谁！当年那个被泰勒所杀，原本在回国后会成为自己观察手的队员！
<br/>　　慕容飞的心理障碍，因为佟千雪的关系已经克服过去，可是这并不能说慕容飞已经忘记了这笔血债！忘记了这笔血债的源头——玛丽科娃！
<br/>　　“她跟小丑有关系？”
<br/>　　慕容飞深吸一口气，让自己恢复冷静，把平板电脑放到赵天宇的办公桌上。
<br/>　　“我们现在初步认为，玛丽科娃跟小丑之间关系亲密，觉得可以通过玛丽科娃能找到小丑。”云岳这个时候，站起身来，对慕容飞说道：“而我们所有人中间，对玛丽科娃了解最多的就是你。”
<br/>　　“当年在‘Red-Crow’行动之后，因为遭到玛丽科娃的报复，我们也想过要找到玛丽科娃，但是这个女人十分狡猾，没有直接的线索。而最后，行动在我杀了泰勒之后，宣告终结。”慕容飞叹了口气，说道：“当年，我们在北美洲，也就是泰勒居住的地方找到了一些东西，可是那些东西都被‘黑水’公司拿走了，而且这么多年过去了，玛丽科娃这个女人不会蠢到还继续使用那些已经被我们发现的情报网络。”
<br/>　　“那也就是说我们还是在原地踏步，除了拿到这张照片，能够说明玛丽科娃跟小丑之间有着特殊关系外，其他并没有什么突破。”罗昊略显失望，本来以为这次会是一个巨大的突破，结果根本就没有什么突破口存在。
<br/>　　“这次我回杭山市倒是发现一件事情，或许顺着这条线能够发现一些东西。”
<br/>　　慕容飞想了想，说道：“上次因为回去相亲，帮助杭山市市局的人解决了一个危险分子，这次回去又遇到了市局的人，他们正在盯梢一个叫李霄力的人，这人具体什么背景我不清楚，但跟他们简单的聊天中，我了解到这个李霄力可能跟境外贩毒组织有联系，而且最近那个李霄力在他所开的夜总会里拍卖一种叫做‘crystal-ball’，‘水晶珠’的东西。”
<br/>　　慕容飞拿起桌子上的平板电脑，在电脑上输入了几个关键词后，很快就从网上找到了关于“水晶珠”的新闻，把平板电脑递给赵天宇说道：“这个‘crystal-ball’是由这家叫做‘维伦恩’生物制药公司最新研发生产，用来解决男性那方面障碍的保健品。”
