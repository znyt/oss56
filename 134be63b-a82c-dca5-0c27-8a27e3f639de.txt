　　第83章
<br/>　　刚走入蛤蟆寨就嗅到一股刺鼻的辣味，小白狼捂住鼻子，看到蛤蟆寨支起十几口大锅，正翻炒着辣椒。
<br/>　　“白当家，好久不见，你这么有空过来？”徐神机迎过来。
<br/>　　“把姓程的叫出来，老娘要和他算账。”
<br/>　　如果是换做旁人在蛤蟆寨大呼小叫，以现在蛤蟆寨的格调，徐神机早就令人抄家伙了。可是眼前这人是小白狼，倒不是因为她的身份，而是因为她的性别。
<br/>　　一个女人找一个男人算账，这还能算什么帐……桃花帐？嗨，大当家下手可是真快呐，根本是没给自己留机会。
<br/>　　只是，也不记得大当家最近去过杏花岭啊，当然，这种龌龊事也不可能广而告之，不管了，自己还是去给她叫人去吧，这母夜叉自己可惹不起，一把钢刀，专门朝人下三路招呼。
<br/>　　程大雷过来的时候，还极有风度的和小白狼打了声招呼，对于那美女如云的杏花岭，他也是身不能至，但心向往之。
<br/>　　谁料小白狼根本没给他好脸色，直接道：“姓程的，你答应我的事还算不算数！”
<br/>　　徐神机的眼睛立刻亮起：果然有事。
<br/>　　附近的秦蛮，林少羽也递过来一只耳朵。
<br/>　　“什么事？”程大雷一头雾水。
<br/>　　“这……”小白狼有些不好意思说出口，一亩辣椒有两三千斤的产量，可种其他粮食不过几百斤，别人会不会说自己穷疯了。
<br/>　　“哼，你难道不知道是什么事嘛！”
<br/>　　唰！
<br/>　　不知多少道目光投到这里，眼中八卦之火熊熊灼烧。这青牛山有名的母夜叉，雌老虎什么时候露出过这种小女儿的姿态，还说他们两个之间没事。
<br/>　　程大雷一头瀑布汗，他仿佛又回忆起被高飞豹和于虬然支配的恐惧。
<br/>　　“你特么究竟什么事？”程大雷咬牙。
<br/>　　“就是辣椒……”
<br/>　　程大雷恍然大悟，道：“辣椒啊，对了，你们山寨收了多少辣椒？”
<br/>　　“有两千多斤吧。”
<br/>　　“这么少……”程大雷嘀咕一声，道：“那运过来吧，还按照之前说好的，一斤辣椒一斤粮食，小麦或大豆任选。喔，对了，运过来之前先炒一遍，干炒。”
<br/>　　程大雷现在已经收到苏樱的信，和那客商谈好了价格，按一斗辣椒一斗小麦的价格收购。
<br/>　　这价格超过程大雷想象很多，种粮食一亩不过几百斤的产量，种辣椒却能达到数千斤。而程大雷和小白狼兑换是用重量做单位，可和客商兑换，却是用容量作单位，一斤辣椒和一斤小麦的体积自然是不同的。
<br/>　　只是那客商吃不下这么大体量的货，只准备要五千斤。不过这已让程大雷很满意，一口吃不成胖子，销路要慢慢打开。
<br/>　　说完见小白狼还没走，程大雷心里嘀咕着，难道这姑娘不想走，想要和我多唠唠，的确蛤蟆寨如今也算得上财大气粗，自己对女性有些吸引力也是应当。
<br/>　　“要不去我屋里聊聊，我新作了一首诗……”
<br/>　　“……”小白狼。
<br/>　　程大雷过高的估计了自己的吸引力，她并不是想和程大雷唠唠，而是一时之间不太敢相信。这次过来，程大雷能补她几百斤的大豆就不错了，所以才特意拿出一副泼妇的态度，怕的是程大雷不认账。
<br/>　　可结果……事情太好了，好到让人不太敢相信。
<br/>　　等反应过来后，小白狼立刻下山，一溜烟不见影了。
<br/>　　程大雷遗憾的叹口气，难道是不喜欢诗，下次聊点别的。
<br/>　　山下还有人守着，看到小白狼失魂落魄的表情，顿时都懂了。
<br/>　　肯定是程大雷不认账了，对呀，世间哪有这样的好事。于是这些人心里的石头也落了地，这种感觉，类似于听说自己一个朋友中了彩票，他在家里翻来覆去睡不着觉，后来又听说彩票丢了，顿时喜上眉梢，结结实实睡个好觉。
<br/>　　“白当家，别往心里去，没多大事。”
<br/>　　“老辈人都说了，天上不会掉馅饼，吃过一次亏以后就都明白了。”
<br/>　　“我当初说什么来着，我当初说什么来着……”
<br/>　　小白狼没有理他们，跨上马就返回山寨，急忙命令山寨的人炒辣椒。两千多斤也得炒几天，杏花岭也没有那么多铁锅。
<br/>　　过后几天，又有几家种了辣椒的山寨来找程大雷兴师问罪，他们怒气冲冲的上山，失魂落魄下山，回山之后就开始炒辣椒。
<br/>　　还是没多少人知道怎么会事，山里也没什么娱乐消遣，最近这件事就成了新闻。大家茶余饭后，免不得发表一下议论。
<br/>　　类似于不听老人言吃亏在眼前之类的。
<br/>　　几家山寨种的辣椒都不多，加起来也不过一万余斤，加上蛤蟆寨自己种的，在四万斤往上。
<br/>　　山寨所有人都在忙碌，程大雷原本准备给刘悲收拾出一间医馆，可无人可用，只好自己亲自上阵。刘悲那恐怖的传染力的确瘆人，但习惯了也没太大事。偶尔悲伤一下，也可以调节情绪么。
<br/>　　如今刘悲还在给杜茂做思想工作，不得不夸一声，杜茂的心志真的很坚定，现在依旧没答应入伙。只是听说精神状态有些不好，已经开始有自残倾向了。
<br/>　　终于将草药都般进医馆，程大雷请柳芷写了两个大字挂在门口，如此，山寨算是有自己的医务室了。
<br/>　　嘟，完成医馆建设，奖励医书一本。
<br/>　　系统提示在此刻传来，程大雷没想到还是个隐藏任务。他很轻松翻出这本医书，看到上面写着三个字：千金方。
<br/>　　千金方：药王所著，医者等级达到优秀可以修习。
<br/>　　程大雷决定交给刘悲，他觉得刘关张这三人的等级都是可以升级的，潜力是传说级别的人物，怎能不重视。
<br/>　　只是，一定要把汉昭烈帝的后人培养成一个医生么？
<br/>　　如今，青牛山有好家山寨都在炒辣椒，味道弥漫得到处都是。终于确切消息还是传出去了……
<br/>　　有人心里开始打鼓，有人不当回事，觉得流言而已，止于智者，而自己就是智者。不过无论如何，已经有人开始睡不着觉了。
<br/>
