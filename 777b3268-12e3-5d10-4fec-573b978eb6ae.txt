　　第746章
<br/>　　毕竟到了帝国第一繁华之地，徐神机和刘发财都想出去逛逛。程大雷也是耐不住寂寞的人，索性便带着刘发财和徐神机一块出去看看。留下鱼洞真保护崔白玉和乔弄云。
<br/>　　谁料正在这个茶摊碰见陈梦。
<br/>　　面对无比热情的程大雷，陈梦与紫脸人明显不知如何应对。同时也感觉自己的智商似乎受到摧残。
<br/>　　“还真是巧了，谁能想到在这里碰上，放心，这顿我请。”程大雷哈哈笑着：“对了，你们怎么来了长安城？”
<br/>　　陈梦愣了好大一会儿，确实不知该以如何态度面对程大雷。他给自己带来莫大耻辱，本该一见面就亮兵器。但面对程大雷热情肆意的笑脸，他觉得亮兵器也有些不合适。
<br/>　　“嗯，那个过来做些事。”
<br/>　　“喔，我明白了，平野城也需观礼。你肯定是代表你父亲来的，对不对？”
<br/>　　陈梦不知说什么好，程大雷也没有猜错，这的确是他的任务之一。
<br/>　　“有没有地方住，不如住我哪里，我的地方还是蛮大的。”
<br/>　　“是呐，是呐，朝廷给了我们一座大宅子，根本住不完，你们可以住进来看看，反正空着也是空着。”徐神机开始夸耀自己受到的礼遇。
<br/>　　陈梦求助的眼神看向自己的师父，他年纪太轻，实在不知该如何应对这样的状况。
<br/>　　紫脸人抬起头，双手抱拳在胸前：“请于程当家一战。”
<br/>　　程大雷脸上的笑容慢慢僵固住，装疯卖傻，终究是没办法避过这一战。他是听到陈梦二人的谈话才过来的，遮遮掩掩，就是不给陈梦开口的机会。谁料这紫脸人还是将话挑明了。
<br/>　　“报个名号吧？”
<br/>　　程大雷随意一句话，陈梦忽地发现，他和刚才产生很大不同。刚才他像个热情的憨货，但此时他挺直腰背，开口你就不得不听。
<br/>　　毕竟是杀千屠万的主儿，程大雷身上的血腥气是想遮也遮不住。
<br/>　　紫脸人抱拳在胸前：“交州徐问天，请于程当家一战。”
<br/>　　程大雷摇摇头：“不好意思，没听说过，想和我动手，你怕还有些不够格。”
<br/>　　就连陈梦，也是第一次知道徐问天的真名实姓。这次长街报名，算是他隐姓埋名十几年后又一次出世。结果，就结结实实撞在程大雷这块铁板上。
<br/>　　徐问天是交州人，交州在帝国算是蛮夷之地。他出身一武馆，其父是开馆授业的武师。
<br/>　　徐问天曾有一小师妹，二人青梅竹马，两小无猜。不过，和所有故事一样，这小师妹心有所属，不恋他这个大师哥，中意一籍籍无名的江湖浪荡客，甚至还有了身孕。
<br/>　　在二人要私奔的时候，徐问天领了师兄弟追捕。他也真是个狠人，亲手将小师妹沉塘，溺死了一对母子。至于那江湖客则是受伤后下落不明，不知去向。
<br/>　　正所谓莫欺少年穷，光阴捻指，徐问天已经娶妻生子，一家人其乐融融。谁料那江湖客竟然又回来了，还不知从哪里学到一身出神入化的武艺。
<br/>　　徐家一夜间死了三十六口，只有徐问天一人逃生。那仇人不要他的命誓不罢休，徐问天一直从交州逃到京州。
<br/>　　这十余年不是他不肯泄露真名，实在是怕。
<br/>　　英雄总是寂寞，也总是难耐寂寞，这十年苦熬着，终于是捱不住。也是觉得锤炼的武艺可以出头了，方才想着借着程大雷的名声往上走。
<br/>　　但程大雷的话也并不是没有道理，想和程大雷动手的人是多的，胜了自然天下扬名，就算输了也不是什么丢脸的事。
<br/>　　这样的机会，他凭什么给你。
<br/>　　徐问天沉吟片刻，道：“程当家若能胜过我，徐某这一世鞍前马后，愿为程当家效力。”
<br/>　　“这个……你也不太够格。”程大雷道：“我手下文有文官，武有武将，你便是自愿投奔我，我也得看你有什么本事。”
<br/>　　徐神机在一旁插口：“咱们是本家，我替你说句话，不如你现在就耍套圈，只要过得去，我就做主留下你。”
<br/>　　程大雷摁住徐神机的手，不要他再说下去，因为徐问天的脸已经越来越难看。
<br/>　　“程当家，如何你才能与我一战，无论如何划下道了，刀山火海，徐某定然做到。”
<br/>　　“嗳，年轻人，你想要出头，也不算什么坏事。但办法得自己想，你老是要我指路，真拿我当仙人指路么？”
<br/>　　徐问天攥紧拳头，青筋暴露，眼见他随时可能发作，程大雷也不敢再刺激他。
<br/>　　“不过也不是没有机会呀？”
<br/>　　徐问天深吸一口气：“程当家请讲。”
<br/>　　“替我杀一个人。”
<br/>　　“谁？”
<br/>　　“百里胜？”
<br/>　　“呃……”
<br/>　　徐问天的智商也有些跟不上，谁不知百里胜正如日中天，是长安城一等一的红人，自己能不能做到还不好说，即便杀了他，也是自绝于天下。
<br/>　　“你瞧瞧。”程大雷抿了口茶：“机会不是不给你，是你自己抓不住啊。”
<br/>　　“不如，程当家换一个人？”徐问天表情也有些尴尬。
<br/>　　“也不是不行。”
<br/>　　徐问天松了口气。
<br/>　　程大雷指了指陈梦：“他。”
<br/>　　陈梦一激灵，睁大眼睛：“干嘛？”
<br/>　　徐问天再次为难：“我只有这一个徒儿，程当家开恩，能否……”
<br/>　　程大雷已懒得再说什么，长身站起，在桌上放下一块碎银。
<br/>　　“路已经给你指了，你走不了就不要怨我。徐英雄，你要知道，想要出头，从来不是一件容易的事啊。”
<br/>　　说罢，程大雷已携着徐神机和刘发财离开，只留下陈梦和徐问天面面相觑。
<br/>　　陈梦忽然觉得脖子后面有些凉意：“师父，你可是不能对我动手啊？”
<br/>　　“咳咳，什么话，为师将一身武艺传授给你，怎么会被他一句话煽动。”
<br/>　　陈梦不自觉拉开与徐问天的距离，知人知面不知心，谁知道你心里怎么想的。能够十余年不露真名实姓，这种事就不是一般心狠做得到的。
<br/>　　徐问天忽地一拍额头，口里骂了一声：“这程大雷果真狡猾，仅凭一句话就挑拨了我们师徒关系。”
<br/>(本章完)
