　　却说老子与多宝道人出了函谷关，继续向西行去，这日，两人来到终南山。
<br/>　　老子在终南山暂住下来，于南山不老松下讲道，在终南山后山清凉山上讲经，楼观台上炼丹，日子过得很是潇洒。
<br/>　　终南山玉柱洞的云中子听闻此事，也经常前来听道，毕竟老子是自己的师叔，自己取听讲也无甚大碍。
<br/>　　老子讲的是小乘佛法，这是老子根据佛教经义而领悟出来的，虽然不能与西方二圣的大乘佛法相比，但毕竟这是圣人领悟而出，对于多宝等人来说，却是受益匪浅了。
<br/>　　“嗡——”
<br/>　　老子讲道，道韵溢出，其余几人皆有些听不懂，多宝道人却是沉迷其中，默默领悟，渐渐在他体内开始有一缕缕金色佛光溢出，脑后一轮金色的神轮缓缓升起，仿若一尊大日金轮。
<br/>　　“此时不悟，更待何时！”不知过了多久，老子突然一声大喝，犹如晨钟暮鼓，在多宝心中响起。
<br/>　　“嗡——”
<br/>　　多宝道人顿时有种醍醐灌顶的感觉，之前不懂之处，瞬间迎刃而解，片刻之后，他身上的金色佛光越来越盛，最终将整片终南山都染成了金色，一轮金色大日横空，笼罩了整片山谷，但却又没有伤害到任何生物。
<br/>　　佛光之中，蕴含着一股普渡众生的意愿，让山谷中众多凶兽缓缓低下头颅，露出虔诚的目光。
<br/>　　刷！
<br/>　　多宝道人陡然睁开双眼，他目中蕴有两道佛光，洞穿周边空间，他身体缓缓升入高空，身下一座金色莲台自然而然地幻化而出，托起他的身体，他一手指天，一手指地，大声喝道：
<br/>　　“天上地下，唯我独尊！”
<br/>　　轰！
<br/>　　多宝道人话音一落，虚空顿时一阵轰鸣声响起，轰鸣的道音如大江翻腾，响彻天际，天地异象显化而出，无尽梵音，禅唱渐渐在周天响起，一道巨大的功德从天而降，多宝道人借此功德，斩出第二尸，成为准圣中期高手！
<br/>　　此刻，多宝道人的一身形象亦是大变，他身上换了一身金色的袈裟，头顶三千青丝尽落，六根清净，脑后金光闪闪，给人一种盛大威严之感。
<br/>　　“阿弥陀佛，贫僧日后不复道人之名，世间只有小乘佛教教主多宝如来！”多宝如来对着老子躬身行了一礼道。
<br/>　　“如此……甚好！你好自为之！”老子点头道。
<br/>　　“贫僧当去西牛贺洲立下道场，传承小乘佛教。望师伯告知老师，弟子不孝！”多宝如来说道。
<br/>　　“你不必多言，为师已然知晓，你去做你的事吧！”此时，通天教主的声音传来，谁也没有发现。
<br/>　　多宝如来朝着碧游宫方向恭敬一拜，便踏空而去，穿越无尽汪洋，来到西牛贺洲，寻得灵山福地，多宝立下小乘佛教道统。
<br/>　　“愿我来世得菩提时，自身光明，炽然照耀无量无数世界，以三十二大丈夫相，八十随形好，庄严其身；令一切有情，如我无异。”
<br/>　　“愿我来世得菩提时，身如琉璃，内外明彻，净无瑕秽；光明广大，功德巍巍，身善安住，焰网庄严过于日月；幽冥众生，悉蒙开晓，随意所趣，作诸事业。”
<br/>　　“愿我来世得菩提时，以无量无边智慧方便，令诸有情皆得无尽所受用物，莫令众生，有所乏少。”
<br/>　　“愿我来世得菩提时……”
<br/>　　“……”
<br/>　　“愿我来世得菩提时，若诸有情贫无衣服，蚊虻寒热，昼夜逼恼；若闻我名，专念受持，如其所好即得种种上妙衣服，亦得一切宝庄严具，华鬘、涂香，鼓乐众伎，随心所玩，皆令满足。”
<br/>　　西方小乘佛教，立！
<br/>　　化胡为佛！
<br/>　　多宝道人化凡之身为胡人，此番化胡为佛，建立小乘佛教，成为小乘佛教佛祖——多宝如来！
<br/>　　此举，在洪荒之中瞬间引起偌大轰动。
<br/>　　三清圣人多少是有些欣慰，因为西方建立小乘佛教，得到天地认可，那么表示灵山大雷音寺的气运又要流出部分，分化到小乘佛教中，因为两者都是西方佛教，却又不为一家。
<br/>　　反观阿弥陀佛与准提佛母两人，两师兄弟大眼瞪小眼，一脸愁苦不已。
<br/>　　想他们师兄弟为了成立西方大乘佛教，谋划了许多，刚刚成立，便遭遇血翅黑蚊夺了三品金莲，之后又有东方佛教的成立，让他们谋划东方气运的计划落得一空，如今又有小乘佛教出现，还建立在自家门前，这简直是挑衅。
<br/>　　两人的命运是如此坎坷，好像上天在作弄他们一般。
<br/>　　两人简直有亲自出手干掉多宝的冲动，但一想到多宝背后的通天，便放弃了这个冲动。
<br/>　　无奈，两人只好将悲愤化为动力，努力修炼新拜老师所传的功法，希望早日实力大进，再去寻仇。
<br/>　　……
<br/>　　南赡部洲之中，有一神山名为王屋山，当年夏禹大帝治水之时所开神山。王屋山中传说住有生灵仙妖，精怪不断，是一座灵脉丰富的神山。
<br/>　　王屋山中有一座山脚灵山，灵山上住着一个神仙，相传此人慈祥庄严、面带微笑，满头白发，身披霞衣，双手持有八卦，活脱脱的一个老奶奶形象。
<br/>　　她便是截教无当圣母化凡之身，圣母化凡，心肠极软，见众生沉沦红尘之中，她不忍心，专渡受苦之人，许多虔诚者都不远万里前来朝拜，她有求必应。
<br/>　　圣母所居住的地方，天下男子皆为兄弟，天下女子皆是姐妹，有点像西方的极乐世界。
<br/>　　这日，王屋山上突然霞光漫天，瑞彩纷飞，神光飞舞，龙凤和鸣，白鹤翱啸，猿猴雀跃……各种惊人异象出现，充满和谐玄幻的意味。
<br/>　　突然，渺渺仙乐响起，光彩将整座大山都笼罩其中，随后，一道宏大的声音传出，传遍方圆亿万里：
<br/>　　“吾乃截教教主座下弟子无当圣母，今日于此立下罗教，以白莲为圣物，以拯救世人于苦海，普渡众生于红尘为大任。罗教，立！”
<br/>　　一道巨大功德从天而降，圣母借此斩出第二尸，善尸化为一慈祥老奶奶形象，正是之前她化凡的形象！
<br/>　　“吾为无生老母！”
<br/>　　“无生老母，真空家乡！”
<br/>　　一声幽幽叹息，传遍周天，令世人震惊。
<br/>　　凡是听到此声音之人，心中烦恼尽去，全身轻松无比，他们纷纷大喜，对于他们来说，无生老母是善良的化身，专门来普渡世人、拯救苍生的。
<br/>　　众生纷纷大喊，声音越来越大，响彻天地：
<br/>　　“无生老母，真空家乡！”
<br/>　　“无生老母，真空家乡！”
<br/>　　“无生老母，真空家乡！”
<br/>　　“无生老母……”
