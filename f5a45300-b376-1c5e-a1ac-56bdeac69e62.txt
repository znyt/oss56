　　在暗古魔尊想要逃的一刻，楚风眠的身形，已经是来到了他的面前，将他直接堵住。
<br/>　　“死！”
<br/>　　楚风眠手中赤炎剑，再次落下，剑锋闪动，直接刺入了暗古魔尊的身躯。
<br/>　　这一次，他的身边，再也没有永恒魔意，帮他抵挡力量了。
<br/>　　一剑刺入到暗古魔尊的身躯之中去，楚风眠的剑中，更是蕴含有着恐怖剑意，吞天神蟒，猛然凝聚出来。
<br/>　　将暗古魔尊身躯中的魂魄，直接一口吞噬干净，让他想要在继续夺舍重生的可能，都一下子打落。
<br/>　　“砰！”
<br/>　　暗古魔尊的身躯，一下子破碎开来，直接化为无数血雾，楚风眠大手一挥，便是全部被他吞噬。
<br/>　　这暗古魔尊的血脉之中，虽然都蕴含魔意，寻常人若是吞噬了，只怕是会直接受到影响，简直是比剧毒还剧毒。
<br/>　　但是在楚风眠的面前，这点小小魔意，便是丝毫没有影响，他的战龙血脉，是一切阴邪之物的克星。
<br/>　　这点小小魔意，被楚风眠镇压在身躯之中，便是几乎一瞬之间，都被全部镇压，毁灭。
<br/>　　剩下的，便都是最为精粹的灵力，全部被楚风眠吞噬殆尽。
<br/>　　“暗古魔尊，死了……？”
<br/>　　“生死境武者，居然死在了御风境武者的手中。”
<br/>　　“我们活下来了……”
<br/>　　那莽通，瞿木，华新三人，看着眼前的这一幕，还都是极为呆滞。
<br/>　　他们都已经是必死的局面，谁能够想到，这暗古魔尊，居然最后是死在了楚风眠的手中。
<br/>　　生死境强者，被御风境武者斩杀，这若是传出去，只怕是人人都会觉得，说出这种话的人，脑子坏了。
<br/>　　整个北域，千年之间，都没有人能够做到这一点，楚风眠居然做到了。
<br/>　　“楚兄，楚前辈，你到底是什么境界？”
<br/>　　莽通看着楚风眠，结结巴巴的问道。
<br/>　　他绝不相信，楚风眠是一尊御风境武者，就连圣者都未成，就能击败生死境强者，这让他们如何相信。
<br/>　　更何况暗古魔尊，更是练就了万古不败真魔体，难缠到极点，同为生死境强者，都不一定可以斩杀。
<br/>　　“我的境界，的确是御风境，不过我有着自己的手段罢了，至少你们在我的面前，不过都是蝼蚁。”
<br/>　　楚风眠看着那三人，冷冷的开口道。
<br/>　　“今日你们看到的事，太多了，我给你们一个机会，敞开心扉，让我在你们的身躯中种下禁制，我可以放你们回去，不然我就毁了你们的记忆，将你们送到其他地方去。”
<br/>　　“这二选一，你们自己选。”
<br/>　　这三人今日目睹了一切，简直是将楚风眠的一切底牌，都给看了下去。
<br/>　　尤其是人皇罐的秘密，极道仙兵，这要是被其他人发现，只怕是会有着源源不断的强者来追杀楚风眠。
<br/>　　不止是北域，整个九域的强者，都会来追杀楚风眠。
<br/>　　毕竟极道仙兵，在九域之中，都是一个传说，七大宗门的人，都会来出手抢夺。
<br/>　　这三人，没有招惹过楚风眠，以楚风眠的性格，不会杀了他们，但是也不会放了他们。
<br/>　　要不就在他们的身躯中，种下诸天生死印，让他们永远无法说出这个秘密。
<br/>　　要不然就是将他们的记忆抹去，将他们带到其他偏远国度，永远不能回来。
<br/>　　这两点，便是任由他们去选。
<br/>　　“这……”
<br/>　　那三人对视一眼，都是还有些目光呆滞，没有从刚刚的震惊中缓和过来，一时间无人开口。
<br/>　　“怎么？你们都不愿意选吗？”
<br/>　　楚风眠冷哼一声开口道，这一次开口，更是蕴含有着一股杀意。
<br/>　　楚风眠不是没给他们机会，要是他们都不愿意选，那便是逼着楚风眠，杀了他们。
<br/>　　起码楚风眠绝不会因小失大，给他自己招惹危险。
<br/>　　“我愿意敞开心扉。”
<br/>　　那莽通，第一个开口道，这杀意落下，令他直接清醒过来，大声喊道。
<br/>　　“少主救了我们一命，我愿意臣服少主，跟随少主这样的强者，才是大机缘。”
<br/>　　“我们也愿意。”
<br/>　　那瞿木，华新，都是很快也跪倒在了楚风眠的面前，纷纷臣服。
<br/>　　楚风眠的实力，他们是有目共睹，足以斩杀暗古魔尊，这等实力，简直已经是超越了北邙学院的内门弟子。
<br/>　　足以跟那些怪物一样的种子弟子媲美，这是他们三人一生都无法达到的。
<br/>　　他们三人的资质，只能算是一般，能够侥幸成为内门弟子，已经是极限。
<br/>　　楚风眠未来的成就，不可限量，跟随楚风眠，也许是他们最好的出路。
<br/>　　“好！”
<br/>　　楚风眠点了点头，在他的指尖，便是三滴金色血液出现，一下子分开，打入到了这三人的身躯之中去。
<br/>　　这金色血液，一下子融入到了他们的血脉中，诸天生死印，也被直接刻入在了他们的内心深处。
<br/>　　如今这秘密，如今就算是死，他们也不可能说的出来。
<br/>　　“好，你们三人既然臣服我了，那我自然不会少了你们的好处，这三件的灵器，从此便是归你们了！”
<br/>　　看着这三人如此识时务，楚风眠大笑一声，大手一挥，三件灵器，便是同时出现。
<br/>　　这三件灵器，都是地级灵器，是楚风眠之前轰杀夺花尊者的手中，得到的。

<br/>　　三人接过灵器，面露喜色，恭敬的说道。
<br/>　　“多谢少主赏赐。”
<br/>　　以他们的地位，这样的地级灵器，便是根本不可能拥有，在北邙学院中，也只有内门弟子，才能保证有着地级灵器。
<br/>　　对他们来说，便是只能用一些好的玄级灵器，楚风眠这赐予，足以让他们的实力，增添数成。
<br/>　　“走，此地不宜久留。”
<br/>　　楚风眠大手一挥，将地上的祭坛碎片，便是全部卷走。
<br/>　　做完这一切后，楚风眠便是直接抓着三人，化为一道遁光，直接飞了出去。
<br/>　　这里可是地魔界，虽然那暗古魔尊的玉符没有发出去，但是这祭坛被毁，很有可能会引来许多地魔门的强者。
<br/>　　一个暗古魔尊，便是如此难缠，地魔门中比起暗古魔尊强大的魔头，比比皆是，楚风眠可不想招惹到危险，一道遁光，直接离开。
