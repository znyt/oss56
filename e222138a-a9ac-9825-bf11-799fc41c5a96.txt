　　“枪是一样的枪，不过刚刚给你的用的是无烟火药！”李承乾的声音解开了马周心中的疑惑，于是拉动枪栓，再次击发，果然还是没有硝烟。
<br/>　　现在马周知道为什么李承乾要淘汰黑火药了，因为那东西实在太差劲了，不是因为其威力小，而是‘副作用’太大。
<br/>　　仅仅一颗子弹就已经有这么重的硝烟，那如果是炮弹的话，估计会让人以为着火了吧！
<br/>　　想到这里，他开始理解李承乾的感受了，只是这份理解还是仅限于武器的更换，而不是将黑火药转为民用。
<br/>　　在淘汰黑火药这一点上，马周依旧持保留态度。
<br/>　　……
<br/>　　靶场上，过足了枪瘾的李承乾带着肩膀已经肿起老高的马周回到了长安，先是命令将作监的火器工坊制造一批威力差一些的黑火药，而后又给刘弘基写了回执，告诉他，矿山使用黑火药可以适当的放开一些口子，不过监管方面却要加强。
<br/>　　正所谓鱼和熊掌不可兼得，有得就必然有失，想要提高矿山的产量，那就只能用火药往里填，谁让大唐到现在还没有实现机械化呢。
<br/>　　而刘弘基在收到来自李承乾的回执之后，心情也是大大的一松，主政一方的他虽然不大管矿山和钢铁厂的事情，但这些地方必竟是在他的辖区，完成不定量他这个行政主管自然要负一些责任。
<br/>　　不过现在好了，看长安那位小爷的意思，似乎黑火药可以敞开的供应，如何能不让老刘喜出望外。
<br/>　　于是乎，接到通知的矿长带来了十辆大车，整整装走了五千斤黑火药，几乎把库存全部搬空。
<br/>　　而接下来的一段时间里，玄菟城煤矿的爆炸声不绝于耳，白天晚上不停的响着，运煤的小车一辆一辆不断的往外推，大量的煤炭被堆放到料场。
<br/>　　最后，一千万斤煤虽然没在规定时间里开采完全，但至少完成了九成，让矿长侥幸险之又险的逃过了被免职的处分。
<br/>　　……
<br/>　　长孙无忌明显没有想到民部对矿产开采计划会逐年往上涨，而且还涨的这么快。
<br/>　　当他知道贞观十六年他承包的矿山竟然需要两千万斤产量的时候，整个人差点没哭出来。
<br/>　　这特么是要人命啊，几年前那么大的矿区计划标准才两百万斤啊，怎么到了现在就翻了这么多翻，如果真的按照这个标准来开采，别说偷偷弄一些出去卖掉，一个不好都不一定能完成任务好么。
<br/>　　带着强烈的不满，老长孙找到了自己的外甥，一把鼻涕一把泪的开始诉苦。
<br/>　　李承乾对这样的老长孙也是没啥好办法，必竟是自己的舅舅，总不能像对一般下属那样。
<br/>　　所以在犹豫了很久之后，李承乾叹了口气说道：“舅舅，开采计划的事情，老实说，我也无能为力，一年两千万斤看似不少，可实际上来说，根本就是入不敷出。”
<br/>　　“那怎么办？高明，你可不能坑舅舅啊。”老长孙似乎有刘备的属性，动不动就抹眼泪，弄的李承乾哭也不是笑也不是。
<br/>　　听了他的话之后只能苦笑着说道：“要不这样好了，我再给您一座铁矿，每年只要您能拿出五百万斤钢材，那两千万的采煤计划就取消了如何？”
<br/>　　“那，那不是一样么？五百万斤钢材，两千万斤煤都不一定够呢。”长孙无忌暗暗计算了一下，总是觉得自己吃亏了。
<br/>　　“那我就也没有办法了，舅舅您是国之宰辅，应该明白外甥我的苦衷，您总不想我给别人都是两千万斤的开采计划，到您这里是一千万斤吧？那样会给长孙家拉去多少仇恨，您想过没有？”
<br/>　　“这……”长孙无忌迟疑了一下。
<br/>　　李承乾说的并没有错，如果别人那是两千万，而他是一千万，那长孙家必然会成为别的眼中钉肉中刺，可能眼下没有什么，但谁也不能保证以后别人不在背后使绊子。
<br/>　　“舅舅，困难只是暂时的，以您的阅历，难道还看不出来大唐正在不断的跳跃前进么？两千万斤的计划或许对您现在来说有些困难，可是您想过没有，其实大唐本身的市场已经够大了，足够长孙家吃上几十年，又何必在乎外面那些零零碎碎。”
<br/>　　李承乾见长孙无忌似乎还有些想不通，索性便把话题挑明了。
<br/>　　毕竟长孙无忌这老头儿现在就是为了钱，大家把话说开了，这老头儿想通了自然也就好了。
<br/>　　而长孙无忌也是，他现在已经是一人之下，万万人之上的存在，权利什么的对他来说已经没什么可图的了，而大唐在对外又没有什么特别难对付的敌人，闲来无事也就只有把目光放到钱上。
<br/>　　所以当李承乾把话说开了之后，这胖老头儿索性也就不再多说什么，只是摆出一副你看着办的表情。
<br/>　　对此，李承乾再次苦笑了一下，然后对长孙无忌说道：“舅舅，钱这东西其实是赚不够的，而且你总要给冲表哥他们留点人生目标不是，如果你把所有事情都安排好了，你让冲表哥他们怎么办？天天混吃等死么？”
<br/>　　“还有就是，那些异族其实也不是什么好东西，您和他们交往过深也没有什么好处，那帮人都是些没有信用的，万一哪一次失信坑您一回，您这半年的努力也就白费了。”
<br/>　　长孙无忌静静听着李承乾在那里自说自话，心里却在想着其他问题。
<br/>　　李承乾已经不是当初那个八岁时，对他言听计从的小孩子了，现在的李承乾手握天下财富，又有军方支持，可以说除了没有登基之外，已经是大唐的第二个皇帝一样的存在。
<br/>　　他对长孙家的看法到底如何，关系到日后长孙家的前途，这让老长孙也不敢过份的撒泼。
<br/>　　但如果说让老长孙就此认命，向这个外甥低头，老实说，他这张老脸还有些挂不住，毕竟再怎么说他也是李承乾的舅舅，算是他的长辈，任由一个小辈来摆布自己，怎么都说不过去。
<br/>　　所以，老长孙陷入了深深的矛盾之中，一方面想要表现自己的特殊地位，另一方面又要考虑长孙家的未来，这让长孙老狐狸一时间很以做出决定。
