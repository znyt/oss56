　　第二天一大早，苏重就被红后轻柔声音唤醒。
<br/>　　“红后，打扰我睡觉你一点儿都不可爱了。”
<br/>　　“抱歉先生。”红后声音里一点儿都没有抱歉的意思，反而有种恶作剧似的得意：“您的朋友正在门外等您，如果您不想见他，我可以让小黑无人机把他轰走。”
<br/>　　苏重无奈起身，在苗家村他认识的人不多。敢大清早来砸他大门的，除了孙老头没第二个。
<br/>　　姜长飞？他没那胆子，他敢来苏重就敢用小黑无人机放到他！
<br/>　　“让那老头进来吧！”苏重有气无力起身。
<br/>　　从回来现实世界苏重就开始忙碌，本来以为今天能睡个懒觉，现在看来又泡汤了。
<br/>　　“小孙，快起来，太阳都晒屁股啦！”苏重正在洗刷，就听到楼下孙建中大呼小叫。
<br/>　　苏重任凭这老头吵闹，仍旧不紧不慢的洗刷整理个人卫生。
<br/>　　下楼后，就看到孙老头正盯着他那一罐子药酒看。
<br/>　　“这个，能给我点儿吗？”孙老头满脸希冀。
<br/>　　“你眼光还真毒。竟然知道这是好东西。”苏重无奈，这老头帮了他不少忙，这点儿药酒虽然珍贵，但给他点儿却毫无问题。
<br/>　　“那咱可说好了！”孙老头喜滋滋。
<br/>　　“你也不用那么兴奋，稍后我给你个方子，你按要求自己配就好。”苏重道。
<br/>　　这药方是他自己琢磨的，能恢复精力延年益寿。以他如今医术造诣，尽管只是寻常药材，但经过精心炮制，依旧有不可思议功效。
<br/>　　“那感情好！”孙老头嘴都快咧到耳根了。
<br/>　　“大清早的，你来找我干什么？”苏重懒得绕弯子，这老头来了准没好事。
<br/>　　“你看你这房子都已经建好了，我想着不是得给你庆祝一下吗。这事你不用操心，我给你张罗，咱也不叫别人，就几个老朋友，怎么样？”
<br/>　　苏重翻了个白眼，这个老头是变着法的想吃他做的菜。
<br/>　　“行，你看着办吧。人可别太多。”苏重也没意见。
<br/>　　上次穿越走的匆忙，房子刚刚建好他就离开。
<br/>　　这次回来，正好赶上父母前来，苏重这些天一直陪着爸妈。
<br/>　　还真没想起乔迁庆祝的事。老头即使不提，过两天他也得办一场。
<br/>　　“包我身上，稍后我就让人给你把食材送上来，咱们也别改日了，就今天晚上怎么样？”
<br/>　　“行吧。”苏重没意见，反正最近他没事。
<br/>　　他特意跟姜长飞打了个电话，通知他过来吃饭。姜长飞替他打理公司，他才能这么悠闲安逸。有这种聚会，苏重不会忘了姜长飞。
<br/>　　下午五点左右人陆陆续续到来。
<br/>　　也真没其他人，孙老头就带了个关胖子，看老头那臭脸就知道，关胖子这是为了一口吃的，直接堵上了自己的职业生涯，死皮赖脸跟来的。
<br/>　　然后就是苗家村老村长。苏重在村里养殖办厂，老村长帮了不少忙。乔迁庆祝，苏重就直接把他给请上了山。
<br/>　　最后就是姜长飞，还有姜长飞带来的一个陌生面孔。
<br/>　　“这是我的秘书纪敏，之后我打算把全部精力用于无人机市场拓展。胖猫食品公司的事情，我想让他去负责，这次正好带过来给您看看。”姜长飞解释道。
<br/>　　“好，我相信你的眼光，就这么办吧。薪资方面也做些的调整，不能让人家只出力不拿钱吧？”
<br/>　　人到齐，苏重让他们参观自家小楼，自己就一头扎进厨房。
<br/>　　很快就整出一桌子香气四溢的晚餐。
<br/>　　几人迫不及待上桌：“小苏快来坐下，菜凉了就不好吃了。”孙老头最着急。
<br/>　　见苏重不入座，反而摆弄胖猫，不由催促道。
<br/>　　孙健中现在也就剩下这几个为数不多的兴趣了，美食占了大头。
<br/>　　苏重不紧不慢，先给胖猫准备好食物然后才上桌。伺候不好猫大爷，苏重可没好日子过。
<br/>　　“大家都是熟人我也就不让了，开吃。”苏重端起酒抿了一口，示意众人随意。
<br/>　　然后场面就一度混乱，所有人都来不及说话，只是一个劲的夹菜往嘴里塞。
<br/>　　纪敏三十岁出头，还很年轻，刚开始还有些放不开。本来还想着谦让一下，保持一下风范，不能第一次在大老板面前吃饭就失态。
<br/>　　可下一刻他就傻眼了。不管是自家上司还是那个来头很大的孙董，全都一副饿死鬼投胎似的，一个比一个抢的快。
<br/>　　他试探性的吃了一块红烧排骨，然后他的味蕾瞬间被攻占，一种发自内心的愉悦让他欲罢不能。
<br/>　　这才是美食啊！
<br/>　　众人只顾着吃饭，一句话也不说，苏重乐呵呵看着，他对此早有预料。
<br/>　　酒足饭饱，一行人坐在小楼前面的院子乘凉。
<br/>　　苏重在小楼前搭了个葡萄架子，架子下面有张石桌，足够众人喝茶休息。
<br/>　　“不错，不错。小苏啊，你这手艺还是没变。”孙建中喝着茶一脸满足。
<br/>　　“你可真是会享受，吃的是最好的。住的地方也舒坦，你说年纪轻轻这，怎么就和个老头似得，这就开始养老？。”孙建中好奇。
<br/>　　他一来小楼，就感觉心平气和，身心舒泰。他知道这肯定又是苏重搞的。
<br/>　　这是一门神奇的手段，他在苏重山下小院里已经见识过。
<br/>　　如果苏重是个老头子，他还不会多么惊奇。但苏重偏偏年轻的过分。但不知道是不是错觉，和苏重交谈，他往往会忽略苏重的年龄。总觉得眼前的人也是个老头子似的。
<br/>　　“老头子可不会给你做饭吃。”苏重抱着胖猫眯着眼睛，享受着山间不时吹来的风。
<br/>　　奇门阵法笼罩之下，整个小楼周围的环境都格外舒适。草木惊奇浓郁，人在其中会心情舒畅格外愉悦。
<br/>　　孙老头撇撇嘴不说话，其他时间他乐意和苏重斗嘴。但关系到美食，他可不敢冒险和苏重争辩。
<br/>　　皱着眉头把茶杯放下，孙老头盯着茶杯不满意道：“我说你这好酒好菜都拿出来了，怎么就舍不得一点儿好茶叶？”
<br/>　　“我这里就只有这种茶叶，想喝好茶，下次请自带。”苏重一点儿都不惯着他。
<br/>　　不过孙老头的话却提醒了苏重，他这茶叶还真不咋地。就是上次大采购时，从超市里买的普通茶叶。
<br/>　　平日里他不怎么喝，没什么感觉，如今刚吃完一桌子美食再喝这个茶，就感觉不太好了。
<br/>　　“下次吧，下次有时间我自己炒点茶。”苏重有了想法。
<br/>　　他记得，之前测试无人机时，曾在后山深处发现过几株老茶树。
<br/>　　有时间采集一些炒制茶叶也是不错的。而且父母喜欢喝茶，可以给老爸老妈送回去一些。
<br/>　　“真的？那炒出茶叶来，一定要卖给我一些。”孙老头眼睛发光。
<br/>　　见识过苏重厨艺，对于苏重炒制的茶叶，他满含期待。
<br/>　　“行，没问题。”苏重眯着眼睛答应，他现在想的不只是茶叶，还有泡茶的水。
<br/>　　现在喝得水是地下井水，经过净化设备之后得到的纯净水，没什么特殊性。苏重想提高一下水的质量。
<br/>　　保护伞身处地下，一切饮用水都经过特殊处理，而恰好苏重就知道设备制造图。
<br/>　　接下来有活干了，他要给他小楼的净化水系统升级，全面提高生活品质！
