有想到，这里竟然有着极品灵石！地球上竟然有极品灵石！还真的让陈默惊讶，地球这种灵气荒漠星球上，竟然有极品灵石！<br/>　　<br/>　　一般，在灵气荒漠的星球上，不会有灵石这种东西，就算有，也是那种下品灵石，而且存量绝对不会多。但是没有想到，竟然在这里发现了灵石，还是极品的！<br/>　　<br/>　　“不对！”陈默细细观察和体会，却发现手中的灵石，已经丧失了大半的灵气！<br/>　　<br/>　　陈默皱着眉头，再次走到一处灵气浓密的地方，再次找到了一颗灵石。不过同样，这颗灵石依然丧失了大半的灵气！<br/>　　<br/>　　就这样，在袭击和驱赶中，陈默找了几处灵气比较浓密的地方，然后都同样的发现了灵石，但是却发现这些极品灵石全部都丧失了大半的灵气！<br/>　　<br/>　　极品灵石丧失了大半的灵气，却依然是极品灵石！不顾陈默现在却绝了将这些灵石带走的心思！<br/>　　<br/>　　因为，他已经明白了，这里有着这么多动植物，其实都是靠着灵石的滋养，才能在地下这种恶劣的环境中，产生如此欣欣向荣的地下生态环境！<br/>　　<br/>　　如果陈默将这些灵石拿走，那么这里过不了多长时间，就会变成死地！灵气滋养着这里的一切，所以这里的生态链才会在如此单一中欣欣向荣。<br/>　　<br/>　　最后，陈默干脆御剑飞行，沿着空间，查看了一下，这里超乎想象的大，高高低低的空间，直接沿着河流有几十公里长，而宽度，也是沿着河流，宽宽窄窄的形成地形，最宽处有着几公里，最窄处也有一公里多！<br/>　　<br/>　　如此一个庞大的空间，有着众多的动植物，真的让人不得不感叹，世界真的很奇妙！<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
