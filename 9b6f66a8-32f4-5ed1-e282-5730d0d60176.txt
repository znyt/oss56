夏夕绾拉开了房间门，门外有人，夏小蝶拽着拳，眼眶通红的怒瞪着她。
<br/>
<br/>        今天这场订婚party是废了，夏振国和李玉兰已经在下面送客善后了，本来一脚已经跨进了苏家大门的夏小蝶瞬间被打回了原型，甚至成为了坊间八卦的笑话，她现在恨不得掐死夏夕绾。
<br/>
<br/>        “夏夕绾，你究竟用了什么办法迷惑住了苏希哥哥，那条短信是不是你发的，你发了什么？”
<br/>
<br/>        夏夕绾勾唇，“这些问题你可以去问你的苏希哥哥。”
<br/>
<br/>        “夏夕绾，你现在开心了，得意了？”
<br/>
<br/>        夏小蝶有一种挫败感，她努力的想打死夏夕绾但不管怎么努力都只能被夏夕绾给打死，这种挫败感让她掉下了愤恨的眼泪。
<br/>
<br/>        夏夕绾看着夏小蝶，“是你们邀请我来参加订婚party的，这份订婚大礼你们可还满意？不要将你一个失败者的眼泪掉给我这个成功者来看，因为我现在真的很开心很得意，你们一再的来找麻烦，甚至将自己的脸送到我手边让我打，如果我不狠狠的扇你们一耳光岂不是对不起你们的一番美意？”
<br/>
<br/>        “你！”夏小蝶感觉自己又被夏夕绾给踩在脚底摩擦再摩擦了。
<br/>
<br/>        这时夏夕绾上前，她停在夏小蝶身畔低低道了一句，“哦，对了，忘了告诉你了，我玩剩下的，都不会给你玩。”
<br/>
<br/>        说完，夏夕绾下了楼。
<br/>
<br/>        夏小蝶就觉得热血往脑袋里一冲，她两眼一闭，真的昏倒了。
<br/>
<br/>        “小蝶！”这时李玉兰跑了过来，一把将夏小蝶给抱住了，“小蝶，你怎么了，快醒醒！夏夕绾，你是觉得破坏别人的订婚充当小三是一件很光荣的事情？”
<br/>
<br/>        这时夏振国也过来了，夏振国跟李玉兰一样都无比希望夏小蝶能够嫁入苏家，现在订婚闹成这样，夏振国已经觉得美梦破碎，颜面扫尽了。
<br/>
<br/>        “夏夕绾，我一直觉得你生长在乡下就是教养差了一些，没想到你的心还这么恶毒，小蝶可是你的亲姐妹啊！”
<br/>
<br/>        夏振国抬起手就想给夏夕绾一巴掌。
<br/>
<br/>        夏夕绾没有回避，她冷冷的看着夏振国，将自己的脸主动的送到了他的掌下，“爸，你打啊，用力的打，既然你这么不喜欢我，那在我妈去世的那一天你就应该打死我的！”
<br/>
<br/>        夏振国的手突然僵在了半空，再也打不下去了。
<br/>
<br/>        夏夕绾扭头，看了李玉兰一眼，“说起当小三这件事，阿姨你可真是我的前辈，只可惜你的女儿并没有深得你真传，空有勾引男人的心，却没有勾引男人的本事。”
<br/>
<br/>        说完，夏夕绾离开。
<br/>
<br/>        这一下李玉兰气的心口都痛了，“振国，叫救护车，快点叫救护车！”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾出了山庄，直接回幽兰苑了。
<br/>
<br/>        她不知道的是，此时路边停着一辆加长版的商务豪车，陆寒霆划下车窗，正看着她的俏影。
<br/>
<br/>        顾夜瑾看着已经赶到的救护车，医护人员抬着担架进进出出的，不禁勾唇，“寒霆，你这个替嫁新娘还真是厉害啊，好好的一场订婚party被她给闹成这样，一人单挑所有人，我看你想驯服她是有点难度的。”
