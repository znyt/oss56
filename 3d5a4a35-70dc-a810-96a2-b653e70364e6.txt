　　第296章
<br/>　　聚义厅大门洞开，从亮处往暗处看，看不清里面情况。像野兽张开的巨口，可以吞没走入的一切。
<br/>　　“谁，谁进去把他们抓出来。”赤檑大喝。
<br/>　　“让俺会会他！”
<br/>　　一举着铁棒的汉子站出来，他身材高大，说话瓮声瓮气，整个人像口水缸。
<br/>　　此人是戎族西颜部的，名叫铁禾，在部落是有名的勇士，以力大著称。便是赤檑遇上他，也不敢说必胜。
<br/>　　铁禾拎着铁棍步入聚义厅，所有人都在关注着他，不得不说，刚才程大雷的表现已经在大家心里建立高手的形象。可铁禾也不是泛泛之辈，他究竟能否擒住程大雷。
<br/>　　咚！
<br/>　　重重的响声，一具身体被丢了出来，大家视线追随着他落地。是铁禾，力大无穷的铁禾，是直接被人用蛮力丢了出来，现在他重重砸在地上，说是没死，但气息奄奄，不知还剩下多少气儿。重伤至此，肯定是救不活了，现在立即死了倒是幸运些。
<br/>　　赤檑折损一员大将。
<br/>　　“你们就不能派出个人么？”赤檑气得大叫。
<br/>　　阴万良脸上神色变幻，却也是没想到程大雷竟然这么棘手。他咬咬牙：“想要除掉他也不难，只要赤檑大人狠得下这条心。”
<br/>　　“狠什么心？”赤檑一头雾水：“我的心还不够狠么？”
<br/>　　“放火。”阴万良道：“一把火烧了聚义厅，只要西颜部肯舍得下那女人。”
<br/>　　“我有什么舍不下的，该舍不下的是你们吧。”赤檑倒抽一口凉气：“都说我们戎族残忍，原来最狠的还是你们帝国这些人呐。”
<br/>　　“放火！”
<br/>　　几十人抱着稻草堵门，聚义厅内瞬间暗了下来。
<br/>　　“他们这是要放火！”樊梨花一惊。
<br/>　　“我看出来啦，不用你提醒。”程大雷心里暗骂：杀人放火，这帮孙子可是够毒的。
<br/>　　“你不用管我，自己逃命去吧，凭你的本事是可以突围出去的。”樊梨花道。
<br/>　　“什么话，一个真正的侠客是不会放下女人自己逃命的。”程大雷又将樊梨花背在肩上，扯了桌布将她与自己缠在一起。
<br/>　　“带着我，你是逃不出去的。”樊梨花。
<br/>　　程大雷当然明白这一点，人力有时而穷，面对百人以上的围攻，靠他一柄剑杀不出去，刚才就险些遇到危险。
<br/>　　“我是问你，这里有没有什么退路，落玉寨这么久，难道就没有什么地道之类的？”程大雷补充一句：“一般都会有的。”
<br/>　　“这个好像没有。”
<br/>　　“这个可以有？”
<br/>　　“这个……真没有。”
<br/>　　此时火舌已经窜了起来，天干物燥，干草见风就燃，火舌在屋内狂舔，座椅墙纸迅速燃烧起来。
<br/>　　程大雷身上负着樊梨花，退，一退再退，四面都燃起火来，整座聚义厅被火海包围。
<br/>　　阴万良看着这一幕，心中松了口气，事情虽然有些意外，但终究是落到自己想要的结局，不过是舍弃一座屋子，日后重建便是。
<br/>　　赤檑却露出可惜的表情，毕竟，这次自己是为首领乌茨木抢亲的，没想到樊梨花却被大火烧死了。
<br/>　　“守住四面，别让他们逃出来。”赤檑道。
<br/>　　“放心啦，火着这么大，除非他们有飞天之能。”阴万良道。
<br/>　　聚义厅是一座单独的木楼，一共二层，孤单单立在山谷中。现在底层已经被火焰包围，火舌往上窜，眼看一座木楼已岌岌可危，随时会倒塌。
<br/>　　程大雷负着樊梨花出现在木楼顶端，烟熏火燎，一张脸快要黑成焦炭。
<br/>　　“他竟然还没死！”赤檑忍不住发出一声惊呼。
<br/>　　“他活不了。”阴万良冷哼一声，大火烧楼，莫说程大雷不敢下来，便是他真敢下来，周围这么多的人，下来后也是死路一条。
<br/>　　程大雷飞身而下，手中剑展开，直扑阴万良而来。在阴万良的眸子里，程大雷的身影变得越来越大，最后堪称庞大。
<br/>　　然后，一剑穿喉，紧跟着，程大雷绕身上的剑，将周围攻击挡开。左突右撞，身周围重重兵，举着各种兵器向程大雷身上砍来。
<br/>　　铛铛铛！
<br/>　　兵器相撞的声音，程大雷手中的剑被砍断，这毕竟是一柄普通的剑，坚韧度不能和匹夫剑相比。程大雷反应很快，劈手夺下一柄铁枪，横着抡开，将周围人击退。
<br/>　　赤檑骑在马上，看着浑身浴血的程大雷拄枪而立，枪身淅沥沥落血。
<br/>　　“好一条杀贼！”赤檑忍不住赞了一声，随即冷哼一声道：“你绝对逃不出去的，放下武器，我给你一个痛快。”
<br/>　　程大雷有几分力疲，如果斧头还在，靠着热血技能，自己还能再撑一段时间，可如今手中握着的，不过是柄普通的铁枪。而对于枪法，程大雷又没学过，无非是仗着力大而已。
<br/>　　赤檑现在倒也不着急杀程大雷，以这杀贼的悍勇，强攻只能多损人命。他不介意一点点耗尽程大雷的体力，然后，一刀致命。
<br/>　　程大雷就像一台自动攻击的机括，立在那里不动，一旦有人走进他的攻击范围，他便立刻出手，砸，抡，挑轻取人命。
<br/>　　很快，程大雷身周就倒下七八具尸体，而程大雷的体力也在一点点流失，他现在已没有突围的力气。
<br/>　　这时候，山谷外突然传来急促的马蹄声，声若擂鼓一般。
<br/>　　“姓樊的，赶紧把我家大当家放了，不然把你山寨上下全部屠净。”一个声音高吼道。
<br/>　　众人循声音看去，猛地吓了一条，只见兵成排，马成行，人人面目狰狞，一脸凶悍。
<br/>　　“怎么，怎么来这么多人？”赤檑倒抽一口凉气，突然在马上大叫：“你们究竟绑了个什么东西？”
<br/>　　看到这一幕，程大雷也松口气，随之怒从心头起，恶向胆边生，破口大骂道：“你们这帮畜生，终于肯来救我了。”
<br/>　　徐神机一伙本想杀进来，抢出程大雷，却是发现这一幕和自己想得不一样。只看到一伙人围攻程大雷，稍微顿了顿，众人大吼一声杀了进来。
<br/>　　铁蹄冲锋，没有人能拦在他们身前，只是一个回合，程大雷面前的人已经被冲散。
