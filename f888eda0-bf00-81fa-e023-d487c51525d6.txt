        &lt;h3&gt;第三卷第一百零七章  准备去洛城试水&lt;/h3&gt;
<br/>
<br/>        傅海生坐在办公室里，眉毛皱成了一个川字。
<br/>
<br/>        他们的信息化项目，因为各种原因，到现在还没有做下来，资金审批到位了，只是找他的人太多了，看着各种复杂的人际关系，他不知道该如何进行抉择。
<br/>
<br/>        就连夏颜都插手了，夏颜提供的公司还是他们的高中同学呼延临庆的公司。
<br/>
<br/>        当初，他和呼延临庆的关系也不错。
<br/>
<br/>        陈子霖他们公司做的方案基本和呼延临庆他们的方案相差不大。
<br/>
<br/>        “傅海生，无论怎么样，我还是你的妻子，我知道之前是我用了小手段，但是我是爱你的，陈子昂不过是你年少时的梦想，人还是应该活在现实中的。
<br/>
<br/>        我和你都不可能和陈子昂成为朋友的，这个道理你懂。如果有一天撕破了脸皮，我们和她都是仇人。傅海生，醒一醒吧，陈子昂是你一生的恶梦。”夏颜尖厉的话响在傅海生的耳边。
<br/>
<br/>        陈子昂是不是傅海生一生的噩梦，傅海生不知道，但是夏颜确实是傅海生一生的噩梦，好又好不了，离又离不开。
<br/>
<br/>        “别想着和我离婚，你和陈子昂没有可能的。”夏颜的笑比冬天的冰还冷。
<br/>
<br/>        他对夏颜无可奈何，他心里藏着怨恨，他将不满发泄在夏颜的、身上，夏颜却搂，着他，笑着说：“海生，你真棒！”
<br/>
<br/>        傅海生想到夏颜看马总的眼神，胃里翻江倒海，冲到卫生间吐了个天翻地覆，从此之后，他就像躲瘟疫一样躲着夏颜。
<br/>
<br/>        夏颜只要一靠近他，他就开始反胃，恶心，呕吐。
<br/>
<br/>        后来，夏颜便也不理他，俩个人都是时令节气的时候回去看看孩子，当着孩子的面装着恩爱的样子。
<br/>
<br/>        孩子在家里有母亲带着，还有高中的同学党桂芳也经常照顾着，当初党桂芳是陈子昂极要好的朋友，他们三个关系一直不错。
<br/>
<br/>        后来他们都在泉城重新买了房，没有想到两家竟然就是门对门，党桂芳担负起了帮他照顾孩子的事情。
<br/>
<br/>        党桂芳也生了一个儿子，两个孩子在同一个学校读书，同一年级不同班，党桂芳没有工作，就一直在家里，将家里打理得很好，周宏生在外面跑生意，他们的小日子能羡慕整个县城的人。
<br/>
<br/>        党桂芳知道傅海生两口子都在外面上班，孩子由奶奶带着，便将俩个孩子都接到自己的家里，吃饭一起吃，写作业一起写，并且也给俩个孩子雇了家教。
<br/>
<br/>        孩子都将党桂芳叫“妈妈”了。
<br/>
<br/>        党桂芳虽然学习不好，但是持家有道，无论是家里还是邻里间都是和睦有加。
<br/>
<br/>        敲门声打断了傅海生的思绪。
<br/>
<br/>        “傅总，信息化项目公告已经挂出来了，挂在省网的官网上。”
<br/>
<br/>        “行，我知道了。”
<br/>
<br/>        “这个项目已经拖了这么长时间了，终于可以实施了。”
<br/>
<br/>        “是啊，可以实施了。”
<br/>
<br/>        汇报工作的人见傅海生没有多，便告辞出来了。
<br/>
<br/>        夏颜之前找过他，给他承诺，傅海生在调走之前是会把他推上去的，只是傅海生说话有些不方便，只能让她出面。
<br/>
<br/>        当他提出陈子昂的公司时，夏颜轻蔑的笑着说：“你傻啊，陈子昂是陈子乔的妹妹，马总肯定是要给陈子乔一份薄面的，为什么吃饭要叫上我，你不明白吗？”
<br/>
<br/>        都是在社会上混的，那能不明白一些事情呢？
<br/>
<br/>        陈子昂他们公司的方案多多少少他也会透露给夏颜同学的。
<br/>
<br/>        而且报名做方案的很多家公司，挤都会将陈子昂他们公司挤走。
<br/>
<br/>        况且，夏颜同学很大方，事情还没有做，就已经帮了他很多，买房帮他，孩子上学帮他。
<br/>
<br/>        本来，他还准备给傅海生说一下情况，但是傅海生没有想问的意思，他就装作不知道吧。
<br/>
<br/>        傅海生拿起电话，给陈子霖打了一个电话，从他们将项目方案写好之后，基本他们就没有来过洛城了，估计发标他们也有可能没有关注。
<br/>
<br/>        还是给他们打一个电话通知一下。
<br/>
<br/>        陈子昂还好吗？
<br/>
<br/>        .........
<br/>
<br/>        陈子霖正在整理资料，江俞轩说十一点多过来交接。他知道江俞轩很忙，交接简单最好。
<br/>
<br/>        电话响起来了，陈子霖看了一下，是傅海生的。
<br/>
<br/>        “傅总，你好。”
<br/>
<br/>        “陈总。”傅海生的声音微微有些尴尬。
<br/>
<br/>        “有什么事情吗？”
<br/>
<br/>        “那个项目昨天发标了，在省网的官网上，你们有时间关注一下。”
<br/>
<br/>        “好的，谢谢傅总。”陈子霖的声音不带一似温度。
<br/>
<br/>        对于这个项目他们原本也没有抱太大的想法。
<br/>
<br/>        陈子霖挂了电话，继续忙他的，也没有安排人去看招标公告。
<br/>
<br/>        工作交接之后，他就回去看看父母，早上三姐和四姐的电话都打通了，说好了周末一起回去。
<br/>
<br/>        大姐也联系好了，就是二哥没有联系上，三姐说二哥由她来联系。
<br/>
<br/>        陈子霖叹了口气，年龄大了，反而兄弟姊妹在一起相聚的日子不多，原来过年过节还能回去，后来都各自有各自的家，像大姐都是当奶奶的人了，不可能再回去过年的了。还有四姐五姐都得回去陪婆家的人。
<br/>
<br/>        子昂太远，原本以为她一直单身，没有想到都有俩个孩子了，这些年真是辛苦子昂了。
<br/>
<br/>        陈子霖正想着家里的事情，感慨着时光的流逝，江俞轩推开门走了进来。
<br/>
<br/>        “俞轩。”
<br/>
<br/>        “都整理好了？”江俞轩说话从来都不拖泥带水。
<br/>
<br/>        “好了。”陈子霖指着一堆堆文件，他已经分门归类，清清楚楚。
<br/>
<br/>        “我已经通知同事们了，等你来了就开个会。现在去会议室吧。”
<br/>
<br/>        “好。”江俞轩对西城的人也都熟悉，但是两个领导工作交接也是必要的程序。
<br/>
<br/>        江俞轩和陈子霖到底达议室的时候，同事们已经在等着他们了，明亮的会议室大屏幕上打出欢迎江俞轩的语录。
<br/>
<br/>        办公桌上摆满水果和茶杯，公司里美丽的小姑娘手捧鲜花。
<br/>
<br/>        江俞轩没有想到他们会以这样的方式来欢迎他。
<br/>
<br/>        原本大家都认识江俞轩，简单的开场白一过，进入正题，半个小时的会议结束后，陈子霖对江俞轩说：“我周末回去看我的父母亲，下周一去北京。洛城的傅海生打电话说那个项目已经挂出来了，在省网的官网上，做不做你来决定。”
<br/>
<br/>        “好，我来安排人。”江俞轩说。
<br/>
<br/>        “我有一次投标的时候碰到夏颜，要不是李凌菲，估计会很困难。”
<br/>
<br/>        “那我就当这次投标试个水吧。”江俞轩微笑着说，原本陈子昂就不太愿意做这个项目，只是江俞轩还想试一试，试一下做人的下限。
<br/>
<br/>        “也行，也许我们两个对换了，会好一些。”陈子霖心里想得还是比较乐观的。
<br/>
<br/>        “不一定，也许和陈子昂有关的事情都不会被她放过。”
<br/>
<br/>        “陈子霖，给你看看这个手镯，你见过没有？”江俞轩突然想起来寒伯安交给他的事情，陈子建有给他说过，但是没有手镯，说什么都是白说。
<br/>
<br/>        “我见过，这不是我们家的手镯吗？这照片你从哪里来的？”陈子霖吃惊的问，他当初并不知道二哥为什么会打二嫂，但是二嫂就是该打。
<br/>
<br/>        因为江俞轩和二哥说话都是头抵头，当时他和三姐夫说话，没有注意到江俞轩和二哥的谈话，只听见二哥问二嫂要手镯。当他到书房的时候，二哥已经打了冯睿。
<br/>
<br/>        “你们家的？”
<br/>
<br/>        “对呀，我原来经常见我父亲拿着，后来不见了，我爸说给我二哥了，我二哥送给你了？不会是误会还没有解开，当你是陈家女婿了吧？”
<br/>
<br/>        “你这脑洞真大，这么珍贵的手镯，你二哥舍得给我？没事了，赶紧忙你的去吧。”江俞轩收起手机。
<br/>
<br/>        这么说，这个手镯是陈家的无疑了？
