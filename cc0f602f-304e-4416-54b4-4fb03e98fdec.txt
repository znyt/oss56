　　一想起之前，自己入魔的情景，叶尘也不由得一阵后怕。
<br/>　　若非在最后生死关头，他恢复了几分理智，然后又一口气吞下了两枚扶摇仙果，压制住了痛苦之源，只怕此时他早已经癫狂而死！
<br/>　　“老匹夫！给我纳命来！”
<br/>　　叶尘一眼便看到，想要逃窜的云千秋，身体微微一晃，便直接冲了过去！
<br/>　　唰！
<br/>　　顺利突破到金丹境之后的叶尘，实力和速度再次暴涨，瞬间便追上了云千秋，连叶尘都微微有些意外。
<br/>　　轰！
<br/>　　直接当头一剑斩落下去，
<br/>　　“不！”
<br/>　　云千秋顿时大骇，一边拼命逃窜，一边祭出各种法宝，想要拦下叶尘的这一剑。
<br/>　　然而，
<br/>　　叶尘所有的紫琼剑，乃是准仙器，而且如今实力大涨，威力自然更加强大，这一剑斩出，势如破竹，瞬间将云千秋的所有法宝，全部击毁！
<br/>　　最终，云千秋一咬牙，又祭出自己的本命飞刀。
<br/>　　铿！
<br/>　　一刀一剑相撞。
<br/>　　这一次，云千秋总算勉强挡住了叶尘攻势，可是整个人却狂喷了一口鲜血，直接倒飞了出去。
<br/>　　而这时，周围其他高手也终于反应了过来，纷纷出手搭救云千秋。
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，等到众人冲至跟前，忽然冲天而起，
<br/>　　“十方无敌！”
<br/>　　唰唰唰！
<br/>　　霎那之间，漫天都是剑影，以叶尘为中心，朝四面八方攻去。
<br/>　　噗噗噗！
<br/>　　周围二十多名元婴高手，顿时死伤一片，
<br/>　　“不好！”
<br/>　　上官云阙顿时大呼一声，同时脸色剧变，
<br/>　　他万万没有想到，叶尘竟然敢在战斗中突破，而且居然还让他成功了，实力比之前竟然又强大了不知道多少。
<br/>　　但更可怕的是，叶尘如今已经不再发疯，恢复了之前的冷静。
<br/>　　他们这些人，还想要用游斗的方式，来对方叶尘，显然是不太可能了。
<br/>　　轰！
<br/>　　“法相天地！”
<br/>　　“琉璃金身！”
<br/>　　叶尘施展出八方无敌之后，身形陡然暴涨，竟然瞬间变成了一个十余米高的居然，身上金光璀璨，手持紫色巨剑，恍若天神！
<br/>　　“虚空斩！”
<br/>　　化身为金色巨人之后的叶尘，抡起紫琼剑，便朝实力偏弱元婴强者冲杀了过去。
<br/>　　柿子自然是先拣软的捏！
<br/>　　轰轰轰！
<br/>　　一眨眼的功夫，叶尘已经又灭掉了七八名，元婴境初期和中期的高手。
<br/>　　到了此时，原本的三十余名元婴强者，此时也只剩下十余人而已。
<br/>　　但是，叶尘似乎没有要停手的意思，用雷霆手段斩杀了那些人，趁着其他人还没有完全反应过来，
<br/>　　第二剑！
<br/>　　第三剑！
<br/>　　第四剑！
<br/>　　……
<br/>　　叶尘每挥出一剑，就至少能带走一名，甚至数名元婴境高手的生命！
<br/>　　一转眼的功夫，仅剩的十余名强者，便只剩下区区五人。
<br/>　　当然，
<br/>　　这五人也三十多名元婴强者中，实力较强的几位。
<br/>　　他们分别是云千秋，和其余四大宗门的宗主。
<br/>　　“叶道友！我太乙门与你并无恩怨，不如我们双方罢手，你看可好？”
<br/>　　太乙门宗主忽然开口高呼道，显然打算求和了。
<br/>　　“我天道宗也愿和叶道友化干戈为玉帛！”
<br/>　　“还有我逍遥派！叶道友，正所谓不打不相识，你和药王宗之间的恩怨，我们就不插手了！”
<br/>　　除了天雷宗宗主上官云阙没有开口之外，其他三大门派的宗主，竟然全部向叶尘求和起来。
<br/>　　“你们……真是无耻之极！”
<br/>　　云千秋顿时气得差点吐血，忍不住痛骂道。
<br/>　　三大宗主顿时全都冷冷发笑，
<br/>　　“云千秋！就是因为你们药王宗，害死我们门中这么多高手，到底是谁无耻啊？”
<br/>　　“不错！云千秋，我劝你还是赶紧自杀谢罪吧！反正你们药王宗已经完了，就不要再连累我们了！”
<br/>　　三大宗主开始纷纷出言嘲讽起来。
<br/>　　然而，
<br/>　　叶尘却仿佛没有听到三人的话一般，已经直接抡起紫琼剑，朝云千秋冲了过去。
<br/>　　“逃！”
<br/>　　面对如同恶魔般的叶尘，而且其他三大宗主，也显然已经放弃了自己，云千秋哪里还有继续抵抗的勇气，脑袋里只剩下一道念头，那就是赶紧逃命。
<br/>　　可惜，他之前早已经受了不轻的伤，而且药王宗的功法，本来就不以速度见长，如何能逃的过叶尘的追杀？
<br/>　　几乎转眼之间，叶尘便已经冲到跟前，直接一剑斩落下去！
<br/>　　“不！”
<br/>　　云千秋顿时大吼了一声，面对后方那恐怖的紫色剑芒，只好再次祭出本命飞刀，抵挡了上去。
<br/>　　然而，
<br/>　　轰隆！
<br/>　　刀剑相撞之下，没有任何悬念，云千秋的本命飞剑瞬间被打飞了出去，而他本人则狂喷了一大口鲜血，狠狠砸落在下方地面上。
<br/>　　彭！
<br/>　　整个地面上，顿时出现了一个，直径长达近百米的巨坑！
<br/>　　云千秋纵然功力深厚，经过几次重创之后，身体已经严重受损，脸色惨白无比，气息奄奄，连爬起来的力气都没有了。
<br/>　　而叶尘身体一晃，已经来到云千秋的跟前，挑起一只脚，踏在了云千秋的胸膛之上，缓缓道：
<br/>　　“老匹夫，你可还有什么遗言？”
<br/>　　云千秋脸色一阵变幻之后，忽然放生狂笑起来。
<br/>　　“叶狂仙！今日是我药王宗败了，不过你若是觉得，我们药王宗仅仅只是这种程度，那你就大错特错了！”
<br/>　　“你听说修真界吗？你知道通天路吗？我药王宗自建宗两千年余年，已经有十余位先祖，踏入通天路，进入更高的位面，而且他们通过秘术，一直和宗门保持着联系！”
<br/>　　“你今日毁我宗门，他日我药王宗先祖得知，返回地球，便是你偿命之时！”
<br/>　　“哈哈哈！”
<br/>　　听到云千秋这番话，叶尘顿时愣了好一会，
<br/>　　他万万没有想到，这云千秋竟然也知道修真界的存在，甚至药王宗的先祖，都已经踏入了修真界！
<br/>　　云千秋看到叶尘犹豫，还以为他是怕了，顿时越发得意，两眼一转道：
<br/>　　“小子！不如我们来做个交易，只要你不杀我，我不但会向先祖们解释此事，而且我还可以告诉你通天路在哪，如何？”
<br/>　　云千秋很自信，
<br/>　　在他想来，但凡是修真之人，没有人会不想知道通天路所在，但是除了五大宗门的太上长老，其他人并不知晓，即便是五大宗门的宗主也未必清楚。
<br/>　　所以叶尘不可能不动心。
