        所以，当电梯门开了后，落以柔看到一个身着西装革履的男人正埋头看着自己手里的文件，并未察觉到有所诧异，便自顾自地一瘸一拐地直接走了进去。
<br/>
<br/>        这男人穿着浅灰色的高级西装，高大健硕的身材隐隐透着王者的风范，手下每翻一页文件的动作，就像舞刀弄枪一样，将他的霸气充分地表现了出来。
<br/>
<br/>        因为他低着头，落以柔看不全他的样貌，但他整体给她的感觉就是一种直来直往的帅气，高贵的气质与那天在“夜惑”里救了她的那个男人很像。
<br/>
<br/>        电梯的按键，只有顶层三十八层的那个按钮是亮着的，而落以柔恰巧要去的是三十七层，可她怎么按按钮，按钮都按不动。
<br/>
<br/>        而电梯已经关门，正往上运行。
<br/>
<br/>        这让落以柔无比尴尬地站在一旁，不敢贸然打扰身旁这个高贵的男人。
<br/>
<br/>        大不了，她跟着他去三十八层，然后再走安全楼梯去三十七层便是。
<br/>
<br/>        其实，这次应聘，她并不是冲着什么傅氏集团的帅总裁去的，而是冲着他们所开的薪水去的。
<br/>
<br/>        再说了，网上的应聘信息上写明的是总裁助理秘书。
<br/>
<br/>        落以柔所理解的应该是总裁的助理的秘书，可就不知道为什么其他那群女人会理解为总裁的助理秘书。
<br/>
<br/>        助理和秘书，难道不是两个不同的职位吗？
<br/>
<br/>        “当——”的一声，电梯抵达顶层。
<br/>
<br/>        落以柔疾步先行，哪知道脚下“咔”的一声，高跟鞋的鞋跟断了，她重心不稳，双手呈划船状。
<br/>
<br/>        眼瞧着她就要摔个人仰马翻，忽然一双厚实黝黑的大手从她后身钳住她的纤腰，一道强劲有力的力度，带动着她一起走出了电梯。
<br/>
<br/>        落以柔下意识地抬起头来，刚巧撞入了这个伸手扶住她的男人的视野。
<br/>
<br/>        “是你？！”落以柔惊讶中又有一丝欣喜，“莫非，你也是来应聘秘书的吗？但是，我看应聘广告上说，只招女性。”
<br/>
<br/>        傅夜沉扶着这女人站稳身后便收回了手，听着这女人的话，顿时黑脸了。
<br/>
<br/>        他难道就长得像是来应聘当秘书的人？！
<br/>
<br/>        更何况，他坐在的还是总裁专属电梯。
<br/>
<br/>        落以柔又想起了什么，连忙颔首道谢：“上次，谢谢你救了我。”
<br/>
<br/>        救过她？！他什么时候救过她？
<br/>
<br/>        傅夜沉对着女人几乎是完全没啥印象了，也不打算跟她废话，而是直接转过身去，准备回自己的总裁办。
<br/>
<br/>        但是，就在这个时候，身后响起的声音，又拉回了他脑海里所有的记忆片段。
<br/>
<br/>        “诶？大哥，你走错方向了！安全楼梯在那边，我们应聘的地方是在三十七楼的助理办公室。”落以柔好心提醒道。
<br/>
<br/>        傅夜沉听着这一声“大哥”，才回想起那晚在“夜惑”里救下的那个女孩。
<br/>
<br/>        原来是她……
<br/>
<br/>        顷刻间，傅夜沉在这个女孩身上恍若看到了童筱颖的影子。
<br/>
<br/>        “大少爷，早上好。”助理郑忠威远远地瞧见电梯前的人影，恭敬地向前展露着问候的笑靥，当他看到傅夜沉身边的落以柔时，顿了顿后，又纳闷地问，“这位是？”
<br/>
<br/>        “你的秘书。”傅夜沉淡淡地应道，低沉的嗓音从彼端传来，隐含绝对的命令。
<br/>
<br/>        郑忠威顿时间就傻眼了，这次他是特招女秘书替他分担女性比较好做的那一部分工作，谁知道会有那么多女人来应聘。
<br/>
<br/>        他好不容易从两千多名女人里筛选得只剩下十五名，这还未正式开始第二次应聘，他家大少爷就这样给他定下来了，未免也太潦草了吧！
<br/>
<br/>        郑忠威心里虽然有疑义，但只要是傅夜沉的话，他还是绝对会服从。
<br/>
<br/>        傅夜沉说完，便只身去了自己的总裁办。
<br/>
<br/>        郑忠威这才看着落以柔问道：“请问，怎么称呼？”
<br/>
<br/>        “我叫落以柔，是来应聘助理秘书的，已经投过简历过了初试！”落以柔从不知所以然地状态中回过神来，微笑着对郑忠威说道。
<br/>
<br/>        “落小姐，请跟我来。”郑忠威转身往办公室走，态度斯文有礼且谦和，唇边挂着一抹浅笑。
<br/>
<br/>        看样子，他家大少爷对这个落小姐是有意思喽？！
<br/>
<br/>        虽然是傅大少爷开了金口，走后门进来的，但是应聘流程还是要走一遍，怎么说，也得达到符合他们的条件才行。
<br/>
<br/>        “落小姐，请进，坐在那里。”郑忠威将落以柔带到自己的办公室后，抬起手来指着前方的椅子，瞧她移动后，便转向自己的位置坐好。
<br/>
<br/>        随后，郑忠威打开了监控视频直播，将自己应聘这女孩的全过程，传输到了总裁办里的傅夜沉的电脑上。
