　　这金色灵剑出手的时刻，他的力量，甚至是要超越那四尊创始者的力量之合。
<br/>　　一剑横扫过去，所向睥睨，没有任何人，足以跟其抗衡。
<br/>　　夜皇，阳皇，风皇，武皇四人，都在这剑芒之下，被纷纷逼退，反震的力量，都让他们的胸口猛然一颤，吐出了一大口的精血。
<br/>　　整个北邙宫之中，如今便是都响起了一声声的剑声。
<br/>　　就连其中没有被波及到了的老祖，都是不少人感觉到天旋地转，脸色恐惧到极点，这样的力量，稍稍波及到，都足以是将他们，彻底毁灭。
<br/>　　剑皇殿，可是当初剑道之主的宫殿，剑道之主的剑意，随着时间都流入了其中，更何况其中还有着像是楚风眠，以及还有他的无数师兄，师姐，都是从这剑皇殿中领悟剑术。
<br/>　　这剑术的领悟，都被剑皇殿看在眼中，可以说他虽然是一位器灵，但是更是一位剑道高手，当世之中，能够与之抗衡的剑道高手，都是寥寥无几。
<br/>　　他一出手，便是显现出了强大的实力。
<br/>　　就连夜皇等人，一个个脸色都是极为震惊，心有余博，他们也没有想到，这一直以来他们所熟悉的北邙宫，居然都有着如此可怕的力量。
<br/>　　而且这北邙宫，居然是出手，来阻止他们，轰杀楚风眠。
<br/>　　刚刚那一剑，仅仅只差一点，就足以是将他们彻底毁灭，遭遇了这等灭顶之灾，如今重伤，已经算是极为幸运的了。
<br/>　　“云剑。”
<br/>　　楚风眠看到这剑皇殿出手，也是不由的轻声呼唤了一声。
<br/>　　对他来说，这也是他，第一次看见这剑皇殿出手，万年之前剑道之主被围攻的时候，楚风眠被三师姐带着一路逃走，对于当初剑皇殿发生的事，他一概不知。
<br/>　　所以也不曾记得，这剑皇殿，居然是被大师兄带走，带到了这北邙学院的事。
<br/>　　“少主。”
<br/>　　楚风眠脑海中，剑皇殿稚嫩的声音再次响起，只是这一次他开口的声音，却是显得有些虚弱。
<br/>　　“我的灵力不够，不足以灭杀这四人。”
<br/>　　剑皇殿的实力再强，却只是一个器灵，就跟那云羽天宫之中的云老一样，有着限制。
<br/>　　云老只能够是在云羽天宫之中行动，无法离开，而这剑皇殿，虽然实力极为强大，却是没有足够的灵力足以支撑他的器灵行动。
<br/>　　“无妨，我先给你一些灵石，让你恢复灵力！”
<br/>　　楚风眠听到剑皇殿的话，直接开口道，他大手一挥，空戒突然打开，从中一堆堆的灵石，都是飞了出来。
<br/>　　这些灵石，全部都是皇品灵石，足足是一座山峰高的皇品灵石，有着数千万枚，全部飞了出去，如同一座高山。
<br/>　　“这是！皇品灵石！少主你是从哪里得到的如此之多的皇品灵石！”
<br/>　　就连剑皇殿，看到如今楚风眠的大手笔，都是吓了一跳。
<br/>　　就算是在万年之前，现在积累如此可怕的财富，都是一件极为困难的事情，楚风眠的境界他也看得出来，仅仅只有生死境的地步，所以剑皇殿也想不到，楚风眠居然会拥有如此之多的财富。
<br/>　　“好！好！在这鬼地方，我一直只能是从虚空中摄取灵力，万年时间才积攒了那么一点，现在有了少主的灵石，足以让我的实力恢复两三成了！”
<br/>　　剑皇殿惊讶之后，便是大喜道，开始调动起力量，将这些灵石全部吞噬。
<br/>　　如今楚风眠正是在这北邙宫，也就是这也剑皇殿之中，他打出的灵石，都在瞬间被吞噬，顿时这剑皇殿的力量，都在这瞬间，开始疯狂的攀升起来。
<br/>　　整个剑皇殿的力量，便是比起之前，已经是开始截然不同的变化起来，上面的纹路，雕刻的符文，都开始显现出来，凌冽的剑意，也都是爆发了出来。
<br/>　　之前楚风眠刚才进入其中的时候，他没有感觉出这北邙宫，正是剑皇殿，也是因为这剑皇殿当时的力量，实在是太虚弱了，一点点的剑意都感觉不到。
<br/>　　所以就算是楚风眠都没有第一时间认了出来，还是一直等到夜皇催动剑皇殿的力量，这才让楚风眠认了出来。
<br/>　　作为曾经剑道之主的宫殿，这剑皇殿的强大，显然远远不止如此。
<br/>　　“少主，太好了，万年了，没想到万年了，居然还能见到您！”
<br/>　　剑皇殿一边吞噬这无数灵石，一边他欣喜的在楚风眠的脑海之中响起。
<br/>　　对他来说，这也是万年一来，第一次见到故人，曾经剑道之主的无数弟子，全部陨落，大师兄建立这北邙学院后不久，也便是陨落了。

<br/>　　这剑皇殿，从万年之前剑道之主后，便是再无主人，这一次便是再度认主。
