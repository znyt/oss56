　　这八十一道无上剑气，所凝聚出的七彩剑光从天降落下来。

<br/>　　楚风眠说到做到，绝不会让任何一人离开。
<br/>　　“咔嚓！”
<br/>　　七彩剑光落下，鹿战神身上的圣王真身，都层层破裂开来，在剑锋之下，被全部毁灭。

<br/>　　“无论如何挣扎，都将没有任何意义！死！”
<br/>　　楚风眠心中杀机浓郁，在他的背后，远古杀神虚影都是凝聚出来，一股股的杀意，全部笼罩在七彩剑光之上。
<br/>　　令这七彩剑光，都是笼罩上了几分血红。
<br/>　　楚风眠的必杀之心，绝不会动摇，鹿战神，必死！
<br/>　　七彩剑光威力暴涨，那鹿战神身上，刚刚凝聚起来的气势，都在一瞬之间，被彻底打破。
<br/>　　“不！”
<br/>　　鹿战神大声嘶吼这，却已经无济于事。
<br/>　　七彩剑光，趁机落下，鹿战神的身躯，全部被泯灭，如今就连一点的灰烬，都无法找到。
<br/>　　只剩下这一身精血，已经是随着剑光，全部落入到了楚风眠的身躯之中。
<br/>　　这鹿战神的精血融入到楚风眠的身躯之中，楚风眠便是感觉到了一种滂湃到极点的力量，炙热无比。
<br/>　　鹿战神虽以死，但是他的境界，可是比起大部分武者的，还要炙热的多，象征着他不败的武意。
<br/>　　不管是剑祖荒，还是鹿战神，放在九域之中都是数十年难遇的天才。
<br/>　　若是没有碰到楚风眠的话，他们二人将来超越生死，都是轻轻松松的事情。
<br/>　　可现在，他们招惹到了楚风眠的头上，便是他们最愚蠢的举动，跟楚风眠为敌之人，还没有谁可以活着离开。
<br/>　　“好，在这鹿战神的空戒之中，居然有着如此之多的诸天奇物！”
<br/>　　楚风眠打开鹿战神的空戒，便是被一股股精粹极致的灵力所包围，在鹿战神的空戒之中，诸天奇物，不知道有多少。
<br/>　　其中的无数诸天奇物，都是已经在九域之中消失了数万年的，就连楚风眠前世都没有见到过的传奇之物，现在都静静的躺在鹿战神的空戒之中。
<br/>　　单单远古灵器，在这鹿战神的空戒中，都足足有着上百件，甚至是都还有着两件天级灵器。
<br/>　　鹿战神身为圣王宗弟子，修炼身躯，不用灵器，这些灵器倒是都保存了下来，现在全部都便宜了楚风眠。

<br/>　　楚风眠长啸一声，黑玄剑阵突然出现，楚风眠大手一挥，在空戒中的所有灵器，都被他全部扔向了黑玄剑阵之中。
<br/>　　天级灵器，对楚风眠来说，除非是祖龙至尊剑，战龙宝车这个级别，不然其他都对楚风眠没有太大用处。
<br/>　　倒不如全部用来培养这黑玄剑阵。
<br/>　　将这黑玄剑阵培养到最后，他的威力，绝不会亚于祖龙至尊剑，战龙宝车，甚至都可能还要更强。

<br/>　　甚至是在其中，楚风眠都发现了一种灵力比起皇品灵石，更要精粹的灵石，圣品灵石。
<br/>　　在如今九域的无数古洞府中，都难得一见的存在。
