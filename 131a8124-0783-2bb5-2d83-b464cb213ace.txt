　　翌日清晨，萧家营地内，所有族人都停下了手头工作，脸色中尽是忧愁与叹息。
<br/>　　他们的目光，全都定格在八十多个年轻子弟身上。这些子弟才来了十多天，还未见历练效果，可如今却要离开了。
<br/>　　延误了历练时间还是其次，最重要的是属于年轻人的锐气，必然会消退许多。
<br/>　　这一次，萧家的子弟，被慕容家子弟打压得这么惨，如今又是灰溜溜地离开，这绝对是年轻一辈的耻辱。
<br/>　　不难想象，多年后，萧家子弟的整体水平必然会远逊色于慕容家。
<br/>　　也难怪族人们一副忧心忡忡的样子。
<br/>　　当然，那些年轻子弟们也很不好过，无不羞愧地低下了平日傲气的头颅，再无以往的自信与少年锐志。
<br/>　　可是，也没有任何办法。他们一个个都身负伤势，不少人还包着绷带，他们需要更好的疗养伤势，只能回家族。
<br/>　　大半个时辰后，所有子弟集结完毕，萧虎看了眼四长老。
<br/>　　四长老摇了摇头。
<br/>　　萧虎叹了口气，对家族子弟们道，“我们走吧。”
<br/>　　八十多个年轻子弟垂头丧气，不甘地准备离开。
<br/>　　恰在此时，一个少年出现在他们面前，并拦住了他们的去路。
<br/>　　“萧逸。”年轻子弟们吃了一惊。
<br/>　　“萧逸。”萧虎则是脸色一喜。
<br/>　　来人正是萧逸。
<br/>　　“萧逸，你成功了？真的炼制了足够的丹药？”四长老快步走了过来，面露希冀地问道。
<br/>　　一晚上一百人份的疗伤人、淬体丹，这是一个一品炼药师绝对无法做到的事；二品炼药师也极为困难；只有三品炼药师才能有十足的把握和速度。
<br/>　　萧逸翻了个白眼，道，“你说呢，要是没成功，我出现在这里干嘛？”
<br/>　　“好小子，果然没有让我们失望。”四长老重重地拍了拍萧逸的肩膀，满是赞赏。
<br/>　　“这个变态，竟然真的成功了，不愧是我们萧家的第一天才。”六长老自豪一笑。
<br/>　　“但，我不准备给他们丹药。”萧逸摇了摇头，脸色冷酷。
<br/>　　“阿？”萧虎愣了愣，原本满脸的笑容忽然一僵。
<br/>　　萧逸冷笑道，“既然矿脉里的疗伤药本来就不够了，倒不如留着给萧壮、萧子木这留下来的二十多个子弟比较好。”
<br/>　　“我辛辛苦苦炼制出来的丹药，可不想给一些无用的人浪费掉。”萧逸耸耸肩，毫不在意地说道。
<br/>　　他这毫不在意的语气，听在那些本就伤重的子弟耳中，简直是最大的蔑视和侮辱。无用的人浪费掉，指的不就是他们吗。
<br/>　　“萧逸，你在故意捉弄我们？”一个年轻子弟出言怒喝道。
<br/>　　萧逸认得这个子弟，正是之前有份进入紫云洞修炼的萧家最出色的那批子弟。
<br/>　　“萧逸，不要胡闹。”四长老沉声说道，“既然丹药炼好了，赶紧拿出来给大家服用。”
<br/>　　“不。”萧逸肯定地说道，“我炼的丹药，一粒也不会给他们。”
<br/>　　“臭小子，你找打。”六长老立刻来气，道，“我知道这些子弟中有不少子弟都欺负过你，你现在想报复，但大家同为萧家族人，何必如此计较。”
<br/>　　所有人都已经看出了萧逸的心思，摆明了就是在为难这些伤重的子弟。
<br/>　　“哼，不给就不给，大不了我们回家族。”那些傲气的子弟满脸愤怒，转身就走，“萧逸，你给我们记住了，今日之辱，他日我们必定百倍奉还。”
<br/>　　萧逸明显引起了众怒。
<br/>　　“臭小子。”六长老立刻道，“我以长老的身份命令你，立刻把丹药拿出来。”
<br/>　　萧逸耸耸肩，道，“不好意思，我是少家主，身份与你齐平，你还命令不了我。”
<br/>　　“你..你...”六长老气急败坏，指着萧逸，气得直哆嗦。
<br/>　　萧逸冷笑一声，道，“萧壮、萧子木他们，与我并肩作战，共同对付慕容家子弟。他们受伤的话，我为他们炼药，理所应当，但那群没用的，即将离开的废物，我何必浪费丹药。”
<br/>　　“至于你们。”萧逸看向了在谩骂着他的子弟，冷冷道，“他日百倍奉还与我？呵呵，你们觉得自己有那个本事？”
<br/>　　“你们被慕容家的子弟打得落花流水，节节败退。而我昨日，凭一己之力就把他们全绑了，一百多号慕容家子弟，在我面前大气都不敢喘一句。你们有什么资格骂我？”
<br/>　　萧逸自傲地说着。
<br/>　　八十多个年轻子弟闻言，纷纷停止了谩骂，露出了羞愧难当的表情，甚至不敢直视萧逸的目光。
<br/>　　“萧逸，够了，出尽风头就算了，做人不要太过分。”萧虎皱起了眉头。
<br/>　　一旁的四长老忽然拉了拉萧虎，示意他不要出言。
<br/>　　萧逸继续道，“看看你们自己，一个个垂头丧气的，还有脸说日后找我算账？”
<br/>　　“你们有那份傲气日后报复我，就没有勇气今日就去找慕容家子弟报仇？”
<br/>　　“你们有那份胆量骂我，难道不想将慕容家子弟踩在脚下好好出一口恶气？”
<br/>　　一个子弟忽然抬起了头，大声道，“谁说我们没有勇气，谁说我们不想出恶气。萧逸，今日老子就不走了，就算你不给我丹药，就算我重伤至死，老子也不受那份憋屈气。”
<br/>　　“我也不走，不揍死慕容家那群龟孙子，老子不服。”
<br/>　　“我也不走，萧逸，我跟你混了，你说什么时候弄慕容家那群混蛋，老子第一个上，绝无二话。”
<br/>　　“我也不走...”
<br/>　　“....”
<br/>　　一个个子弟接二连三地抬起头，脸上满是战意，那份属于年轻人的锐气和傲气，表现得淋漓尽致。
<br/>　　“好，有种。”萧逸大喝一声。
<br/>　　“肯留下来继续跟慕容家那群小人拼的，值得老子为你们炼药。”萧逸大笑一声。
<br/>　　“萧虎执事，分发丹药的事就交给你了。”萧逸拿出乾坤袋，递给萧虎。
<br/>　　“额...哦，好的。”萧虎愣了愣，反应了过来。
<br/>　　一旁的四长老笑了起来，“萧逸这小子。”
<br/>　　“原来如此。”六长老也明悟过来，道，“我就知道，萧逸这小子辛辛苦苦炼了一晚上的丹药，现在才说不给，这不多此一举嘛，原来是想让他们恢复斗志。”
<br/>　　四长老点点头，道，“你知道？你知道个屁，刚才你还一直在骂萧逸。”
<br/>　　“如果萧逸就这样把丹药给他们，就算他们恢复了伤势，若毫无斗志，倒不如不要留在这里历练。否则，日后遇到慕容家的子弟还是会被打得落花流水。”
<br/>　　六长老道，“本来吧，这群子弟就算回家族疗伤，耽误一些历练时间也问题不大。”
<br/>　　“麻烦就麻烦在他们锐气受挫，心里始终有着一根刺，对日后的成长和修炼都会有很大的影响。现在倒是给萧逸解决了这个麻烦。”
<br/>　　四长老笑了笑，“萧逸这次处理得很好，我现在终于有些明白了，为何大长老、二长老还有三长老都力保他的少家主之位。”
<br/>　　“萧家日后在此子的带领下，我相信我们老一辈的家伙不用再去担心了。”
<br/>　　另一边，萧虎一边派发丹药，一边听着子弟们的由衷崇拜之词。
<br/>　　“真不愧是少家主，他的炼药本事连家族里那些老家伙都要自愧不如了吧。”
<br/>　　“可不是嘛，我们萧家的第一天才，岂是浪得虚名。”
<br/>　　“按我说，少家主昨日生擒了慕容家那群杂碎，那才叫霸气。”
<br/>　　“....”
<br/>　　萧虎笑了笑，瞥了萧逸一眼，他知道，眼中的这个少年，日后在家族中的威信，将不亚于任何一个长老，起码，所有年轻一辈都会百分百信服这个少年。
<br/>　　萧逸也听到了这些话语，但没有放在心上，转身便欲离开，一晚上高强度地炼制这么多丹药，还是非常耗费心神的，他打算回房里休息。
<br/>　　无论萧家子弟怎么看他，仇视也好，崇拜也好，他无所谓。他之所以愿意辛苦炼丹，愿意今天演这一出戏，仅仅是为了三长老。
<br/>　　他不想看到三长老也为这些子弟而烦忧。
<br/>　　不过说实话，他发现，原来听着别人由衷的崇拜，是一件很不错的事。
<br/>　　或许，原本独来独往惯的他，已经开始愿意接受这个家族，接受自己是这个家族一份子的事吧。
<br/>　　当然，这归功于三长老与依依。
<br/>　　正当萧逸准备离开时，一位族人忽然快速来到四长老面前，道，“四长老，属下有要事禀报。”
<br/>(本章完)
