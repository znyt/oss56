　　“他们到了。”帕里斯通对身旁比杨德道。
<br/>　　两人站在一间大型监控室内，墙壁上挂着一面巨大的显示屏。房间内放着五排圆弧状长桌，桌子上摆满各种仪器电脑，近百人正在紧张忙碌。
<br/>　　透过显示屏，能够清晰看到整个东拓国皇宫。
<br/>　　帕里斯通是整个嵌合蚁计划的参与者，并且是主要策划人。稍微费了些功夫，最后的收尾工作，顺利的落在他手中。
<br/>　　“蔷薇准备好了吗？”
<br/>　　“已经就位。”一名秃头中年人连忙回答，他满头大汗，一脸的紧张。
<br/>　　“吝啬蔷薇是大威力武器，准备一百多组，是不是太多了。”他小心翼翼道，一边说一边狂擦额头汗珠。这些脏弹放在一起，能轰平整个东拓国！
<br/>　　比杨德眼睛眯起，他都不知道，帕里斯通竟准备了那么多炸弹！
<br/>　　“不多，以防万一而已。嵌合蚁的危害太大，我们要做好万全准备。”帕里斯通瞥了一眼比杨德，淡然解释道。
<br/>　　“让我们的人即刻出发，近距离隐蔽起来，等待下一步指示。”帕里斯通对一名联络员吩咐道。
<br/>　　“是！”
<br/>　　“好戏即将开场，敬请期待吧。”帕里斯通带着耳机，一手插在裤兜里，脸上带着完美的笑容。
<br/>　　“希望能如你所料，一切顺利。”比杨德盯着帕里斯通看了数秒，最终淡淡道。
<br/>　　……
<br/>　　“小蚂蚁，想和我斗，你还差了点。”苏重冷然一笑，右手狠狠压下。
<br/>　　五张巨大现网，带着尖锐鸣叫，陨石般冲向东拓国皇宫。
<br/>　　嗖！
<br/>　　突然之间，一道人影带着残像出现在半空，看着头顶砸下来的线网。抬手一拳，随意至极的打在线网上。
<br/>　　嗡！砰！
<br/>　　几乎瞬间，线网剧烈震颤，只是僵持了三息。整个线网就崩溃成点点黑红荧光。
<br/>　　蚁王！果然很强！
<br/>　　尼特罗眼睛眯起，他第一次见到蚁王梅路艾姆。尽管早对蚁王实力有所估计，但这么轻松打碎线网……嵌合蚁，真是个危险种族！
<br/>　　这才几个月，竟凭空造就出这么一个，甚至能横扫人类社会的怪物！
<br/>　　还好。尼特罗看向不远处苏重。人类有着各种不堪。但也出现很多闪耀人物。眼前这位的崛起历程，精彩程度，可一点儿都不比蚁王差！
<br/>　　苏重眼睛一亮，那么小的一拳，竟能同时打破五张线网。由点击面，这是极其高端的劲力技巧！
<br/>　　上次碰到蚁王，对方还只是个一身蛮力的野兽。现在竟懂得如此高明技巧，实力必然大进！
<br/>　　他不惊反喜，压力越大，他越高兴。只有在生死危机下，才能迫使魔龙意志顺利融合。
<br/>　　“走吧，你不希望在这里打吧。”蚁王梅路艾姆，瞥了一眼下方。皇宫前面巨大空地上，那里正停留着数万人类。是从东拓国各地聚集来的人。
<br/>　　这些人全都被催眠，好似呆傻一般，静静的站成一个巨大方阵。
<br/>　　一旦两人肆无忌惮的宣泄力量，这些人必死无疑！
<br/>　　苏重嘿然一笑：“我看是你不希望在这里打吧。”苏重下巴微抬，示意皇宫内那些巨大蚕茧似的东西。
<br/>　　那里面，全都是蚁王正准备转化的战斗兵蚁。是他为自己准备，用来征服世界的兵器。
<br/>　　蚁王脸色不变：“那些？我不在乎的。只要我活着，只要有足够的食物，族群想怎么扩大就怎么扩大。”说到食物二字时，还似笑非笑的瞥了地面的人类空地一眼。
<br/>　　苏重同样面不改色：“是吗，其实人类也很好养活。食物匮乏一些也没问题，人类可是非常热衷于繁衍的。而且，你要相信，在某些人眼里，少个万把人，并不是什么大事。”
<br/>　　“我看，你们还是找个地方打吧。有我在场，相比你们也打不尽兴吧。”尼特罗笑呵呵道。
<br/>　　蚁王冷血，苏重同样不是善人。两人真要在这里干起来，底下那数万人都得遭殃！这不是尼特罗想看到的。
<br/>　　蚁王扫了尼特罗一眼，不屑一笑，直接转身飞走。
<br/>　　苏重诧异的看着远去的梅路艾姆。对方竟能凭空飞行！仔细观察，这不就是自己的飞行方式吗？念力喷涌，借助反推力，实现腾空飞行。
<br/>　　是上次战斗，苏重恍然。当时揍蚁王，苏重就曾经在半空奔腾追赶。
<br/>　　只看了一次就学会了吗？可怕的学习能力！嵌合蚁，果然是个危险种族。如果给它们足够的时间，崛起人类大陆，甚至反攻黑暗大陆都有可能！
<br/>　　两人速度飞快，很快就消失在半空。十多分钟的时间，两人就飞出数百公里。
<br/>　　一处空旷原野，梅路艾姆率先落下地面，苏重紧跟其后。
<br/>　　“你就不怕尼特罗，扫了你的老巢？”蚁王梅路艾姆，不可能感觉不到尼特罗的实力。但即便如此，蚁王却不理会尼特罗，直接离开。难道有什么底牌？
<br/>　　“他？只要他有那个本事。”梅路艾姆不屑一笑。
<br/>　　果然有底牌！这个蚁王，和原著中的绝不相同！
<br/>　　蚁王刚出生，就被苏重揍了一顿。清晰认识到世界残酷的蚁王，不择手段了解世界、提升实力。
<br/>　　蚁王本就天赋卓绝，智慧不小。短时间成长，不只是力量大增，竟然懂得推演谋划！此刻敢独自和苏重放对，显然有着其他安排。
<br/>　　嘿！希望尼特罗老头，能顶住蚁王后手吧。苏重幸灾乐祸一笑。至于尼特罗安危，管他去死！他哪里会在乎！
<br/>　　“来吧，尽情发挥你的实力。不然，你就没机会了。”蚁王站在远处，淡然道。那高高在上的语气，一点儿都没把苏重放在眼里。
<br/>　　“忘了当初被我打的有多惨了？”苏重眉头一挑。。
<br/>　　“你也就只能用历史来彰显自己了。”蚁王丝毫不生气，甚至怜悯的看着苏重：“人类，果然是一个劣等种族。不思进取，只会沉浸在过去的荣光岁月。”
<br/>　　说着梅路艾姆摇了摇头叹了口气：“这么大一片大陆，竟然被这种劣等种族占据，暴殄天物。”
<br/>　　苏重被气乐了。张口劣等种族，闭口劣等种族，真他么火大！
<br/>　　极翼！
<br/>　　轰。右手成刀，整条右臂猛然抡起来。好似一条长鞭大力甩动，带着手刀狠狠斩向梅路艾姆。

<br/>　　天空云团，瞬间变成两半。整片空间都被狂暴气息充满。
