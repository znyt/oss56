　　迦南学院阁楼上，紫袍斗尊为首，苏千与魏长老等人站在身后，望着慕清霄与小医仙离去的背影，脸上皆是挂着一丝苦笑。
<br/>　　离开迦南学院后，慕清霄携美同行，算不上赶路，只能说是游山玩水。
<br/>　　路径黑角域的时候，两人并没有在那种影响好心情的地点停留。
<br/>　　在小医仙心中，哪怕魔兽山脉中，张着血盆大口的魔兽，有时候都比各种阴暗的心理可爱的多。
<br/>　　经过塔戈尔大沙漠时，彩鳞回了一趟蛇人族部落。
<br/>　　此时，蛇人族部落已经有了新的首领，在沙漠深处繁衍生息，彩鳞只是在远处稍稍看了一眼，已经没有回去的想法。
<br/>　　两日后的清晨，小医仙站在青山镇附近，魔兽山脉中的一座悬崖上，望着脚下一株白兰果，眸子中闪过一抹美好的回忆。
<br/>　　现在白兰果已经重新长出果实，回想起当初，两人就是这座悬崖上相识……
<br/>　　斜靠在树梢上，慕清霄语气温柔的说道：“仙儿，我们该走了。”
<br/>　　“嗯，来啦。”
<br/>　　回过神来，小医仙蹦蹦跳跳的转身离开，温柔笑道：“现在我们去哪里？”
<br/>　　“云岚宗。”
<br/>　　“云岚宗？那不是加玛帝国最大的势力吗？”
<br/>　　慕清霄微微点头：“就是不知道她还在不在云岚宗，若是不在的话，我们到时候就直接去中州。”
<br/>　　说罢，看向西侧的方向，目光仿佛跨越空间，在加玛帝国圣城西侧，有一座雪白的山峰，巍峨矗立。
<br/>　　其上，隐隐有剑鸣声冲天而起。
<br/>　　……
<br/>　　……
<br/>　　云岚宗，加玛帝国最强大的势力，代代相传已经让这个古老的宗派屹立在了加玛帝国的巅峰。
<br/>　　若非宗派不可夺王权，云岚宗恐怕早就掌控了整个加玛帝国。
<br/>　　也正是因此，每一代帝国的皇帝，都对这个近在咫尺的庞然大物极其忌惮。
<br/>　　只是，半年前的变故，让云岚宗一蹶不振。
<br/>　　云岚宗前任宗主云山陨落，现任宗主云韵退位，墙角众人推，不仅皇室，就连周围的小型宗门都打上了云岚宗的主意。
<br/>　　饶是如此，云岚宗的底蕴也不是其他宗门能比的。
<br/>　　结果，某些宵小被云岚宗众长老一举击杀，皇室与宗门都不敢轻举妄动。
<br/>　　加玛帝国圣城中的一座酒楼中，小医仙听着周围客人的谈论声，嫣然一笑。
<br/>　　“我们似乎要去中州了呢。”
<br/>　　慕清霄微微点头，既然云韵不在云岚宗，那就只能前往中州了。
<br/>　　看来由于某些关系，剧情再度偏离轨迹，萧炎只是大斗师，想必是药老出手将云山击杀，只是为何没有魂殿的消息。
<br/>　　“休息一晚就去中州吧。”
<br/>　　见他脸上的坏笑，小医仙似乎想到了什么，柔美至极的俏脸上顿时浮现一抹诱人的红晕，美眸狠狠的瞪了他一眼。
<br/>　　深夜，酒楼最高档的卧室中，便响起一阵婉转低吟。
<br/>　　只是有空间结界的存在，内部的事情不被外人所知晓。
<br/>　　……
<br/>　　……
<br/>　　竖日清晨，慕清霄与小医仙简单洗漱后，便迈上了前往中州的路途。
<br/>　　蔚蓝的天空上，七彩小蛇卷缩在慕清霄的肩膀上，道：“我们如何去中州？”
<br/>　　“我知道，我知道。”
<br/>　　话语刚落，小医仙调动灵气在慕清霄周围环绕，她非常喜欢这种不受拘束，自由自在的感觉。
<br/>　　以前她都只能乘坐蓝鹰在天空上翱翔，哪怕成为斗灵后，都饱受厄难毒体的折磨，根本无法用斗气飞翔。
<br/>　　现在，调动灵气就能翱翔于空，让她感觉到前所未有的快乐。
<br/>　　“彩鳞姐姐，我听说想要去中州，需要先抵达一座名为‘天涯城’的城市，在城市里，我们可以通过空间虫洞前往中州。”
<br/>　　“本王听说过，空间虫洞是中州大陆特有的东西，是由斗尊施展空间之力连通两个空间点所形成，加玛帝国与中州的距离，就算斗宗也需要半年时间才能跨越，若走空间虫洞就只需一个月的时间。”
<br/>　　说着，彩鳞看了一眼慕清霄，道：“但是，我们没必要走空间虫洞，有他在根本就不必如此麻烦。”
<br/>　　闻言，小医仙吐了吐舌头，可能是一直在身边的原因，她潜意识的就会忽略慕清霄的可怕。
<br/>　　现在想想，自己的丈夫似乎是传说中的斗圣，空间虫洞对他来说，还不是小意思？
<br/>　　慕清霄一眼就看出了小医仙的心思，微笑道：“彩鳞，你似乎也没去过中州吧？”
<br/>　　“……本王只是听说，并没有真正的去过，你也知道，本王的实力，在中州根本无法立足，现在也有些勉强。”
<br/>　　当初，彩鳞只是斗皇级别，却能在加玛帝国称霸一方，可斗皇满地跑的中州根本不稀奇。
<br/>　　哪怕现在她已经是斗宗，在中州依旧不算强者，毕竟，专门修理空间虫洞的工程师都是斗宗级别。
<br/>　　甚至，每一个可能都有不输给她的实力。
<br/>　　慕清霄揉了揉小医仙的脑袋，道：“我们不急着赶路，既然没去过，那就到处走一走，我们修的是仙路，心境至关重要。”
<br/>　　此言，他可不是胡编乱造，修仙与斗气不同。
<br/>　　哪怕他独自一人赶往目的地的时候，如果没有必要，他也喜欢游山玩水，各处看看风土人情，如此一来，心胸开阔，更有利于修行。
<br/>　　或许，就在这种不知不觉的放松下，瓶颈就会松动许多。
<br/>　　当然，此行对于慕清霄来说，未必有意义，但小医仙与彩鳞已经改修仙路，四处走走也不失为一件好事。
<br/>　　“好耶，彩鳞姐姐，我们去天涯城吧。”
<br/>　　话语刚落，小医仙欢快的笑了起来，如同黄鹂一样。
<br/>　　彩鳞金色蛇瞳眨了眨，无奈的瞪了一眼慕清霄，似乎找不到拒绝的理由。
<br/>　　“随你们吧。”
<br/>　　见彩鳞同意，小医仙抱住慕清霄，轻轻在他脸上吻了一下。
<br/>　　慕清霄则溺爱的捏了捏她的小鼻子，道：“走吧，天涯城距离此处可是有几万里，速度快的话，半日就能抵达。”
<br/>　　其实，以慕清霄的速度，空间挪移，两个呼吸的时间就能抵达天涯城。
<br/>　　可是见小医仙一脸享受的神情后，他就没有开口。
<br/>　　……
<br/>　　……
