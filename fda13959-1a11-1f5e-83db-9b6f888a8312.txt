　　一拳震退慕容杀后，萧逸追击而上。
<br/>　　澎湃紫炎，立刻化为一条条炎蛇。
<br/>　　慕容杀的四肢、腰间，全被束缚住。
<br/>　　“不好。”慕容杀一惊，连忙调动全身真气，覆盖于自己周身，试图抵消掉紫炎。
<br/>　　但他却惊骇地发现，自己全身真气已然调动，竟只堪堪和紫炎做着抵消。
<br/>　　紫炎缠绕，距离他的皮肤还有一丝距离。
<br/>　　但他也无法完全将紫炎化去。
<br/>　　他乃洞玄六重武者，本身真气，远超萧逸。
<br/>　　但萧逸此刻借助了灵器的力量。
<br/>　　“慕容杀，我劝你还是不要乱动为好。”萧逸冷笑一声，手上烈焰手套气势惊人。
<br/>　　成就冰火道体，以及身体素质大增后，他对灵器的掌控度更高了。
<br/>　　烈焰手套，他可调动十成力量，而不至于受反噬。
<br/>　　烈焰手套十成力量下，增幅的紫炎，其数量和澎湃度，达到一个惊人的地步。
<br/>　　足以抵消掉萧逸与慕容杀的真气差距。
<br/>　　“你的乾坤袋，便交给我保管了。”萧逸冷笑着，一把夺过他腰间的乾坤袋。
<br/>　　“你….。“慕容杀怒不可遏，却无可奈何。
<br/>　　他不敢乱动，怕碰到恐怖的紫炎。
<br/>　　萧逸身影一闪，来到慕容胤身前，冷声道，“你的乾坤袋，也交给我了。“
<br/>　　慕容胤脸色难看，却不敢有任何反抗。
<br/>　　而后，萧逸在已经死去的护卫身上，夺过一个个乾坤袋。
<br/>　　至于其他商物，由于这些乾坤袋都是早已装满的，他只能用自己的乾坤袋。
<br/>　　他自己多的是乾坤袋。
<br/>　　做好一切，萧逸转身便走，并没有杀慕容胤和慕容杀。
<br/>　　两人冷冷地看着萧逸离去，却毫无办法。
<br/>　　“易霄，今日的事，给我记着，我不会放过你的。“慕容杀不甘地吼了一声。
<br/>　　“不会放过我？“萧逸停下脚步，转头看了他一眼。
<br/>　　“就冲你这句话，我便继续陪你玩下去。“
<br/>　　萧逸说罢，身影瞬间消失在原地。
<br/>　　“哎呀，慕容杀执事，你糊涂阿。“慕容胤气急败坏地说道。
<br/>　　“这易霄，一看就知道不是善茬。从他神秘崛起，到如今，每一次针对我们盛宝商行，都必是我们先与他结下仇怨。“
<br/>　　“而每一次的仇怨，他都是锱铢必报，不达目的，誓不罢手。“
<br/>　　“他就是个疯子。“
<br/>　　“此番你再威胁他，也不知他会做出些什么疯狂的事来。“
<br/>　　慕容杀脸色一变，但还是镇定地说道，“怕什么，难道你以为他还敢去百武城攻打盛宝商行不成？“
<br/>　　慕容杀故作镇定地说着，却不知，此刻的萧逸，真的就往百武城盛宝商行而去。
<br/>　　…….
<br/>　　百武城，盛宝商行。
<br/>　　虽然只是一个分行，却从无人敢到那里放肆，更别说抢劫了。
<br/>　　因为，这是一个分行遍布北山郡的庞然大物。
<br/>　　背后，更有北山慕容家这样的霸主势力。
<br/>　　但今日，这里，却迎来了一位不速之客。
<br/>　　这位不速之客，刚踏入盛宝商行，周围的护卫，已经马上围了过来。
<br/>　　“紫炎易霄，你还敢来这里？真是自投罗网。“
<br/>　　不速之客，自然便是萧逸。
<br/>　　萧逸看了眼周围的护卫，心里暗暗想着，“果然没有猜错，盛宝商行内，还有两个洞玄境二重的武者。”
<br/>　　那些包围他的护卫，清一色是先天九重武者。
<br/>　　为首的，是两个洞玄二重的中年人。
<br/>　　“真是天堂有路你不走，地狱无门你偏闯来。”
<br/>　　那两个中年人冷笑一声。
<br/>　　“给我看好他，待得慕容杀执事和慕容胤执事回来，必将这小贼扒皮拆骨。”
<br/>　　“是。”
<br/>　　众护卫应答一声，谨慎地包围住萧逸。
<br/>　　他们哪里知道，他们口中的慕容杀和慕容胤，现在还被萧逸的紫炎困住呢。
<br/>　　“据闻盛宝商行宝物众多，此番看来，果然不假。”
<br/>　　“易某近来囊中羞涩，便来随便取些。”
<br/>　　萧逸淡淡地说道。
<br/>　　随便取些？这简直是抢好吧。
<br/>　　萧逸话音刚落，瞬间出手。
<br/>　　一众护卫，不出数秒，尽皆倒地。
<br/>　　两个洞玄二重武者，也被一拳打得重伤。
<br/>　　萧逸大手一挥，直接将展览台内的商物取入乾坤袋。
<br/>　　妖兽精血、妖兽内丹、天材地宝、亚灵器等等，大堆大堆地拿走。
<br/>　　不多时，原本琳琅满目的商行，硬生生出现了一大片空荡。
<br/>　　商物，被拿走了三成以上。
<br/>　　做完这一切，萧逸转身离去，没有再拿。
<br/>　　“盛宝商行，易某恩怨分明。这次，与你们的仇怨还不算大，便不拿光了。“
<br/>　　“这些宝物，就当作给易某赔罪吧。“
<br/>　　说着，萧逸还得意地扬了扬手中的乾坤袋，扬长而去。
<br/>　　放荡不羁的笑声，响彻整个盛宝商行，却无人敢拦。
<br/>　　…….
<br/>　　数分钟后，慕容胤和慕容杀急急地回到盛宝商行。
<br/>　　其实，萧逸之前刚走不久，他们身上的紫炎便被他们用真气抵消了。
<br/>　　所以，现在萧逸前脚刚走，他们后脚就回来了。
<br/>　　当两人看着商行内狼藉一片，护卫、人员个个受伤时，顿时双眼冒火，杀意澎湃。
<br/>　　“易霄，我要你不得好死。“慕容杀疯狂地吼了起来。
<br/>　　“哎。“慕容胤叹了口气，道，“慕容杀执事，若刚才你不威胁他，说不定现在就不会这样了。”
<br/>　　“那你要我如何？”慕容杀吼道，“难道任凭那个小子骑在我头上，而我却不得埋怨半句不成？”
<br/>　　“我不是那个意思。”慕容胤道，“哎，算了。“
<br/>　　恰在此时，商行外，两道人影来临，当看到商行内的情况，也是吃了一惊。
<br/>　　“慕容杀执事，到底怎么回事？“那两人惊讶地问道。
<br/>　　慕容杀见了这两人，立刻收敛了怒气，皱眉问道，“家族的人？你们怎么来了？“
<br/>　　那两人拱拱手道，“奉慕容影长老之命，特来一番。”
<br/>　　这两人乃是北山慕容家内的子弟，皆是先天武者。
<br/>　　“慕容影长老？”慕容杀的眉头皱得更紧，同时，他已经知道这两人来此的目的了。
<br/>　　“嗯。”那两人道，“影长老让我们带一句话。”
<br/>　　“他在郡都忙得很，分不开身。再者，紫云城那种小地方的小家族，要他亲自出手对付不成？”
<br/>　　“不敢。”慕容杀连忙低头，再拱了拱手。
<br/>　　这时，两人在他耳边，低语道，“慕容杀长老，慕容墨那两个小辈，在家族里哄得影长老可开心了。我想，此事，你还是尽快处理为好。”
<br/>　　慕容杀脸色变了变，道，“我知道，但这边的商行，也出了大事。”
<br/>　　“我们北山慕容家，乃是商业家族，自当以生意为先。”
<br/>　　“百武城靠近裂天剑派，这附近武者众多。故，这里的商行也是重中之重，可不能出差池。”
<br/>　　“这样吧。”慕容杀说道，“告诉影长老，给我半年时间，我必定处理好一切事宜。”
<br/>　　“这段时间，便让慕容墨那两个小辈先安心修炼。半年后，我亲自带他们回紫云城，让那小家族在他们眼中，亲眼灭亡。”
<br/>　　“好吧。”那两人点点头，道，“我们回如实转告影长老的。“
<br/>　　说罢，两人转身离去，赶回郡都。
<br/>　　下一秒，慕容杀的脸色，再次变为冰冷至极，杀意凛然。
<br/>　　“紫炎易霄，我与你不共戴天。半年之内，我必取你性命。“慕容杀咬着牙，拳头握得劈啪作响。
<br/>　　……
<br/>　　他怎么都想不到，在暗中，一直有一道人影在注视着他。
<br/>　　他更想不到，易霄抢完盛宝商行，竟还敢待在盛宝商行内。
<br/>　　“半年？哼。“萧逸心头冷哼一声，身影瞬间消失在原地，往百武城外而去。
<br/>　　他特地留在盛宝商行，就是为了看慕容杀的反应。
<br/>　　他抢盛宝商行，可不是为了宝物，只为牵制住慕容杀。
<br/>　　所以他很有分寸地只抢了三成。
<br/>　　若全抢了，估计北山慕容家就要直接派高手来对付他了。
<br/>　　现在只抢三成，一切还在慕容杀处理范围内。
<br/>　　而盛宝商行内的护卫，他也一个没杀，仅仅是打伤。
<br/>　　只有那两个洞玄二重受了重伤。
<br/>　　为的是让慕容杀手头上没有可以调动的力量，让他更加焦头烂额。
<br/>　　他之前说的话，做的事，全是为了误导慕容杀。
<br/>　　让他们以为自己是个锱铢必报的人。
<br/>　　他对付盛宝商行，全是因为他们之前得罪过自己。
<br/>　　现在看来，他的计划成功了。
<br/>　　不过，他并未因此而放松。
<br/>　　“半年，看来我要加快脚步了。“萧逸暗暗想着，”只要突破洞玄境，后面的修炼，我有把握在短时间内快速突破修为。“
<br/>　　从之前慕容杀的话来看，萧家如今正被紫云城那边的盛宝商行打压着。
<br/>　　但他暂时没有办法。
<br/>　　紫云城，在破玄城十数城范围内。
<br/>　　不过，破玄城那边的武者和家族都相对较弱。
<br/>　　那边的盛宝商行分行，负责人不过是个洞玄一重武者。
<br/>　　暂时威胁不了萧家，最多只是打压。
<br/>　　一切还是要靠自己，只要自己实力足够强，便什么都不用怕了。
<br/>　　想罢，萧逸身影一闪，往四季城白家而去。
